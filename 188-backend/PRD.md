# 医疗护理培训签到考核与技能档案系统 PRD

## 项目概述
本系统面向医院护理部、带教老师、考核老师与护理人员的培训管理场景，提供培训项目、护理人员档案、培训班次、培训签到、技能项目、技能考核、证书档案、续训提醒、实操记录、考核申诉和培训通知的一体化管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 护理培训管理员：维护培训项目、培训班次、技能项目和培训通知
- 带教老师：组织班次培训、登记签到、维护实操记录和指导结果
- 考核老师：维护技能考核、评分记录、申诉处理和考核结论
- 护理人员：查看培训班次、完成签到、参与考核、查看证书和续训提醒
- 护理人事：维护护理人员档案、证书档案和续训周期

## 功能模块
- 培训项目：项目编号、项目名称、培训类型、负责人、计划时间和项目状态维护
- 护理人员档案：档案编号、护士姓名、科室类型、责任老师、建档时间和在岗状态维护
- 培训班次：班次编号、班次名称、培训类型、带教老师、开班时间和班次状态维护
- 培训签到：签到编号、关联班次、签到类型、签到人员、签到时间和签到状态维护
- 技能项目：技能编号、技能名称、技能类型、维护人员、更新时间和技能状态维护
- 技能考核：考核编号、考核项目、考核类型、考核老师、考核时间和考核状态维护
- 证书档案：证书编号、证书名称、证书类型、归档人员、发证时间和证书状态维护
- 续训提醒：提醒编号、证书项目、提醒类型、提醒人员、提醒时间和续训状态维护
- 实操记录：实操编号、实操项目、实操类型、指导老师、实操时间和记录状态维护
- 考核申诉：申诉编号、关联考核、申诉类型、申诉人员、申诉时间和处理状态维护
- 培训通知：通知编号、通知对象、通知类型、发布人员、发布时间和通知状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护
