# 200-frontend UI 美化展示

## 项目信息
- **项目名称**: 非遗工坊课程预约与作品展销平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 非遗棕
- **美化完成时间**: 2026-05-30

## 设计理念

采用"非遗棕"设计理念，以棕色和古铜色为主色调，体现非物质文化遗产的传统、典雅和匠心。通过面具、画笔等装饰元素，营造出非遗传承、文化自信的传统文化氛围。
