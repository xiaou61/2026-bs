# 电动车充电桩预约与运维管理系统开发计划

## 一、项目定位
围绕新能源充电、充电桩预约、故障运维、收益统计和能耗监测建设前后端分离管理系统，覆盖基础档案、监测记录、预警处置、任务调度、统计看板和操作审计。

## 二、技术路线
后端使用 Spring Boot 2.7.18、MyBatis 注解 SQL + PageHelper 分页查询、MySQL、Redis 和 JWT；前端使用 Vue3、Vite、Pinia、Element Plus、Axios 和 ECharts。

## 三、实施步骤
1. 初始化后端工程、配置 8124 端口、MySQL 数据库 ev_charging_124 和 Redis 27 号库。
2. 完成账号权限、充电站点、充电桩位、用户车辆、预约订单、充电记录、故障报修、维修工单、保养计划 等核心模块实体、Mapper、Service 和 Controller。
3. 编写初始化 SQL，提供四类演示账号和每个核心模块的示例数据。
4. 搭建 Vue3 前端，完成登录、布局、通用数据页、状态动作、统计看板和路由。
5. 更新合集 README 和候选清单，执行静态数量、配置、关键词和注释扫描。

## 四、验收范围
- 后端模块、接口、SQL 初始化脚本完整。
- 前端页面、路由、API 封装和看板完整。
- MySQL 账号密码使用 root / 1234。
- 按仓库规则仅做静态验证，不执行编译构建。
