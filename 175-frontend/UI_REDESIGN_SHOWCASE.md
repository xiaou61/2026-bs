# 175-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园图书漂流借阅与读书打卡平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 书香紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"书香紫"设计理念，以紫色和蓝色为主色调，体现阅读的文艺、浪漫和智慧感。通过书本、星星等装饰元素，营造出书中自有黄金屋的阅读氛围。

## 颜色方案

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳效果 |
| 顶部条 | bookGradient | 彩虹流光 |
| 按钮 | hover | 悬浮上移 |

## 总结

175-frontend采用"书香紫"设计理念，通过紫色和蓝色的搭配，营造出文艺、智慧的阅读氛围。书本弹跳动画突出了图书漂流借阅的特色。
