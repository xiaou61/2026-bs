# 智慧工地安全巡检与隐患整改系统开发计划

## 一、项目定位
围绕智慧工地、安全巡检、隐患整改、班组责任和验收闭环建设前后端分离管理系统，覆盖基础档案、业务流转、状态处理、统计看板和操作审计。

## 二、技术路线
后端使用 Spring Boot 2.7.18、MyBatis 注解 SQL + PageHelper 分页查询、MySQL、Redis 和 JWT；前端使用 Vue3、Vite、Pinia、Element Plus、Axios 和 ECharts。

## 三、实施步骤
1. 初始化后端工程、配置 8122 端口、MySQL 数据库 smart_worksite_safety_122 和 Redis 25 号库。
2. 完成账号权限、工地项目、施工班组、安全员、巡检计划、巡检任务、隐患上报、整改工单、验收记录 等核心模块实体、Mapper、Service 和 Controller。
3. 编写初始化 SQL，提供四类演示账号和每个核心模块的示例数据。
4. 搭建 Vue3 前端，完成登录、布局、通用数据页、状态动作、统计看板和路由。
5. 更新合集 README 和候选清单，执行静态数量、配置、关键词和注释扫描。

## 四、验收范围
- 后端模块、接口、SQL 初始化脚本完整。
- 前端页面、路由、API 封装和看板完整。
- MySQL 账号密码使用 root / 1234。
- 按仓库规则仅做静态验证，不执行编译构建。
