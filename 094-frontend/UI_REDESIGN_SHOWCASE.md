# 094-frontend UI 美化展示

## 项目信息
- **项目名称**: 宠物咖啡馆平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 咖啡橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"咖啡橙"设计理念，以橙色和金色为主色调，体现宠物咖啡馆的温馨、可爱和活力感。通过猫咪、咖啡等装饰元素，营造出和猫咪一起喝咖啡的惬意氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 咖啡橙 | #FF6B35 | 主色调 |
| 暖橙 | #F7931E | 辅助色 |
| 金色 | #FFB347 | 强调色 |
| 亮金 | #FFD700 | 装饰色 |
| 深橙 | #E65100 | 文字色 |
