# 电子合同签署与印章审批管理系统 PRD

## 项目定位
电子合同签署与印章审批管理系统面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

## 用户角色
系统管理员、法务专员、业务申请人、审批负责人

## 功能模块
- 合同模板：模板编号、模板名称、合同类型、适用场景、模板版本和状态维护
- 对方单位：单位编号、单位名称、联系人、联系邮箱、合作类型和状态维护
- 签署方档案：签署方编号、签署方姓名、所属部门、联系方式、证件类型和状态维护
- 合同草稿：草稿编号、合同名称、申请部门、拟签时间、草稿版本和状态维护
- 用印申请：申请编号、合同名称、印章类型、申请时间、申请人和状态维护
- 审批流程：流程编号、申请编号、审批节点、审批意见、审批时间和状态维护
- 合同签署：签署编号、合同名称、签署方式、签署时间、签署人和状态维护
- 用印记录：记录编号、合同名称、印章类型、用印时间、经办人和状态维护
- 归档记录：归档编号、合同名称、归档位置、归档时间、归档人和状态维护
- 到期提醒：提醒编号、合同名称、到期日期、提醒对象、提醒方式和状态维护
- 风险条款：条款编号、合同名称、风险类型、风险描述、责任部门和状态维护
- 合同通知：通知编号、合同名称、通知类型、通知内容、接收人和状态维护
- 操作日志：关键模块操作人、动作、对象、详情和结果记录

## 技术要求
- 后端：Spring Boot 2.7.18、MyBatis 2.3.1 注解 SQL、PageHelper 1.4.7、MySQL、Redis、JWT
- 前端：Vue3、Vite、Element Plus、Pinia、Axios、ECharts
- 数据库：electronic_contract_140
- 默认账号密码：各角色账号均使用 `123456`
