# 153-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园二手物品寄卖与信用评价系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 校园橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"校园橙"设计理念，以橙色和金色为主色调，体现校园二手交易的活力、热情和青春感。通过循环、金币等装饰元素，营造出闲置不闲、校园共享的二手交易氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | spin | 持续旋转 |

| 按钮 | hover | 悬浮上移 |

## 总结

153-frontend采用"校园橙"设计理念，通过橙色和金色的搭配，营造出活力、青春的校园二手交易氛围。循环旋转动画突出了闲置物品循环利用的特色。
