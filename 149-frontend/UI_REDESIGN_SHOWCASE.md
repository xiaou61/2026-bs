# 149-frontend UI 美化展示

## 项目信息
- **项目名称**: 高校实验设备共享预约与违规使用追踪系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 实验蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"实验蓝"设计理念，以蓝色为主色调，体现实验设备管理的专业性、严谨性和科技感。通过显微镜、试管等装饰元素，营造出实验设备、共享共赢的高校实验室管理氛围。

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | labGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
