# 143-frontend UI 美化展示

## 项目信息
- **项目名称**: 社区公益时间银行互助服务平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 公益粉
- **美化完成时间**: 2026-05-30

## 设计理念

采用"公益粉"设计理念，以粉色和玫红色为主色调，体现社区公益的温暖、关爱和互助精神。通过时钟、爱心等装饰元素，营造出时间银行、爱心互助的社区服务氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | tick | 左右摇摆 |
| 顶部条 | timeGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

143-frontend采用"公益粉"设计理念，通过粉色和玫红的搭配，营造出温暖、关爱的社区公益氛围。时钟摇摆动画突出了时间银行互助的特色。
