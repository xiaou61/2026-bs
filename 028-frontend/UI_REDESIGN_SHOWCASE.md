# 028-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园共享自行车租赁系统
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 骑行绿
- **美化完成时间**: 2026-05-29

## 设计理念

采用"骑行绿"设计理念，以绿色和橙色为主色调，体现绿色出行和环保理念。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 骑行绿 | #4CAF50 | 主色调 |
| 活力橙 | #FF9800 | 辅助色 |
| 天空蓝 | #03A9F4 | 强调色 |
| 背景色 | #E8F5E9 | 页面背景 |
