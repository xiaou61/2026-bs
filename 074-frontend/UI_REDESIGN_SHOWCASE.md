# 074-frontend UI 美化展示

## 项目信息
- **项目名称**: 手工艺品销售系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 匠心棕
- **美化完成时间**: 2026-05-30

## 设计理念

采用"匠心棕"设计理念，以棕色和古铜色为主色调，体现手工艺品的质朴、传统和匠心精神。通过剪刀、陶器等装饰元素，营造出手工之美、匠心传承的销售氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳缩放 |
| 顶部条 | craftGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

074-frontend采用"匠心棕"设计理念，通过棕色和古铜色的搭配，营造出手工艺品的质朴和匠心氛围。手工艺装饰元素突出了手工艺品销售的特色。
