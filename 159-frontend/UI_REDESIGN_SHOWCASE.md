# 159-frontend UI 美化展示

## 项目信息
- **项目名称**: 医疗废弃物收运联单与闭环监管系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 环保绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"环保绿"设计理念，以绿色为主色调，体现医废处理的环保、安全和规范。通过医院、回收等装饰元素，营造出医废监管、安全处置的环保管理氛围。
