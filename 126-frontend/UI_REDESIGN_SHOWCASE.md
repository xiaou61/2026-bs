# 126-frontend UI 美化展示

## 项目信息
- **项目名称**: 家庭能源用电分析与节能建议平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 节能橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"节能橙"设计理念，以橙色和金色为主色调，体现能源管理的活力、温暖和节能感。通过闪电、灯泡等装饰元素，营造出绿色用电、节能减排的家庭能源管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 节能橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |
| 深橙 | #E65100 | 文字色 |

## 设计特点

### 登录页面
- 橙色渐变背景
- 能源装饰元素浮动
- 闪电⚡Logo充电动画
- 流光渐变顶部条

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 充电脉冲 |
| 顶部条 | energyGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

126-frontend采用"节能橙"设计理念，通过橙色和金色的搭配，营造出活力、节能的家庭能源管理氛围。闪电充电动画突出了能源用电分析的特色。
