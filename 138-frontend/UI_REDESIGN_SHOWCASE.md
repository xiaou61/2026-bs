# 138-frontend UI 美化展示

## 项目信息
- **项目名称**: 在线考试反作弊行为分析与证据管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 考试红
- **美化完成时间**: 2026-05-30

## 设计理念

采用"考试红"设计理念，以红色为主色调，体现考试的严肃性、公正性和权威感。通过试卷、摄像头等装饰元素，营造出诚信考试、公平竞争的考试管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |

| 元素 | 动画 | 说明 |
|------|------|------|
