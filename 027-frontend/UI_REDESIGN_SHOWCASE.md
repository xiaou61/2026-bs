# 027-frontend UI 美化展示

## 项目信息
- **项目名称**: 线上理发预约系统
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 时尚粉
- **美化完成时间**: 2026-05-29

## 设计理念

采用"时尚粉"设计理念，以粉色和金色为主色调，体现美发沙龙的时尚和优雅。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 时尚粉 | #E91E63 | 主色调 |
| 优雅金 | #FFB300 | 辅助色 |
| 时尚紫 | #9C27B0 | 强调色 |
| 背景色 | #FCE4EC | 页面背景 |
