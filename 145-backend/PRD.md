# 城市噪声投诉监测与执法协同平台 PRD

## 项目定位
城市噪声投诉监测与执法协同平台面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

## 用户角色
系统管理员、市民投诉人、执法人员、环保监督员

## 功能模块
- 投诉工单：工单编号、投诉标题、噪声类型、投诉区域、投诉时间和状态维护
- 监测点位：点位编号、点位名称、点位类型、所属街道、监测阈值和状态维护
- 噪声源档案：源档编号、噪声源名称、责任单位、源头类型、联系人和状态维护
- 执法人员档案：执法编号、执法姓名、负责片区、联系方式、入职时间和状态维护
- 处置任务：任务编号、投诉标题、责任人、下发时间、处理时限和状态维护
- 现场巡查：巡查编号、投诉标题、巡查人、巡查时间、巡查结论和状态维护
- 整改通知：通知编号、投诉标题、整改要求、下发时间、责任单位和状态维护
- 复测记录：复测编号、投诉标题、复测人、分贝值、复测时间和状态维护
- 处罚决定：处罚编号、投诉标题、处罚类型、处罚对象、处罚金额和状态维护
- 公众回访：回访编号、投诉标题、回访主题、回访方式、回访时间和状态维护
- 预警规则：规则编号、投诉标题、预警指标、阈值配置、生效时间和状态维护
- 公告公示：公告编号、投诉标题、公告类型、公告内容、发布时间和状态维护
- 操作日志：关键模块操作人、动作、对象、详情和结果记录

## 技术要求
- 后端：Spring Boot 2.7.18、MyBatis-Plus 3.5.5、MySQL、Redis、JWT
- 前端：Vue3、Vite、Element Plus、Pinia、Axios、ECharts
- 数据库：noise_monitor_145
