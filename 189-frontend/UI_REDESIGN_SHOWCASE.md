# 189-frontend UI 美化展示

## 项目信息
- **项目名称**: 乡镇农机作业预约与维修保养跟踪平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 农机绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"农机绿"设计理念，以深绿色为主色调，体现农机服务的生态、自然和科技感。通过拖拉机、稻穗等装饰元素，营造出科技兴农、机械强农的农业服务氛围。
