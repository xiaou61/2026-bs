# 184-frontend UI 美化展示

## 项目信息
- **项目名称**: 停车场月租合同续费与异常占位管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 停车蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"停车蓝"设计理念，以蓝色为主色调，体现停车场管理的专业性、规范性和科技感。通过停车场、汽车等装饰元素，营造出智慧停车、便捷出行的停车管理氛围。

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |

| 按钮 | hover | 悬浮上移 |
