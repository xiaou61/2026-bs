# 083-frontend UI 美化展示

## 项目信息
- **项目名称**: 老年人体检管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 健康粉
- **美化完成时间**: 2026-05-30

## 设计理念

采用"健康粉"设计理念，以粉色和玫红色为主色调，体现对老年人健康的关爱和呵护。通过医院、爱心等装饰元素，营造出关爱健康、呵护晚年的医疗管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|

| 浅粉 | #F06292 | 辅助色 |
| 柔粉 | #F48FB1 | 强调色 |
| 深红 | #AD1457 | 文字色 |
| 淡粉 | #FCE4EC | 背景色 |

## 设计特点

### 登录页面
- 粉色渐变背景
- 医疗装饰元素浮动
- 医院🏥Logo脉冲动画
- 流光渐变顶部条

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | healthGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

083-frontend采用"健康粉"设计理念，通过粉色和玫红色的搭配，营造出关爱、呵护的老年人体检管理氛围。医院装饰元素突出了医疗健康管理的特色。
