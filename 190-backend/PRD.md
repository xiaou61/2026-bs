# 智慧楼宇访修协同与设备保养提醒系统 PRD

## 项目概述
本系统面向写字楼、园区物业、入驻单位、维修技师与巡检人员的楼宇运维场景，提供楼宇档案、设备档案、入驻档案、访修工单、维修派工、保养计划、保养任务、故障预警、巡检记录、备件库存、服务评价和操作日志的一体化管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 物业管理员：维护楼宇档案、设备档案、入驻档案和服务规则
- 访修调度员：处理访修工单、维修派工、服务评价和跨岗位协同
- 维修技师：接收维修派工、执行保养任务并处置故障预警
- 设备巡检员：维护巡检记录、上报故障预警并跟进设备状态
- 入驻用户：提交访修工单、查看维修进度并完成服务评价

## 功能模块
- 楼宇档案：楼宇编号、楼宇名称、楼层区域、物业负责人、建档时间和楼宇状态维护
- 设备档案：设备编号、设备名称、设备类型、管护人员、启用时间和设备状态维护
- 入驻档案：档案编号、入驻单位、租户类型、联系人、入驻时间和服务状态维护
- 访修工单：工单编号、报修位置、访修类型、报修人员、提交时间和工单状态维护
- 维修派工：派工编号、关联工单、维修类型、维修技师、派工时间和派工状态维护
- 保养计划：计划编号、保养设备、保养类型、计划负责人、计划时间和计划状态维护
- 保养任务：任务编号、保养计划、任务类型、执行人员、执行时间和任务状态维护
- 故障预警：预警编号、预警设备、预警类型、处置人员、预警时间和预警状态维护
- 巡检记录：巡检编号、巡检区域、巡检类型、巡检人员、巡检时间和巡检状态维护
- 备件库存：备件编号、备件名称、备件类型、库管人员、入库时间和库存状态维护
- 服务评价：评价编号、服务工单、评价类型、评价人员、评价时间和评价状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
