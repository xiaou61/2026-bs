# 112-frontend UI 美化展示

## 项目信息
- **项目名称**: 零信任设备准入与访问控制管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 零信任灰
- **美化完成时间**: 2026-05-30

## 设计理念

采用"零信任灰"设计理念，以灰色为主色调，体现零信任架构的严谨性、专业性和中性感。通过锁、设备等装饰元素，营造出永不信任、始终验证的安全管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |

| 元素 | 动画 | 说明 |
