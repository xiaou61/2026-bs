# 172-frontend UI 美化展示

## 项目信息
- **项目名称**: 口腔门诊治疗预约与耗材计费管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 口腔蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"口腔蓝"设计理念，以青色和蓝绿色为主色调，体现口腔医疗的专业性、清洁感和信赖感。通过牙齿、笑容等装饰元素，营造出呵护牙齿、绽放笑容的口腔医疗氛围。

| 元素 | 动画 | 说明 |
|------|------|------|
