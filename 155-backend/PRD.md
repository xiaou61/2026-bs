# 社区党建活动报名与积分激励平台 PRD

## 项目概述
本系统面向社区基层党建治理场景，提供党组织维护、党员档案、党建活动、活动报名、签到记录、志愿任务、积分记录、积分兑换、组织关系和榜单激励的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 党务书记：党组织、党员档案、组织关系、榜单统计和通知公告维护
- 活动组织员：党建活动、活动报名、签到记录、志愿任务和积分兑换维护
- 志愿骨干：活动报名、签到记录、志愿任务、积分记录和积分兑换维护
- 社区党员：活动报名、积分记录、积分兑换、组织关系和榜单查看

## 功能模块
- 党组织维护：组织编号、组织名称、组织类型、负责人、成立时间和组织状态维护
- 党员档案：党员编号、党员姓名、所属支部、联系方式、入党时间和档案状态维护
- 党建活动：活动编号、活动主题、活动类型、组织人、活动时间和发布状态维护
- 活动报名：报名编号、党建活动、报名类型、报名人、报名时间和报名状态维护
- 签到记录：签到编号、关联活动、签到方式、签到人、签到时间和签到状态维护
- 志愿任务：任务编号、任务名称、服务类型、负责人、服务时间和任务状态维护
- 积分记录：积分编号、积分事项、积分类型、获得人、积分时间和积分状态维护
- 积分兑换：兑换编号、兑换项目、兑换类型、申请人、申请时间和兑换状态维护
- 组织关系：转接编号、转接对象、转接类型、经办人、申请时间和转接状态维护
- 榜单统计：榜单编号、榜单名称、榜单类型、统计人、统计周期和发布状态维护
- 通知公告：公告编号、公告标题、公告类型、发布人、发布时间和公告状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
