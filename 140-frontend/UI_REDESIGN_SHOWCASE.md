# 140-frontend UI 美化展示

## 项目信息
- **项目名称**: 电子合同签署与印章审批管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 合同蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"合同蓝"设计理念，以蓝色为主色调，体现电子合同的专业性、可靠性和安全性。通过文档、印章等装饰元素，营造出电子签约、安全可信的合同管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | sign | 签名摇摆 |
| 顶部条 | contractGradient | 流光渐变 |
