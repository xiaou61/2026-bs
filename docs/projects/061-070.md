**后端**

- MyBatis-Plus 3.5.5
- MySQL 8.0
- Redis
- JWT认证
- Hutool

**前端**
- Vue 3.4.0
- Element Plus 2.4.4
- Vite
- Pinia
- Axios
- ECharts 5.4.3
- ECharts 5.4.3

#### 🎯 功能模块

1. **用户与认证** - 注册登录、JWT鉴权、用户状态管理
2. **游戏分类管理** - 分类维护、排序与状态控制
3. **交易商品管理** - 商品发布、编辑、上下架、库存销量管理、封面上传
4. **交易市场** - 商品浏览、筛选搜索、详情查看
5. **订单交易** - 下单、支付、发货、完成、取消全流程
6. **售出订单** - 卖家查看售出订单并执行发货
7. **收藏系统** - 收藏/取消收藏、收藏列表
8. **评价系统** - 完成订单后评分评价、后台评价管理
9. **申诉处理** - 用户发起交易申诉、管理员回复处理并恢复订单流程
10. **个人中心** - 用户资料维护、密码修改、安全退出
11. **公告中心与看板** - 公告发布查看、运营统计图表

#### ✨ 特色亮点

- **面向游戏交易场景**：账号、道具、服务等多类型商品统一建模
- **完整交易闭环**：浏览→下单→支付→发货→完成→评价→申诉
- **资金与库存联动**：支付时自动扣减库存并进行买卖双方余额结算
- **双端视角**：管理员后台治理 + 普通用户交易操作
- **可视化运营看板**：分类分布、订单趋势、热销商品实时展示

#### 🎬 演示流程

1. 管理员登录后维护游戏分类与平台公告
2. 卖家发布游戏商品并设置价格、库存、交易方式
3. 买家进入交易市场筛选商品并提交订单
4. 买家支付订单，系统自动扣库存并结算双方余额
5. 卖家/管理员发货，买家确认完成并提交评价
6. 发生争议时买家发起申诉，管理员后台回复处理
