### 171 - 应急物资储备盘点与调拨审批平台

#### 🏷️ 项目名称
基于SpringBoot+Vue3的应急物资储备盘点与调拨审批平台

#### 💻 技术栈
**后端**
- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

**前端**

- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

#### 🎯 功能模块
1. **账号权限** - 系统管理员/仓储管理员/盘点人员/申领单位/调度人员/审核人员六角色登录、JWT鉴权和Redis登录态
2. **储备仓库** - 仓库编号、仓库名称、仓库类型、负责人员、启用时间和仓库状态维护
3. **物资分类** - 分类编号、分类名称、保障类型、负责人员、启用时间和分类状态维护
4. **物资台账** - 物资编号、物资名称、物资类别、管理人员、建档时间和物资状态维护
5. **库存批次** - 批次编号、物资名称、储备仓库、仓储人员、入库时间和批次状态维护
6. **库存盘点** - 盘点编号、盘点仓库、盘点类型、盘点人员、盘点时间和盘点状态维护
7. **盘点差异** - 差异编号、关联盘点、差异类型、处理人员、发现时间和差异状态维护
8. **申领工单** - 工单编号、申领物资、申领单位、申请人员、申请时间和工单状态维护
9. **调拨审批** - 审批编号、调拨物资、审批类型、审核人员、审批时间和审批状态维护
10. **调度任务** - 任务编号、调度物资、调度类型、调度人员、调度时间和任务状态维护
11. **出库记录** - 出库编号、出库物资、出库类型、仓储人员、出库时间和出库状态维护
12. **库存预警** - 预警编号、预警物资、预警类型、责任人员、预警时间和预警状态维护
13. **操作日志** - 日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

#### ✨ 特色亮点

- **后端接口**：55+个API
- **前端页面**：16个页面
- **代码量**：约5200+行

#### 🎓 适合场景

- 应急管理、物资储备、库存盘点、调拨审批和出库调度课程设计

- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

**前端**
- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

#### 🎯 功能模块
1. **账号权限** - 系统管理员、物业管理员、业主住户、派单员、维修人员、财务客服六类角色登录、JWT鉴权和Redis登录态
2. **小区区域** - 区域编号、区域名称、区域类型、物业人员、启用时间和区域状态维护
3. **业主档案** - 业主编号、业主姓名、住户类型、服务管家、入住时间和业主状态维护
4. **报修分类** - 分类编号、分类名称、维修类型、负责班组、启用时间和分类状态维护
5. **报修工单** - 工单编号、报修位置、报修类型、报修业主、提交时间和工单状态维护
6. **派单分配** - 派单编号、报修工单、派单类型、维修人员、派单时间和派单状态维护
7. **上门处理** - 服务编号、报修工单、服务类型、维修人员、上门时间和处理状态维护
8. **物料使用** - 物料编号、维修工单、物料类型、领用人员、使用时间和物料状态维护
9. **费用登记** - 费用编号、维修工单、费用类型、登记人员、登记时间和费用状态维护
10. **支付记录** - 支付编号、费用登记、支付类型、支付业主、支付时间和支付状态维护
11. **满意评价** - 评价编号、服务工单、评价类型、评价业主、评价时间和评价状态维护
12. **投诉回访** - 回访编号、投诉事项、回访类型、客服人员、回访时间和回访状态维护
13. **操作日志** - 日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

#### ✨ 特色亮点
- 选题贴合物业服务方向，覆盖报修工单、派单分配、上门处理、费用登记、支付记录、满意评价和投诉回访
- 形成“小区区域-业主档案-报修分类-报修工单-派单分配-上门处理-物料使用-费用登记-支付记录-满意评价-投诉回访”完整业务闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和状态完成
- 首页看板展示核心指标、近7日趋势和状态分布
