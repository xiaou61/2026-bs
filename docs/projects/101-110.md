### 101 - 多模态招聘材料解析与岗位匹配系统

#### 🏷️ 项目名称
基于SpringBoot+Vue3的多模态招聘材料解析与岗位匹配系统（简历解析 + 证书核验 + 岗位要求 + 匹配推荐 + 面试排期 + 反馈审计闭环）

#### 💻 技术栈
**后端**
- Spring Boot 2.7.18
- MyBatis-Plus 3.5.5
- MySQL 8.0
- Redis
- JWT认证

**前端**
- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

#### 🎯 功能模块
1. **账号权限** - 管理员/HR/候选人/面试官四角色登录、JWT鉴权和Redis登录态
2. **候选人档案** - 维护姓名、学历、工作年限、目标岗位、技能标签和期望薪资
3. **简历材料** - 管理简历文件地址、简历文本、学历、技能、经验和解析状态
4. **证书材料** - 管理证书名称、类型、发证机构、材料地址和HR核验结果
5. **岗位管理** - 维护岗位名称、部门、类型、地点、薪资、描述和发布状态
6. **岗位要求** - 按岗位配置技能、学历、经验、证书等关键词与权重
7. **解析任务** - 创建并启动简历解析任务，生成结构化解析结果
8. **解析结果** - 查看学历、技能、经验、分数、结论并支持人工复核
9. **匹配任务** - 为候选人和岗位创建匹配任务，启动后生成推荐结果
10. **匹配结果** - 展示匹配技能、缺失技能、匹配分、推荐等级和复核意见
11. **面试排期** - 安排面试官、时间、方式、地址并支持确认、取消、完成
12. **面试反馈** - 面试官填写评分、优势、不足、建议和面试结果
13. **统计审计** - 首页指标、图表看板和关键操作日志

#### ✨ 特色亮点
- 选题贴合AI招聘、简历解析、人才画像和智能岗位推荐热点，适合人工智能应用方向答辩
- 形成“候选人建档-材料上传-证书核验-简历解析-岗位匹配-面试安排-反馈归档”完整闭环
- 解析任务会根据工作年限、技能数量和已核验证书生成材料完整度评分
- 匹配任务会按岗位要求权重计算匹配技能、缺失技能、匹配分和推荐等级
- 四角色演示路径清晰，既能展示HR工作台，也能展示候选人材料和面试官反馈

#### 🎬 演示流程
1. 管理员登录维护用户账号并查看操作日志
2. 候选人维护个人档案、简历材料和证书材料
3. HR核验证书，通过或驳回证书并填写核验意见
4. HR维护岗位和岗位要求，发布或关闭招聘岗位
5. HR创建解析任务并启动，系统生成解析结果和材料评分
6. HR创建匹配任务并启动，系统生成岗位匹配结果和推荐等级
7. HR安排面试计划，面试官填写面试反馈，首页查看整体招聘指标

- **数据库表**：13张表
- **后端接口**：45+个API
- **前端页面**：16个页面
- **代码量**：约5500+行
- **功能模块**：13个核心模块

- 简历解析模拟算法输出学历、技能、经验、评分和结论
- 岗位匹配模拟算法按要求权重输出匹配分、缺失项和推荐等级
- ECharts 展示候选人、岗位、解析、匹配、面试和待办数据

#### 🎓 适合场景
- 多模态招聘材料解析与岗位匹配类毕设项目
