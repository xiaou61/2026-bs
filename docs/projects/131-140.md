### 140 - 电子合同签署与印章审批管理系统

#### 🏷️ 项目名称
基于SpringBoot+Vue3的电子合同签署与印章审批管理系统

#### 💻 技术栈
**后端**
- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

**前端**
- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

#### 🎯 功能模块
1. **账号权限** - 系统管理员/法务专员/业务申请人/审批负责人四角色登录、JWT鉴权和Redis登录态
2. **合同模板** - 模板编号、模板名称、合同类型、适用场景、模板版本和状态维护
3. **对方单位** - 单位编号、单位名称、联系人、联系邮箱、合作类型和状态维护
4. **签署方档案** - 签署方编号、签署方姓名、所属部门、联系方式、证件类型和状态维护
5. **合同草稿** - 草稿编号、合同名称、申请部门、拟签时间、草稿版本和状态维护
6. **用印申请** - 申请编号、合同名称、印章类型、申请时间、申请人和状态维护
7. **审批流程** - 流程编号、申请编号、审批节点、审批意见、审批时间和状态维护
8. **合同签署** - 签署编号、合同名称、签署方式、签署时间、签署人和状态维护
9. **用印记录** - 记录编号、合同名称、印章类型、用印时间、经办人和状态维护
10. **归档记录** - 归档编号、合同名称、归档位置、归档时间、归档人和状态维护
11. **到期提醒** - 提醒编号、合同名称、到期日期、提醒对象、提醒方式和状态维护
12. **风险条款** - 条款编号、合同名称、风险类型、风险描述、责任部门和状态维护
13. **合同通知** - 通知编号、合同名称、通知类型、通知内容、接收人和状态维护
14. **操作日志** - 关键模块操作人、动作、对象、详情和结果记录

#### ✨ 特色亮点
- 选题贴合电子合同流转、用印审批、在线签署、归档检索和风险条款管理方向
- 形成“合同模板-对方单位-签署方档案-合同草稿-用印申请-审批流程-合同签署-用印记录-归档记录-到期提醒-风险条款-合同通知”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 139 的 MyBatis-Plus 路线继续交替
- 用印申请、审批流程、合同签署、归档记录、到期提醒、风险条款和合同通知都有状态流转动作
- 首页看板展示合同、审批、签署、归档指标，以及审批趋势和合同类型分布

#### 📊 项目规模
- **数据库表**：14张表
- **后端接口**：60+个API
- **前端页面**：17个页面
- **代码量**：约6200+行
- **功能模块**：14个核心模块

#### 🎯 技术亮点
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 用印审批、合同签署、到期提醒和风险处置状态机联动
- ECharts 展示审批趋势和合同类型分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互

#### 🎓 适合场景
- 电子合同签署与印章审批管理系统类毕设项目
- 合同管理、法务流程、审批流、档案管理和风险控制课程设计
- 多角色后台管理系统综合实战
- 需要展示合同签署闭环、审批轨迹、归档检索和到期提醒能力的答辩项目
