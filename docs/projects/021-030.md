### 021 - 校园二手交易平台 🔥最新

#### 🏷️ 项目名称

校园二手交易平台（闲置物品交易+信用评价+在线沟通）

#### 💻 技术栈

**后端**

- Spring Boot 3.2.0
- MyBatis-Plus 3.5.5
- MySQL 8.0
- JWT 0.12.3
- Hutool工具库

**前端**

- Vue 3.4.0

- Vite 5.0.8
- Pinia 2.1.7
- Axios 1.6.2
- Vue Router 4.2.5

#### 🎯 功能模块

1. **用户系统** - 学生注册登录、个人信息管理、信用积分系统
2. **商品管理** - 商品发布编辑、图片上传、上下架管理、商品详情
3. **分类浏览** - 6大分类（教材书籍、数码电子、生活用品、服装饰品、运动器材、其他）
4. **搜索筛选** - 关键词搜索、分类筛选、成色筛选、价格/时间/热度排序
5. **订单交易** - 在线下单、订单状态跟踪、交易确认、资金安全
6. **在线沟通** - 买卖双方聊天、价格议价、交易细节沟通
7. **收藏功能** - 商品收藏、收藏列表管理、收藏提醒
8. **信用评价** - 5星评价、评价标签、信用分计算、用户信誉展示
9. **个人中心** - 我的发布、我的订单、我的收藏、信用统计

- 信用评价体系：5星评价+信用分机制，建立可信交易环境

- 安全交易流程：完整的订单状态管理，保障买卖双方权益
- 商品状态管理：在售/已售/下架状态切换，灵活控制商品展示
- 浏览统计功能：记录商品浏览量和收藏数，为热门推荐提供数据
- 响应式设计：适配PC和移动端，随时随地发布和浏览商品
- JWT无状态认证：安全的用户身份验证和授权机制

#### 🎬 演示流程

1. 学生注册登录 → 完善个人信息 → 获得初始信用分

3. 商品进入审核 → 审核通过后正式上架展示
4. 其他用户浏览商品 → 搜索筛选 → 查看商品详情
5. 感兴趣可收藏商品 → 与卖家在线沟通议价
6. 确认购买后下单 → 生成订单记录 → 卖家确认交易

8. **评价系统** - 双向评价、多维度评分、评价展示

#### ✨ 特色亮点

- 完整交易闭环：从需求发布→画师接单→作品交付→评价，全流程管理
- 订单状态机：待支付定金→制作中→草图提交→修改中→待付尾款→已完成，8个状态流转
- 资金安全保障：定金30% + 尾款70%分阶段支付，平台担保交易
- 修改机制：支持有限次数修改（默认2次），平衡双方利益
- 双向评价体系：客户评价画师（质量/态度/速度），画师评价客户（沟通/配合度）
- 画师认证系统：申请→审核→认证通过，确保画师质量
- 作品集展示：精美的作品展示，支持分类、标签、精选
- 平台抽成机制：8%平台手续费，自动计算画师收益
- 灵活接稿模式：支持指定画师委托和公开招标两种方式
- 余额系统：充值、消费、提现全流程管理

#### 🎬 演示流程

1. 用户注册登录 → 充值余额
2. 普通用户申请成为画师 → 提交资质信息
3. 管理员审核画师申请 → 通过后用户角色升级为ARTIST
4. 画师上传作品到作品集 → 展示个人作品
5. 客户浏览画师广场 → 查看画师详情和作品
6. 客户发布需求 → 填写需求详情、预算、截止日期
7. 画师浏览需求广场 → 选择合适需求
8. 双方协商后创建订单 → 客户支付定金（30%）
9. 画师开始创作 → 提交草图
10. 客户查看草图 → 确认或要求修改
11. 画师提交最终成品 → 客户支付尾款（70%）
12. 平台扣除8%手续费 → 转账给画师
13. 双方互相评价 → 订单完成
