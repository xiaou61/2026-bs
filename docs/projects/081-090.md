#### 💻 技术栈
**后端**
- Spring Boot 2.7.18
- MyBatis-Plus 3.5.3
- MySQL 8.0
- Redis
- JWT认证

**前端**
- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

#### 🎯 功能模块
1. **用户权限** - 管理员/审核员/申请人三角色登录与会话校验
2. **儿童档案** - 儿童基础信息、健康记录、可申请状态管理
3. **申请人档案** - 家庭信息、婚姻信息、收入情况与收养意愿维护
4. **收养申请** - 在线提交申请、材料补充、状态流转跟踪
5. **审核评估** - 材料初审、家访登记、审批意见记录
6. **匹配签约** - 匹配推荐、审批确认、收养协议登记
7. **回访管理** - 签约后回访记录、下次回访计划维护
8. **公告看板** - 公告发布展示与业务统计可视化

#### ✨ 特色亮点
- 同时提供机构后台与申请人门户，适合毕设演示完整双端流程
- 形成“儿童档案-在线申请-审核评估-匹配审批-签约回访”闭环
- MyBatis-Plus + JWT + Redis 组合实现高效率开发与登录态校验
- 儿童公开信息和申请进度分端展示，角色边界清晰
- 统计看板聚合开放儿童、申请总量、签约数量和回访情况

#### 🎬 演示流程
1. 管理员维护儿童档案、健康信息与系统公告
2. 申请人注册登录后完善资料并浏览可申请儿童信息
3. 申请人提交收养申请并补充相关材料
4. 审核员完成材料初审、家访评估与匹配审批
5. 管理员登记协议并在后续回访模块持续跟踪

#### 📊 项目规模
- **数据库表**：12张表
- **后端接口**：30+个API
- **前端页面**：17个页面
- **代码量**：约5200+行
- **功能模块**：8个核心模块

#### 🎯 技术亮点
- Spring Boot + Vue3 前后端分离
- MyBatis-Plus 快速 CRUD 与分页查询
- JWT + Redis 登录态双校验
- Element Plus 双端页面快速搭建
- ECharts 统计看板可视化展示

#### 🎓 适合场景
- 儿童福利机构信息化管理毕设
- 民生服务与流程审批类系统课程设计
- 多角色管理后台 + 门户站点综合实战
- 需要完整业务闭环和答辩演示路径的项目

### 089 - 铁路订票平台 🔥最新

#### 🏷️ 项目名称
基于SpringBoot+Vue3的铁路订票平台（管理员后台 + 调度审核 + 乘客订票端）

#### 💻 技术栈
**后端**
- Spring Boot 2.7.18
- MyBatis-Plus 3.5.5
- MySQL 8.0
- Redis
- JWT认证

**前端**
- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

#### 🎯 功能模块
1. **用户权限** - 管理员/调度员/乘客三角色登录与权限控制
2. **基础资料** - 车站信息、列车档案、车厢模板统一维护
3. **班次管理** - 出发到达站、日期时间、票价与余票排期管理
4. **在线订票** - 余票查询、Redis座位锁定、常用乘车人选座下单
5. **支付出票** - 余额支付、电子车票生成、核验流程处理
6. **订单中心** - 订单查询、取消订单、行程完成与历史记录查看
7. **退改签** - 乘客提交申请、调度员审核、退款回滚与改签记录
8. **公告统计** - 公告发布展示、销售趋势与热门车次可视化

#### ✨ 特色亮点
- 以铁路场景重构订票闭环，演示路线比普通票务系统更贴近日常生活
- 使用 Redis 做座位锁定，避免多人同时下单造成超卖
- 一个系统覆盖后台资料维护、调度审核、乘客购票三端角色
- 电子票、乘车人、退改签等业务链路完整，适合答辩展示
- 统计看板聚合销售趋势、订单总量与热门车次排行

#### 🎬 演示流程
1. 管理员维护车站、列车和车厢模板基础数据
2. 调度员创建班次并配置票价、发车时间和售卖状态
3. 乘客维护常用乘车人后进入余票选座页提交订单
4. 乘客完成余额支付，系统自动生成电子车票
5. 乘客提交退改签申请，调度员在后台审核处理并查看统计看板

#### 📊 项目规模
- **数据库表**：12张表
- **后端接口**：35+个API
- **前端页面**：16个页面
- **代码量**：约5000+行
- **功能模块**：8个核心模块

#### 🎯 技术亮点
- Spring Boot + Vue3 前后端分离
- MyBatis-Plus 快速 CRUD 与分页查询
- JWT + Redis 登录态双校验
- Redis 座位锁定机制支撑高并发选座场景
- ECharts 运营统计看板可视化展示

#### 🎓 适合场景
- 铁路订票与交通出行类毕设
- 多角色票务与审核流程系统课程设计
- 需要展示下单、支付、售后完整闭环的答辩项目
- 中型前后端分离管理系统综合实战
