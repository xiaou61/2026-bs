- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

**前端**
- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

#### 🎯 功能模块
1. **账号权限** - 管理员、团委审核员、社团负责人、社团财务员、物资管理员、社团成员六类角色登录，JWT鉴权和Redis登录态
2. **社团档案** - 社团编号、社团名称、社团类型、指导老师、成立时间和社团状态维护
3. **成员档案** - 成员编号、成员姓名、所属社团、负责人、入社时间和成员状态维护
4. **活动立项** - 活动编号、活动名称、活动类型、负责人、活动时间和立项状态维护
5. **预算申请** - 预算编号、活动名称、预算类型、申请人、申请时间和预算状态维护
6. **预算明细** - 明细编号、费用项目、费用类型、填报人、填报时间和明细状态维护
7. **审批记录** - 审批编号、审批事项、审批类型、审批人、审批时间和审批状态维护
8. **报销申请** - 报销编号、报销活动、报销类型、申请人、申请时间和报销状态维护
9. **票据归档** - 票据编号、票据事项、票据类型、归档人、归档时间和票据状态维护
10. **物资台账** - 物资编号、物资名称、物资类型、保管人、入库时间和物资状态维护
11. **物资借用** - 借用编号、借用物资、借用类型、借用人、借用时间和借用状态维护
12. **归还验收** - 验收编号、归还物资、验收类型、验收人、验收时间和验收状态维护
13. **操作日志** - 日志编号、操作模块、操作类型、操作人、操作时间和执行结果记录

#### ✨ 特色亮点
- 选题贴合高校社团财务与资产协同方向，覆盖活动立项、预算申请、报销票据、物资借用和归还验收
- 形成“社团档案-成员档案-活动立项-预算申请-预算明细-审批记录-报销申请-票据归档-物资台账-物资借用-归还验收”完整业务闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和归档完成
- 首页看板展示预算借用趋势、审批状态分布和社团财资核心指标
