基于SpringBoot+Vue3的非遗工坊课程预约与作品展销平台

#### 💻 技术栈
**后端**
- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

**前端**
- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

#### 🎯 功能模块
1. **账号权限** - ADMIN/WORKSHOP/TEACHER/CURATOR/SALES/VISITOR 六类角色登录、JWT鉴权和Redis登录态
2. **工坊档案** - 工坊编号、工坊名称、非遗门类、负责人、建档时间和工坊状态维护
3. **传承人档案** - 传承人编号、传承人姓名、技艺方向、管理人员、建档时间和档案状态维护
4. **课程目录** - 课程编号、课程名称、课程类型、授课讲师、启用时间和课程状态维护
5. **工坊排期** - 排期编号、课程名称、排期类型、授课讲师、开课时间和排期状态维护
6. **课程预约** - 预约编号、课程名称、预约类型、预约学员、预约时间和预约状态维护
7. **预约审核** - 审核编号、课程预约、审核类型、审核人员、审核时间和审核状态维护
8. **课程签到** - 签到编号、课程排期、签到类型、记录人员、签到时间和签到状态维护
9. **作品档案** - 作品编号、作品名称、作品类型、创作人员、入库时间和作品状态维护
10. **作品展销** - 展销编号、作品名称、展销类型、展陈人员、上架时间和展销状态维护
11. **展销订单** - 订单编号、作品名称、订单类型、下单用户、下单时间和订单状态维护
12. **展销结算** - 结算编号、结算周期、结算类型、财务人员、结算时间和结算状态维护
13. **操作日志** - 日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

#### ✨ 特色亮点
- 选题贴合非遗文创服务方向，覆盖工坊建档、传承人课程、课程预约审核、课程签到、作品展销、订单和结算
- 形成“工坊档案-传承人档案-课程目录-工坊排期-课程预约-预约审核-课程签到-作品档案-作品展销-展销订单-展销结算”完整业务闭环
- 六类角色默认账号：admin、workshop、teacher、curator、sales、visitor，统一密码123456
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和状态完成
- 首页看板展示核心指标、近7日趋势和状态分布

## 快速启动
1. 导入数据库脚本（每个项目的 `sql` 文件夹）
2. 修改 `application.yml` 配置数据库连接
3. 后端：`mvn spring-boot:run` 或直接运行主类
4. 前端：`npm install && npm run dev`（Vue项目）
5. 访问项目地址（默认8080端口）
