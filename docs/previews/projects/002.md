# 002 - 在线选课与成绩管理系统

## 项目信息

- 项目编号：`002`
- 组件类型：`backend, frontend`
- 后端入口：`http://127.0.0.1:18080`
- 前端入口：`http://127.0.0.1:3002`
- 账号来源：002-backend\README.md
- 已收录截图：`20` 张

## 默认账号

- `管理员`：`admin` / `123456`
- `学生`：`student1` / `123456`
- `学生`：`student2` / `123456`
- `教师`：`teacher1` / `123456`
- `教师`：`teacher2` / `123456`

## 预览备注

- 当前预览环境实际使用 `backend:18080`、`frontend:3002`。
- 本机 `8080` 已被其他 Java 服务占用，因此本项目后端改挂到 `18080`，前端通过 `PREVIEW_API_TARGET` 转发到该端口。
- 默认种子数据里的选课时间停留在 `2024-09`，在当前环境下学生真实选课会被后端判定为“不在选课时间内”。
- 为完整覆盖教师和学生页面，本轮预览补入了 1 条选课记录，并由教师账号真实提交了 1 条成绩记录。
