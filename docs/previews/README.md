# 项目预览说明

本目录用于存放毕业设计项目的运行预览结果。

## 目录结构

- `groups/`
  - 每 10 个项目一个索引页
- `projects/`
  - 每个项目一个详览页
- `assets/`
  - 项目截图资源
- `runtime/`
  - 项目运行清单和运行时信息

## 查看方式

1. 先看根目录 `readme.md` 中的“项目预览”入口。
2. 进入对应分组页，例如 `001-010.md`。
3. 再进入单项目详览页查看多图预览。

## 推进策略

- 按项目编号顺序推进
- 当前从 `001` 开始
- 每个项目尽量覆盖多角色、多页面和关键功能
