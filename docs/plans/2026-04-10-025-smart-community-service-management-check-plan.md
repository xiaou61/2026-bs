# 025 智慧社区综合服务管理平台巡检计划

## 巡检目标

围绕 `025` 项目完成以下事项：

1. 核对 PRD、README、配置、SQL 与前后端实现的一致性
2. 验证 JDK 17 / Maven / Node 环境下是否可测试、可构建、可启动
3. 对明确问题直接修复，并完成复测
4. 输出单项目检查报告并同步总台账

## 当前已知范围

- 后端：`025-backend`
- 前端：`025-frontend`
- PRD：`025-backend/PRD/product_requirements_document.md`

## 执行顺序

1. 核对后端依赖、配置、SQL、控制器、服务层
2. 核对前端路由、请求层、页面与构建配置
3. 运行后端测试 / 编译与前端构建
4. 启动前后端并抽测登录、业主、缴费、报修、公告、访客和车位链路
5. 对构建错误、启动错误、前后端断链做最小闭环修复
6. 回填巡检文档和总台账
