# 083 ElderCare Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 完成基于B/S的老年人体检管理系统083项目，交付前后端可联动的管理端代码与初始化SQL。

**Architecture:** 后端采用Spring Boot + MyBatis-Plus + Redis + JWT，按controller/service/mapper/entity分层。前端采用Vue3 + Element Plus + Pinia + Router，按页面模块化实现统一CRUD交互。

**Tech Stack:** Spring Boot 2.7.14, MyBatis-Plus 3.5.3, MySQL 8.0, Redis, Vue 3.4.0, Element Plus 2.4.4, Axios, ECharts
