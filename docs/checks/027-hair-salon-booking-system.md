- 当前默认端口：
  - 后端：`8027`
  - 前端开发端口：`3027`

## 3. 本轮已修复问题

1. 修复 `AppointmentService` 非法方法名，解决后端 JDK 17 编译失败。
2. 修复多个控制器 `Result<Void>` 与字符串返回值不兼容的问题，恢复后端编译。
3. 将 `mybatis-plus-boot-starter` 切换为 `mybatis-plus-spring-boot3-starter`，解决 Spring Boot 3.2 启动时报 `factoryBeanObjectType` 异常。
4. 将默认运行端口改为后端 `8027`、前端开发端口 `3027`，避免与其他项目冲突。
5. 新增 H2 运行时依赖，并将默认数据源从本地 MySQL 切换为项目自带 H2 文件库。
6. 新增 `schema-h2.sql` 与 `data-h2.sql`，实现无外部数据库的默认初始化。
7. 补充 MyBatis-Plus 分页拦截器，修复分页接口 `records` 有数据但 `total/pages=0` 的问题。
8. 补充 `MetaObjectHandler`，修复新增预约等记录的 `createdAt/updatedAt` 为空导致排序异常的问题。
9. 修正默认种子用户密码哈希，并补充一条未支付订单种子，便于支付与评价链路抽测。
