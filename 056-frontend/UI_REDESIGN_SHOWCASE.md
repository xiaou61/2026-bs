# 056-frontend UI 美化展示

## 项目信息
- **项目名称**: 短文写作竞赛管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 文采橙
- **美化完成时间**: 2026-05-29

## 设计理念

采用"文采橙"设计理念，以橙色和金色为主色调，体现写作的热情、创意和文采。通过笔、书本等装饰元素，营造出文采飞扬、妙笔生花的写作竞赛氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 文采橙 | #FF6D00 | 主色调 |
| 深橙 | #E65100 | 辅助色 |
| 金色 | #FFB300 | 强调色 |
| 暖橙 | #FF9100 | 装饰色 |
| 红橙 | #BF360C | 文字色 |

## 设计特点

### 登录页面
- 橙色渐变背景
- 写作装饰元素浮动
- 笔✍️Logo弹跳动画
- 流光渐变顶部条

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳效果 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

056-frontend采用"文采橙"设计理念，通过橙色和金色的搭配，营造出文采飞扬的写作竞赛氛围。笔和书本装饰元素突出了写作竞赛的特色。
