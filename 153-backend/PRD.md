# 校园二手物品寄卖与信用评价系统 PRD

## 项目概述
本系统面向校园二手物品交易场景，提供寄卖上架、上架审核、担保订单、支付交接、信用评价、违约记录和纠纷申诉的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 平台运营：物品分类、学生档案、上架审核、担保订单、公告和统计看板维护
- 寄卖学生：寄卖物品、担保订单、交接确认和信用评价维护
- 购买学生：担保订单、担保支付、交接确认和信用评价维护
- 信用仲裁员：纠纷申诉、违约记录和信用仲裁处理

## 功能模块
- 物品分类：分类编码、分类名称、适用品类、维护人、更新时间和启用状态维护
- 学生档案：学号、姓名、学院班级、联系方式、认证时间和账号状态维护
- 寄卖物品：寄卖编号、物品名称、物品类型、寄卖人、上架时间和寄卖状态维护
- 上架审核：审核编号、寄卖物品、审核类型、审核人、审核时间和审核状态维护
- 担保订单：订单编号、寄卖物品、交易类型、买家、下单时间和订单状态维护
- 担保支付：支付编号、关联订单、支付渠道、付款人、支付时间和支付状态维护
- 交接确认：交接编号、关联订单、交接方式、确认人、交接时间和确认状态维护
- 信用评价：评价编号、评价对象、评价类型、评价人、评价时间和处理状态维护
- 违约记录：违约编号、关联订单、违约类型、责任人、记录时间和处置状态维护
- 纠纷申诉：申诉编号、关联订单、申诉类型、申诉人、提交时间和处理状态维护
- 平台公告：公告编号、公告标题、公告类型、发布人、发布时间和公告状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

## 技术栈
- 后端：Spring Boot 2.7.18、MyBatis 注解 SQL、PageHelper、MySQL、Redis、JWT
- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
- 数据库：campus_resale_153
