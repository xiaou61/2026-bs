# 127-frontend UI 美化展示

## 项目信息
- **项目名称**: 企业碳排放核算与减排任务管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 碳中和绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"碳中和绿"设计理念，以绿色为主色调，体现碳中和的环保、可持续和未来感。通过地球、绿叶等装饰元素，营造出碳中和、绿色未来的环保管理氛围。

| 元素 | 动画 | 说明 |
|------|------|------|
