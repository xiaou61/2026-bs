# 033-frontend UI 美化展示

## 项目信息
- **项目名称**: 婚纱写真馆管理系统
- **技术栈**: React + Ant Design + Vite
- **设计主题**: 浪漫粉
- **美化完成时间**: 2026-05-29

## 设计理念

采用"浪漫粉"设计理念，以粉色和金色为主色调，体现婚纱摄影的浪漫和优雅。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 浪漫粉 | #E91E63 | 主色调 |
| 优雅金 | #FFB300 | 辅助色 |
| 浪漫紫 | #9C27B0 | 强调色 |
| 背景色 | #FCE4EC | 页面背景 |
