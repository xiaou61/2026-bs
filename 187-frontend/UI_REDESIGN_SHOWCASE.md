# 187-frontend UI 美化展示

## 项目信息
- **项目名称**: 旅行社团建行程报名与费用分摊管理平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 旅行蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"旅行蓝"设计理念，以天蓝色为主色调，体现旅行的自由、清新和探索感。通过飞机、地球等装饰元素，营造出世界那么大、我们去看看的旅行氛围。

## 设计特点

### 登录页面
- 蓝色渐变背景
- 旅行装饰元素浮动

- 流光渐变顶部条

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | fly | 飞行轨迹 |
| 顶部条 | travelGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

187-frontend采用"旅行蓝"设计理念，通过天蓝的主色调，营造出自由、清新的旅行氛围。飞机飞行动画突出了旅行社团建行程的特色。
