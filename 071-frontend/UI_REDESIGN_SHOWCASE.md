# 071-frontend UI 美化展示

## 项目信息
- **项目名称**: 共享单车管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 环保绿
- **美化完成时间**: 2026-05-29

## 设计理念

采用"环保绿"设计理念，以绿色为主色调，体现共享单车的环保、低碳和健康理念。通过自行车、绿叶等装饰元素，营造出绿色出行、低碳生活的共享出行氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳缩放 |
| 顶部条 | gradientMove | 流光渐变 |
