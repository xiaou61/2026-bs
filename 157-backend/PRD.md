# 物流园区车辆入场预约与道口调度平台 PRD

## 项目概述
本系统面向物流园区车辆入场与月台调度场景，提供承运商预约、车辆档案、司机档案、时段计划、门岗核验、排队叫号、道口资源、道口分配、装卸任务和周转统计的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 调度主管：承运商档案、车辆档案、时段计划、道口资源和周转统计维护
- 门岗核验员：门岗核验、车辆放行、排队叫号和异常入场处理
- 场内调度员：时段计划、排队叫号、道口资源、道口分配和装卸任务维护
- 承运商代表：入场预约、排队叫号、道口分配和装卸任务查看

## 功能模块
- 承运商档案：承运商编号、承运商名称、业务类型、联系人、合作时间和合作状态维护
- 车辆档案：车辆编号、车牌号码、车辆类型、司机姓名、备案时间和车辆状态维护
- 司机档案：司机编号、司机姓名、证件类型、承运商、备案时间和司机状态维护
- 入场预约：预约编号、预约车辆、预约类型、预约人、预约时间和预约状态维护
- 时段计划：时段编号、时段名称、作业类型、计划人、计划时间和开放状态维护
- 门岗核验：核验编号、入场车辆、核验类型、核验人、核验时间和核验状态维护
- 排队叫号：叫号编号、排队车辆、作业类型、叫号人、叫号时间和排队状态维护
- 道口资源：道口编号、道口名称、道口类型、负责人、启用时间和道口状态维护
- 道口分配：分配编号、分配车辆、作业类型、调度人、分配时间和分配状态维护
- 装卸任务：任务编号、货物批次、装卸类型、负责人、作业时间和任务状态维护
- 周转统计：统计编号、统计车辆、统计类型、统计人、统计周期和统计状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护
