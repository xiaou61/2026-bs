# 校园社团活动预算报销与物资借用系统 PRD

## 项目概述
本系统面向高校学生社团活动预算、报销票据与物资借还协同场景，提供社团档案、成员档案、活动立项、预算申请、预算明细、审批记录、报销申请、票据归档、物资台账、物资借用和归还验收的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 团委审核员：社团档案、活动立项、预算审批和经费使用复核
- 社团负责人：活动立项、预算申请、报销申请和成员协同维护
- 社团财务员：预算明细、报销申请、票据归档和费用核对维护
- 物资管理员：物资台账、物资借用、归还验收和库存状态维护
- 社团成员：活动申请、物资借用、归还反馈和报销进度查看

## 功能模块
- 社团档案：社团编号、社团名称、社团类型、指导老师、成立时间和社团状态维护
- 成员档案：成员编号、成员姓名、所属社团、负责人、入社时间和成员状态维护
- 活动立项：活动编号、活动名称、活动类型、负责人、活动时间和立项状态维护
- 预算申请：预算编号、活动名称、预算类型、申请人、申请时间和预算状态维护
- 预算明细：明细编号、费用项目、费用类型、填报人、填报时间和明细状态维护
- 审批记录：审批编号、审批事项、审批类型、审批人、审批时间和审批状态维护
- 报销申请：报销编号、报销活动、报销类型、申请人、申请时间和报销状态维护
- 票据归档：票据编号、票据事项、票据类型、归档人、归档时间和票据状态维护
- 物资台账：物资编号、物资名称、物资类型、保管人、入库时间和物资状态维护
- 物资借用：借用编号、借用物资、借用类型、借用人、借用时间和借用状态维护
- 归还验收：验收编号、归还物资、验收类型、验收人、验收时间和验收状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

## 技术栈
- 后端：Spring Boot 2.7.18、MyBatis 注解 SQL、PageHelper、MySQL、Redis、JWT
- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
- 数据库：club_finance_160
