# 079-frontend UI 美化展示

## 项目信息
- **项目名称**: 计算机学院校友网
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 校友紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"校友紫"设计理念，以紫色为主色调，体现校友会的学术氛围、情谊和尊贵感。通过毕业帽、学校等装饰元素，营造出桃李满天下、校友情谊长的校友网络氛围。

## 动画效果

| 元素 | 动画 | 说明 |
