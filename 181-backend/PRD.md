# 社区儿童托管签到接送与安全告警系统 PRD

## 项目概述
本系统面向社区儿童托管、入园签到、接送授权、接送核验、安全告警、健康晨检、活动安排和家长通知场景，提供托管班级、儿童档案、监护人档案、签到记录、接送授权、接送记录、安全告警、健康晨检、托管活动、家长通知和事件跟进的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 中心管理员：维护托管班级、儿童档案、接送规则和通知配置
- 托管老师：执行儿童签到、接送核验、活动安排和日常记录
- 家长：查看儿童档案、授权接送人员、接收通知并确认接送记录
- 安全员：处理接送核验、安全告警和事件跟进
- 保健老师：维护健康晨检、异常提醒和儿童健康记录

## 功能模块
- 托管班级：班级编号、班级名称、托管类型、负责老师、开班时间和班级状态维护
- 儿童档案：儿童编号、儿童姓名、托管类型、监护家长、入托时间和档案状态维护
- 监护人档案：监护编号、监护姓名、关系类型、关联儿童、登记时间和监护状态维护
- 签到记录：签到编号、托管儿童、签到类型、签到老师、签到时间和签到状态维护
- 接送授权：授权编号、授权儿童、授权类型、接送人员、有效时间和授权状态维护
- 接送记录：接送编号、托管儿童、接送类型、接送人员、接送时间和接送状态维护
- 安全告警：告警编号、告警位置、告警类型、处理人员、告警时间和告警状态维护
- 健康晨检：晨检编号、托管儿童、健康类型、保健老师、晨检时间和健康状态维护
- 托管活动：活动编号、活动名称、活动类型、负责老师、活动时间和活动状态维护
- 家长通知：通知编号、通知标题、通知类型、发布人员、发布时间和通知状态维护
- 事件跟进：跟进编号、安全事件、跟进类型、跟进人员、跟进时间和跟进状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

## 技术栈
- 后端：Spring Boot 2.7.18、MyBatis 注解 SQL、PageHelper、MySQL、Redis、JWT
- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
