# 021-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园交易平台
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 社交紫
- **美化完成时间**: 2026-05-29

## 设计理念

采用"社交紫"设计理念，以紫色和粉色为主色调，体现社交交易的活力和亲切感。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 社交紫 | #9C27B0 | 主色调 |
| 活力粉 | #E91E63 | 辅助色 |
| 清新蓝 | #2196F3 | 强调色 |
| 背景色 | #F5F5F5 | 页面背景 |

021-frontend采用"社交紫"设计理念，通过紫色和粉色的搭配，营造出活力、亲切的社交交易氛围。聊天和购物装饰元素突出了校园交易的功能特色。
