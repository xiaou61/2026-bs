# 078-frontend UI 美化展示

## 项目信息
- **项目名称**: 网上团购系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 团购橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"团购橙"设计理念，以橙色和金色为主色调，体现团购的热情、优惠和活力。通过购物车、礼物等装饰元素，营造出拼团优惠、省钱购物的电商氛围。

## 颜色方案

|------|------|------|

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳缩放 |
| 顶部条 | groupGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
