# 013-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园二手交易平台
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 环保绿
- **美化完成时间**: 2026-05-29

## 设计理念

采用"环保绿"设计理念，以绿色和橙色为主色调，体现环保、循环利用的绿色交易理念。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 环保绿 | #00C853 | 主色调 |
| 阳光橙 | #FF9800 | 辅助色 |
| 天空蓝 | #03A9F4 | 强调色 |
| 背景色 | #F5F5F5 | 页面背景 |
