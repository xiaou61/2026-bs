# 155-frontend UI 美化展示

## 项目信息
- **项目名称**: 社区党建活动报名与积分激励平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 党建红
- **美化完成时间**: 2026-05-30

## 设计理念

采用"党建红"设计理念，以红色为主色调，体现党建工作的庄重、严肃和使命感。通过星星、奖章等装饰元素，营造出不忘初心、牢记使命的党建管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 党建红 | #D32F2F | 主色调 |
| 深红 | #C62828 | 辅助色 |
| 暗红 | #B71C1C | 强调色 |
| 浅红 | #E53935 | 装饰色 |
| 淡红 | #FFEBEE | 背景色 |

## 设计特点

### 登录页面
- 红色渐变背景

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | partyGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

155-frontend采用"党建红"设计理念，通过红色的主色调，营造出庄重、严肃的党建管理氛围。星星装饰元素突出了社区党建活动的使命感。
