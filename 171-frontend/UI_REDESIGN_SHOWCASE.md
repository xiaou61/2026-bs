# 171-frontend UI 美化展示

## 项目信息
- **项目名称**: 应急物资储备盘点与调拨审批平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 应急红
- **美化完成时间**: 2026-05-30

## 设计理念

采用"应急红"设计理念，以红色为主色调，体现应急物资管理的紧迫性、重要性和保障感。通过包裹、消防车等装饰元素，营造出应急储备、保障有力的物资管理氛围。
