# 105-frontend UI 美化展示

## 项目信息
- **项目名称**: API接口文档生成与测试用例管理平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: API紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"API紫"设计理念，以紫色为主色调，体现API管理的科技感、专业性和现代感。通过链接、测试等装饰元素，营造出API驱动、质量保障的开发管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | apiGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
