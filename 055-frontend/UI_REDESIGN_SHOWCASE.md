# 055-frontend UI 美化展示

## 项目信息
- **项目名称**: 企业OA管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 商务蓝
- **美化完成时间**: 2026-05-29

## 设计理念

采用"商务蓝"设计理念，以深蓝色为主色调，体现企业办公的专业性、可靠性和效率。通过办公、图表等装饰元素，营造出高效、智慧的企业办公氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|

| 天蓝 | #2196F3 | 强调色 |

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |

| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
