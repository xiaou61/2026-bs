# 160-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园社团活动预算报销与物资借用系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 社团橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"社团橙"设计理念，以橙色和金色为主色调，体现社团活动的活力、热情和青春感。通过面具、礼物等装饰元素，营造出青春社团、精彩无限的校园社团管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 社团橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳缩放 |
| 顶部条 | clubGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
