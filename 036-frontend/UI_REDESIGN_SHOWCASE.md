# 036-frontend UI 美化展示

## 项目信息
- **项目名称**: 公益项目捐赠系统
- **技术栈**: Vue 3 + Element Plus + Pinia
- **设计主题**: 爱心红
- **美化完成时间**: 2026-05-29

## 设计理念

采用"爱心红"设计理念，以红色和粉色为主色调，体现公益事业的温暖、关爱和希望。通过心形装饰、浮动粒子和心跳动画，营造出充满爱心的公益捐赠氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 背景粒子 | floatParticle | 上下浮动 |
| 心形Logo | heartbeat | 心跳效果 |
| 顶部条 | shimmer | 流光渐变 |
| 输入框 | focus | 聚焦阴影 |
| 按钮 | hover | 悬浮上移 |
| 注册链接 | hover | 滑动效果 |
