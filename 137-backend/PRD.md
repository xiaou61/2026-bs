# 大学生创新创业项目孵化管理平台 PRD

## 项目定位
大学生创新创业项目孵化管理平台面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

## 用户角色
系统管理员、项目导师、创业学生、评审专家

## 功能模块
- 孵化项目：项目编号、项目名称、所属赛道、团队名称、孵化阶段和状态维护
- 导师档案：导师编号、导师姓名、擅长方向、所属学院、可辅导项目数和状态维护
- 团队档案：团队编号、团队名称、负责人、成员人数、联系邮箱和状态维护
- 项目申报：申报编号、项目名称、申报人、申报阶段、提交时间和状态维护
- 孵化计划：计划编号、项目名称、孵化目标、计划周期、负责导师和状态维护
- 导师辅导：辅导编号、项目名称、导师姓名、辅导主题、辅导时间和状态维护
- 路演活动：路演编号、项目名称、活动主题、路演日期、场地名称和状态维护
- 路演评分：评分编号、项目名称、评委姓名、评分结果、评分时间和状态维护
- 经费记录：经费编号、项目名称、经费类型、金额、记录时间和状态维护
- 里程碑任务：任务编号、项目名称、任务阶段、负责人、截止日期和状态维护
- 成果展示：成果编号、项目名称、成果名称、成果类型、展示时间和状态维护
- 孵化通知：通知编号、项目名称、通知类型、通知内容、接收人和状态维护
- 操作日志：关键模块操作人、动作、对象、详情和结果记录
