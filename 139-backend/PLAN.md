# 企业培训学习路径与能力画像系统 开发计划

## 实施范围
- 后端工程：实体、Mapper、Service、Controller、统一响应、JWT 登录、Redis Token 校验、统计接口
- 前端工程：登录页、布局菜单、通用数据页、模块页面、统计看板、接口封装和路由守卫
- 数据脚本：MySQL 建库建表、初始化账号和演示业务数据

## 模块拆分
- 培训项目：项目编号、项目名称、培训主题、适用部门、项目周期和状态维护
- 课程目录：课程编号、课程名称、课程类别、主讲讲师、课时数量和状态维护
- 学员档案：学员编号、学员姓名、所属部门、岗位名称、联系邮箱和状态维护
- 学习路径：路径编号、路径名称、适用岗位、阶段数量、发布时间和状态维护
- 学习任务：任务编号、课程名称、指派对象、截止日期、完成进度和状态维护
- 课程报名：报名编号、课程名称、学员姓名、报名时间、学习方式和状态维护
- 考核考试：考试编号、课程名称、考试方式、考试日期、监考讲师和状态维护
- 考试成绩：成绩编号、考试名称、学员姓名、考试分数、成绩时间和状态维护
- 技能标签：标签编号、标签名称、技能分类、适用岗位、标签等级和状态维护
- 能力画像：画像编号、学员姓名、能力维度、当前等级、评估时间和状态维护
- 认证记录：认证编号、学员姓名、证书名称、认证机构、获取时间和状态维护
- 学习提醒：提醒编号、学员姓名、提醒类型、提醒内容、发送时间和状态维护
- 操作日志：关键模块操作人、动作、对象、详情和结果记录

## 验收标准
- 具备 PRD、PLAN、后端、前端和 SQL 初始化脚本
- 后端端口、前端端口、数据库名、Redis DB 与项目编号一致
- 静态结构达到 14 张表、14 个实体、14 个 Mapper、16 个 Controller、17 个前端页面
- 无旧项目包名、标题、端口、数据库名残留
