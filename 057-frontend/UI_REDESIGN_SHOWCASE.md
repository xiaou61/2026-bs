# 057-frontend UI 美化展示

## 项目信息
- **项目名称**: 招生管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 学府紫
- **美化完成时间**: 2026-05-29

## 设计理念

采用"学府紫"设计理念，以紫色为主色调，体现高等教育的学术氛围和庄重感。通过毕业帽、学校等装饰元素，营造出公平公正、择优录取的招生管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |

| 亮紫 | #AB47BC | 强调色 |
| 暗紫 | #4A148C | 文字色 |
| 浅紫 | #CE93D8 | 装饰色 |

- 紫色渐变背景

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳效果 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

057-frontend采用"学府紫"设计理念，通过紫色的主色调，营造出庄重、学术的招生管理氛围。毕业帽装饰元素突出了招生录取的教育特色。
