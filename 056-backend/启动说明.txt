056 短文写作竞赛管理小程序启动说明

一、后端启动

1. 进入后端目录：

   cd 056-backend

2. 默认使用 H2 内存数据库启动：

   mvn spring-boot:run

3. 后端访问地址：

   http://localhost:8056

4. H2 控制台：

   http://localhost:8056/h2-console

5. H2 连接信息：

   JDBC URL：jdbc:h2:mem:writing_competition
   用户名：sa
   密码：留空

二、管理端启动

1. 进入管理端目录：

   cd 056-frontend

2. 安装依赖：

   npm install --no-audit --no-fund

3. 启动开发服务：

   npm run dev

4. 管理端访问地址：

   http://localhost:3056

三、小程序端

1. 小程序目录：

   056-miniapp

2. 请求地址：

   http://localhost:8056

3. 可使用 HBuilderX 或 uni-app 工具链打开运行。

四、默认账号

默认密码均为 123456。

管理员：admin / 123456
评委：judge / 123456
评委：judge2 / 123456
参赛者：user / 123456
参赛者：user2 / 123456
参赛者：user3 / 123456

五、MySQL 模式

默认演示环境不需要 MySQL。如需切换 MySQL，请先创建 writing_competition 数据库并导入 sql/init.sql，然后运行：

   mvn spring-boot:run -Dspring-boot.run.profiles=mysql

MySQL 配置文件：

   src/main/resources/application-mysql.yml

六、验证命令

后端测试：

   mvn test

管理端构建：

   cd 056-frontend
   npm run build

本轮已修复默认 MySQL 和 8080 强依赖、H2 自举、分页方言、游客分类接口拦截、密码响应泄露、基础权限缺失、评分标准保存异常、管理端和小程序端口配置、以及小程序缺失静态资源引用等问题。
