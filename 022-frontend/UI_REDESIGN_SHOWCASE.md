# 022-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园自习室座位预约系统
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 宁静蓝
- **美化完成时间**: 2026-05-29

## 设计理念

采用"宁静蓝"设计理念，以蓝色和橙色为主色调，体现学习的宁静和专注。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 宁静蓝 | #5C6BC0 | 主色调 |
| 温暖橙 | #FF7043 | 辅助色 |
| 清新绿 | #66BB6A | 强调色 |
| 背景色 | #F5F5F5 | 页面背景 |
