# 工厂危险作业审批与监护巡检管理系统 PRD

## 项目概述
本系统面向化工、制造、能源等工厂危险作业场景，提供作业票申请、分级审批、安全交底、现场监护、气体检测和隐患整改闭环的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 安全主管：作业区域、风险源台账、作业票、隐患闭环、应急预案和统计看板维护
- 审批负责人：作业票审批、审批链路流转和审批意见维护
- 现场监护人：监护安排、监护记录、气体检测和隐患上报维护
- 作业申请人：作业票申请、安全交底确认和整改反馈提交

## 功能模块
- 作业区域：区域编码、区域名称、危险等级、负责人、管控时段和区域状态维护
- 风险源台账：风险编号、风险源名称、风险类型、责任人、巡检周期和管控状态维护
- 作业人员档案：工号、姓名、资质等级、所属班组、有效期和资质状态维护
- 作业票申请：作业票号、作业项目、作业类型、申请人、作业时间和申请状态维护
- 审批链路：审批编号、作业票号、审批节点、审批人、审批时间和审批状态维护
- 安全交底：交底编号、作业项目、交底类型、交底人、交底时间和确认状态维护
- 监护安排：安排编号、作业票号、监护类型、监护人、监护时段和监护状态维护
- 监护记录：记录编号、作业票号、监护项目、记录人、记录时间和现场状态维护
- 隐患闭环：隐患编号、隐患位置、隐患类型、发现人、整改期限和整改状态维护
- 气体检测：检测编号、作业票号、检测类型、检测人、检测时间和检测状态维护
- 应急预案：预案编号、预案名称、适用场景、负责人、演练时间和启用状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护
