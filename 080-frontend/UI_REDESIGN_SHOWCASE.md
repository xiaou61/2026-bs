# 080-frontend UI 美化展示

## 项目信息
- **项目名称**: 贫困地区儿童资助网站
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 爱心粉
- **美化完成时间**: 2026-05-30

## 设计理念

采用"爱心粉"设计理念，以粉色和玫红色为主色调，体现公益事业的温暖、关爱和希望。通过爱心、彩虹等装饰元素，营造出爱心助学、点亮希望的公益氛围。

## 设计特点

### 登录页面
- 粉色渐变背景
- 爱心装饰元素浮动
- 爱心❤️Logo心跳动画
- 流光渐变顶部条
- 爱心公益标语

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | heartbeat | 心跳效果 |
| 顶部条 | charityGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

080-frontend采用"爱心粉"设计理念，通过粉色和玫红色的搭配，营造出温暖、关爱的公益氛围。爱心心跳动画突出了贫困地区儿童资助的公益特色。
