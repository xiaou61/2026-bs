# 校园体育赛事报名编排与裁判评分系统 PRD

## 项目概述
本系统面向高校体育赛事组织与裁判评分场景，提供体育赛事、参赛队伍、运动员档案、赛事报名、报名分组、赛程编排、场地资源、裁判指派、裁判评分、成绩公示和申诉复核的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 赛事组织员：体育赛事、报名分组、赛程编排、场地资源和成绩公示维护
- 队伍教练：参赛队伍、运动员档案、赛事报名、赛程确认和成绩申诉维护
- 参赛学生：赛事报名、赛程查看、成绩公示和申诉进度查看
- 裁判员：裁判指派、赛事执裁、裁判评分和赛果确认维护
- 仲裁委员：评分复核、成绩公示、申诉复核和争议处置维护

## 功能模块
- 体育赛事：赛事编号、赛事名称、项目类型、主办老师、开赛时间和赛事状态维护
- 参赛队伍：队伍编号、队伍名称、参赛组别、队伍教练、报名时间和队伍状态维护
- 运动员档案：学号、学生姓名、所属队伍、指导教练、登记时间和参赛状态维护
- 赛事报名：报名编号、报名项目、参赛队伍、报名负责人、提交时间和报名状态维护
- 报名分组：分组编号、分组名称、竞赛项目、分组负责人、抽签时间和分组状态维护
- 赛程编排：赛程编号、比赛名称、竞赛轮次、编排人员、比赛时间和赛程状态维护
- 场地资源：场地编号、场地名称、场地类型、场馆管理员、开放时间和场地状态维护
- 裁判指派：指派编号、比赛项目、裁判岗位、裁判员、指派时间和指派状态维护
- 裁判评分：评分编号、比赛对象、评分项目、裁判员、评分时间和评分状态维护
- 成绩公示：公示编号、公示项目、名次组别、发布人员、公示时间和公示状态维护
- 申诉复核：申诉编号、申诉赛事、申诉类型、复核委员、申诉时间和复核状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

## 技术栈
- 后端：Spring Boot 2.7.18、MyBatis 注解 SQL、PageHelper、MySQL、Redis、JWT
- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
