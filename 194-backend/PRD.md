# 工业园区危废暂存与转运联动监管平台 PRD

## 项目概述
本系统面向工业园区管委会、危废暂存仓库、转运单位、环保巡查人员和产废企业，提供园区企业、危废类别、暂存设施、入库登记、暂存巡检、转运联单、车辆调度、称重记录、处置交接、风险预警、台账审计和操作日志的一体化监管能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 园区监管：维护园区企业、危废类别、暂存设施和监管台账
- 暂存管理员：登记入库、执行暂存巡检、配合称重和风险预警处置
- 转运调度：维护转运联单、车辆调度、称重记录和处置交接
- 环保巡查：核查暂存巡检、风险预警、处置交接和台账审计
- 产废企业：维护企业相关危废信息，提交入库登记并查看转运进度

## 功能模块
- 园区企业：企业编号、企业名称、行业类别、环保联系人、入驻时间和监管状态维护
- 危废类别：危废编号、危废名称、危废类别、责任人员、建档时间和风险状态维护
- 暂存设施：设施编号、设施名称、设施类型、管理人员、启用时间和设施状态维护
- 入库登记：入库编号、危废名称、来源企业、登记人员、入库时间和入库状态维护
- 暂存巡检：巡检编号、暂存设施、巡检类型、巡检人员、巡检时间和巡检状态维护
- 转运联单：联单编号、危废批次、转运类型、调度人员、计划时间和联单状态维护
- 车辆调度：调度编号、转运联单、车辆类型、驾驶人员、调度时间和调度状态维护
- 称重记录：称重编号、转运批次、称重类型、称重人员、称重时间和称重状态维护
- 处置交接：交接编号、转运联单、接收单位、交接人员、交接时间和交接状态维护
- 风险预警：预警编号、危废批次、预警类型、处理人员、预警时间和预警状态维护
- 台账审计：审计编号、审计对象、审计类型、审计人员、审计时间和审计状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
