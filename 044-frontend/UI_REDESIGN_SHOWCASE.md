# 044-frontend UI 美化展示

## 项目信息
- **项目名称**: 民宿预订系统
- **技术栈**: Vue 3 + TypeScript + Element Plus
- **设计主题**: 旅行蓝
- **美化完成时间**: 2026-05-29

## 设计理念

采用"旅行蓝"设计理念，以蓝色和天蓝色为主色调，体现旅行的自由、清新和远方。通过房屋、树木、山脉等装饰元素，营造出诗意栖居的民宿预订氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|

| 深蓝 | #0288D1 | 文字色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳效果 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
