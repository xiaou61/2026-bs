# 001-frontend UI 重新设计方案

## 项目信息
- **项目名称**: 校园事务管理系统
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 清新校园风

## 设计理念
采用"清新校园"设计理念，以蓝绿色为主色调，营造轻松、现代的校园管理氛围。

## 颜色方案
- **主色调**: #2E7D32 (校园绿)
- **辅助色**: #1565C0 (天空蓝)
- **强调色**: #FF6F00 (活力橙)
- **背景色**: #F5F7FA (浅灰)
- **文字色**: #2C3E50 (深灰)

## 字体方案
- **主字体**: 'PingFang SC', 'Microsoft YaHei', sans-serif
- **标题字体**: 'Noto Sans SC', sans-serif

## 组件美化方案

### 1. 登录页面
- 添加校园背景图片或渐变
- 使用毛玻璃效果的登录卡片
- 添加校园元素图标

### 2. 侧边栏
- 使用深绿色渐变背景
- 添加校园图标元素
- 优化菜单项悬停效果

### 3. 头部导航
- 添加用户头像
- 优化下拉菜单样式
- 添加通知图标

### 4. 仪表盘
- 使用卡片式布局
- 添加数据可视化图表
- 优化统计卡片样式

### 5. 列表页面
- 使用斑马纹表格
- 优化分页器样式
- 添加搜索和筛选区域

### 6. 表单页面
- 使用现代化表单设计
- 添加输入验证提示
- 优化按钮样式

## 实施步骤

1. 创建全局样式变量文件
2. 修改登录页面样式
3. 修改侧边栏和头部导航样式
4. 修改仪表盘页面样式
5. 修改列表页面样式
6. 修改表单页面样式
7. 添加动画效果
8. 测试和调整

## 预期效果
- 更现代化的界面设计
- 更好的用户体验
- 统一的视觉风格
- 响应式设计支持
