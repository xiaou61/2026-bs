# 城市共享充电宝投放巡检与收益结算系统 PRD

## 项目概述
本系统面向城市共享充电宝运营企业、点位商户、巡检人员、财务结算人员和合作商户，提供投放点位、设备柜档案、充电宝档案、点位投放、设备巡检、故障维修、异常回收、租借订单、商户收益、收益结算、库存调拨和操作日志的一体化管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 运营管理员：维护投放点位、设备柜档案、充电宝档案和点位投放计划
- 点位商户：查看设备柜、充电宝状态、巡检维修和点位运营记录
- 巡检人员：处理设备巡检、故障维修、异常回收和库存调拨
- 财务结算：核对租借订单、商户收益和收益结算闭环
- 合作商户：查看点位收益、结算记录和订单分成

## 功能模块
- 投放点位：点位编号、点位名称、点位类型、负责人、建档时间和点位状态维护
- 设备柜档案：设备柜编号、点位名称、设备类型、维护人员、投放时间和设备柜状态维护
- 充电宝档案：设备编号、所属机柜、电池规格、维护人员、入库时间和设备状态维护
- 点位投放：投放编号、点位名称、投放批次、运营人员、计划时间和投放状态维护
- 设备巡检：巡检编号、设备柜名称、巡检类型、巡检人员、巡检时间和巡检状态维护
- 故障维修：维修编号、故障设备、故障类型、维修人员、报修时间和维修状态维护
- 异常回收：回收编号、异常设备、异常类型、回收人员、回收时间和回收状态维护
- 租借订单：订单编号、点位名称、租借类型、下单用户、下单时间和订单状态维护
- 商户收益：收益编号、合作点位、收益类型、商户负责人、入账时间和收益状态维护
- 收益结算：结算编号、结算周期、结算类型、财务人员、结算时间和结算状态维护
- 库存调拨：调拨编号、调拨仓点、调拨类型、经办人员、调拨时间和调拨状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
