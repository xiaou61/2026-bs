# 174-frontend UI 美化展示

## 项目信息
- **项目名称**: 社区慢病档案随访与服药依从性管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 健康粉
- **美化完成时间**: 2026-05-30

## 设计理念

采用"健康粉"设计理念，以粉色和玫红色为主色调，体现慢病管理的温暖、关爱和专业感。通过药丸、爱心等装饰元素，营造出慢病管理、健康随行的社区卫生氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | heartbeat | 心跳效果 |
| 顶部条 | chronicGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
