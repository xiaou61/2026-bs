# 186-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园餐厅后厨留样与卫生巡检台账系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 餐饮橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"餐饮橙"设计理念，以橙色和金色为主色调，体现餐饮管理的温暖、安全和活力。通过餐具、蔬菜等装饰元素，营造出食品安全、舌尖守护的餐饮管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 餐饮橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | foodGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

186-frontend采用"餐饮橙"设计理念，通过橙色和金色的搭配，营造出温暖、安全的餐饮管理氛围。餐具装饰元素突出了校园餐厅后厨管理的特色。
