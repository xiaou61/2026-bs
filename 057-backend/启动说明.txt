057 招生管理系统启动说明

一、后端启动

1. 进入后端目录：

   cd 057-backend

2. 默认使用 H2 内存数据库启动：

   mvn spring-boot:run

3. 后端访问地址：

   http://localhost:8057

4. H2 控制台：

   http://localhost:8057/h2-console

5. H2 连接信息：

   JDBC URL：jdbc:h2:mem:enrollment_db
   用户名：sa
   密码：留空

二、管理端启动

1. 进入管理端目录：

   cd 057-frontend

2. 安装依赖：

   npm install --no-audit --no-fund

3. 启动开发服务：

   npm run dev

4. 管理端访问地址：

   http://localhost:3057

三、默认账号

超级管理员：admin / 123456
普通管理员：user / 123456

四、MySQL 模式

默认演示环境不需要 MySQL。如需切换 MySQL，请先创建 enrollment_db 数据库并导入 sql/init.sql，然后运行：

   mvn spring-boot:run -Dspring-boot.run.profiles=mysql

MySQL 配置文件：

   src/main/resources/application-mysql.yml

五、验证命令

后端测试：

   mvn test

管理端构建：

   cd 057-frontend
   npm run build

六、说明

本轮已修复默认 MySQL/8080 强依赖、H2 自举、分页方言固定 MySQL、JWT Bearer 兼容、401/403 HTTP 状态、普通管理员越权访问管理员管理、Redis 不可用阻断演示登录、密码响应泄露风险、管理端端口和代理配置等问题。
