# 199-frontend UI 美化展示

## 项目信息
- **项目名称**: 运动康复训练计划与体测评估管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 健身橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"健身橙"设计理念，以橙色和金色为主色调，体现健身运动的热情、活力和坚持。通过肌肉、跑步等装饰元素，营造出生命在于运动、运动在于坚持的健身管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 健身橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |
| 深橙 | #E65100 | 文字色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | flex | 弯曲缩放 |
| 顶部条 | fitnessGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
