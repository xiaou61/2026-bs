# 133-frontend UI 美化展示

## 项目信息
- **项目名称**: 实验室耗材采购审批与库存预警系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 实验紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"实验紫"设计理念，以紫色为主色调，体现实验室的科学性、严谨性和创新感。通过试管、显微镜等装饰元素，营造出科学实验、安全第一的实验室管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bubble | 冒泡上升 |
| 顶部条 | labGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
