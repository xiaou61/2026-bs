# 165-frontend UI 美化展示

## 项目信息
- **项目名称**: 企业访客预约登记与门禁联动管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 门禁蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"门禁蓝"设计理念，以蓝色为主色调，体现门禁系统的安全性、专业性和科技感。通过建筑、锁等装饰元素，营造出智慧门禁、安全通行的企业管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | accessGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
