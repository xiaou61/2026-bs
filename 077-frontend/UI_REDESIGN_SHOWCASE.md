# 077-frontend UI 美化展示

## 项目信息
- **项目名称**: MES生产制造执行系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 工业橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"工业橙"设计理念，以橙色和深蓝色为主色调，体现MES系统的专业性、执行力和工业感。通过工厂、齿轮等装饰元素，营造出智能制造、精准执行的生产管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 工业橙 | #FF6F00 | 主色调 |
| 深蓝 | #1A237E | 辅助色 |
| 暖橙 | #FF8F00 | 强调色 |
| 金色 | #FFB300 | 装饰色 |
| 暗橙 | #E65100 | 文字色 |

- 深蓝渐变背景

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | mesGradient | 橙色流光 |
| 按钮 | hover | 悬浮上移 |

## 总结

077-frontend采用"工业橙"设计理念，通过橙色和深蓝的搭配，营造出专业、高效的MES生产管理氛围。工厂装饰元素突出了生产制造的行业特色。
