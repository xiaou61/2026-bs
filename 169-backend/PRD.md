# 校车乘车预约与学生上下车核验系统 PRD

## 项目概述
本系统面向中小学校车乘车预约、学生上下车核验、异常处置与家校通知场景，提供校车线路、站点、车辆、司机、学生、家长、乘车预约、上车核验、下车核验、异常告警和家长通知的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 校方管理员：校车线路、站点、学生、家长和安全台账维护
- 调度管理员：车辆、司机、乘车预约、上下车核验和异常调度维护
- 校车司机：查看线路站点，维护上车核验、下车核验和异常上报
- 跟车老师：维护学生乘车核验、异常处置和家长通知
- 学生家长：提交乘车预约，查看学生上下车核验、异常告警和通知

## 功能模块
- 校车线路：线路编号、线路名称、线路类型、负责人员、启用时间和线路状态维护
- 站点档案：站点编号、站点名称、站点类型、负责人员、启用时间和站点状态维护
- 车辆档案：车辆编号、车牌号码、车辆类型、管理人员、投用时间和车辆状态维护
- 司机档案：司机编号、司机姓名、准驾类型、管理人员、入职时间和司机状态维护
- 学生档案：学生编号、学生姓名、年级班级、班主任、建档时间和学生状态维护
- 家长档案：家长编号、家长姓名、关联学生、联系人员、建档时间和家长状态维护
- 乘车预约：预约编号、预约线路、预约类型、预约学生、预约时间和预约状态维护
- 上车核验：上车编号、核验线路、核验类型、核验人员、上车时间和上车状态维护
- 下车核验：下车编号、核验线路、核验类型、核验人员、下车时间和下车状态维护
- 异常告警：告警编号、告警学生、告警类型、处置人员、告警时间和告警状态维护
- 家长通知：通知编号、通知对象、通知类型、发送人员、发送时间和通知状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护
