# 实现计划

## 问题陈述

构建一个围绕发布审批、部署执行和回滚治理的前后端分离系统，覆盖发布环境、应用服务、流水线、发布计划、发布单、审批流程、部署任务、回滚预案和变更检查。

## 当前状态

`106-backend` 与 `106-frontend` 已完成 DevOps 发布管理系统的基础实现，当前进入精修阶段，重点核对认证安全、审批链路、状态流转和文档准确性。

## 实施方案

### 第一阶段：后端开发
1. Spring Boot 2.7.18 + MyBatis 2.3.1 + PageHelper + Redis + JWT
2. 创建 `devops_release_106` 数据库和 14 张业务表
3. 实现统一返回、异常处理、JWT 拦截、Redis 配置和分页查询
4. 实现 Entity、Mapper、Service 和 Controller
5. 实现发布审批、部署执行、回滚处理和检查清单管理
6. 实现首页统计和操作日志

### 第二阶段：前端开发
1. Vue3 + Vite + Element Plus + Pinia + Axios + ECharts
2. 登录页、布局页、首页看板
3. 发布环境、应用服务、流水线、发布计划、发布单、审批、部署、回滚和检查页面
4. 业务按钮支持提交审批、部署、回滚、确认和归档

### 第三阶段：文档更新
1. 更新 `readme.md`
2. 更新 `readme_simple.md`
3. 更新候选清单

## 完成标准

- 后端结构完整
- 前端结构完整
- SQL 包含演示数据
- 认证返回不暴露敏感字段
- 审批、部署、回滚等关键流程口径一致
- 文档描述与实际实现保持一致
- 不执行编译验证
