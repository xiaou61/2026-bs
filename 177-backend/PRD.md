# 直播基地主播排班与选品样品管理系统 PRD

## 项目概述
本系统面向直播基地运营、主播排班、商品选品、样品借还、直播计划、直播场次和直播复盘场景，提供直播间档案、主播档案、商家档案、商品选品、样品台账、样品借还、主播排班、选品评测、直播计划、直播场次和直播复盘的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 基地运营：维护直播间档案、主播档案、主播排班、直播计划和统计规则
- 主播：查看排班、参与样品借还、执行直播场次并提交直播复盘
- 选品专员：维护商品选品、选品评测、直播计划和样品协同
- 样品管理员：维护样品台账、样品借还、样品归还和状态流转
- 商家代表：维护商家档案、商品提报、样品寄送和直播复盘反馈

## 功能模块
- 直播间档案：直播间编号、直播间名称、直播类型、运营人员、启用时间和直播间状态维护
- 主播档案：主播编号、主播姓名、主播类型、运营人员、签约时间和主播状态维护
- 商家档案：商家编号、商家名称、经营类型、对接人员、入驻时间和商家状态维护
- 商品选品：商品编号、商品名称、商品类型、提报商家、提报时间和选品状态维护
- 样品台账：样品编号、样品名称、样品类型、仓储人员、入库时间和样品状态维护
- 样品借还：借还编号、借还样品、借还类型、借用主播、借用时间和借还状态维护
- 主播排班：排班编号、排班主播、排班类型、运营人员、排班时间和排班状态维护
- 选品评测：评测编号、评测商品、评测类型、选品人员、评测时间和评测状态维护
- 直播计划：计划编号、计划主题、直播类型、负责人、计划时间和计划状态维护
- 直播场次：场次编号、直播主题、场次类型、主播人员、开播时间和场次状态维护
- 直播复盘：复盘编号、直播场次、复盘类型、复盘人员、复盘时间和复盘状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 后端：Spring Boot 2.7.18、MyBatis 注解 SQL、PageHelper、MySQL、Redis、JWT
- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
