# 029-frontend UI 美化展示

## 项目信息
- **项目名称**: 中药食疗推荐交流平台
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 养生绿
- **美化完成时间**: 2026-05-29

## 设计理念

采用"养生绿"设计理念，以绿色和棕色为主色调，体现中药食疗的自然和健康理念。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 养生绿 | #4CAF50 | 主色调 |
| 药材棕 | #795548 | 辅助色 |
| 温暖橙 | #FF9800 | 强调色 |
| 背景色 | #F1F8E9 | 页面背景 |
