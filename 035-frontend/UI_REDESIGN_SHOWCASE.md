# 035-frontend UI 美化展示

## 项目信息
- **项目名称**: 水稻收割预约系统
- **技术栈**: Vue 3 + Element Plus + Pinia
- **设计主题**: 田园绿
- **美化完成时间**: 2026-05-29

## 设计理念

采用"田园绿"设计理念，以绿色和金色为主色调，体现农业的自然、生态和丰收感。通过稻穗、拖拉机、太阳等装饰元素，营造出田园牧歌式的农业管理系统氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 稻穗 | sway | 左右摇摆 |
| 拖拉机 | drive | 横向行驶 |
| 太阳 | pulse | 脉冲闪烁 |
| 装饰物 | float | 上下浮动 |
| Logo | bounce | 弹跳效果 |
| 卡片 | hover | 悬浮上移 |
