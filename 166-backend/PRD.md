# 农贸市场摊位巡检与食品追溯台账系统 PRD

## 项目概述
本系统面向农贸市场日常监管与食品安全追溯场景，提供市场区域、摊位档案、商户档案、商品溯源、摊位巡检、问题整改、抽检记录、检测结果、进货台账、处置记录和风险预警的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 市场管理员：市场区域、摊位档案、商户档案、巡检计划和台账汇总维护
- 巡检员：摊位巡检、问题发现、整改跟踪、处置记录和风险预警维护
- 摊位商户：商品溯源、进货台账、整改反馈和检测结果查看维护
- 抽检员：抽检记录、检测结果、样品流转和风险线索维护
- 监管审核员：问题整改复核、检测结果审核、风险预警和监管闭环维护

## 功能模块
- 市场区域：区域编号、区域名称、经营品类、区域负责人、开放时间和区域状态维护
- 摊位档案：摊位编号、摊位名称、经营类型、摊主姓名、入驻时间和摊位状态维护
- 商户档案：商户编号、商户名称、证照类型、联系人、登记时间和经营状态维护
- 商品溯源：溯源码、商品名称、商品类别、供货商、入场时间和溯源状态维护
- 摊位巡检：巡检编号、巡检摊位、巡检类型、巡检员、巡检时间和巡检状态维护
- 问题整改：整改编号、问题摊位、问题类型、责任商户、整改时间和整改状态维护
- 抽检记录：抽检编号、抽检商品、抽检类型、抽检人员、抽检时间和抽检状态维护
- 检测结果：检测编号、检测样品、检测项目、检测机构、出具时间和检测状态维护
- 进货台账：台账编号、进货商品、来源类型、供货单位、进货时间和台账状态维护
- 处置记录：处置编号、处置对象、处置类型、处置人员、处置时间和处置状态维护
- 风险预警：预警编号、预警对象、预警类型、处置负责人、预警时间和预警状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

## 技术栈
- 后端：Spring Boot 2.7.18、MyBatis 注解 SQL、PageHelper、MySQL、Redis、JWT
- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
- 数据库：market_trace_166
