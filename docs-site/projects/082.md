## 项目简介

本项目是面向公务员考试备考场景的学习平台，包含 Spring Boot 后端和 Vue 管理端。核心流程包括用户登录注册、学科课程、题库试题、模拟试卷、考试记录、学习计划、公告和统计看板。

## 特色亮点

- 公考业务闭环：课程学习→刷题训练→模拟考试→学习计划复盘
- 三角色协同：管理员统筹平台、讲师维护内容、学员学习考试
- Redis缓存：学科与公告公开列表缓存，减轻数据库查询压力
- 统一接口风格：RESTful + Result 统一返回结构
- 统计可视化：ECharts 饼图展示学科分布，仪表盘展示通过率

## 演示流程

1. 管理员登录后台，查看平台数据看板
2. 维护学科与课程，完善基础教学内容
3. 讲师创建题库并录入试题
4. 管理员配置模拟试卷并发布
5. 学员参加模拟考试，生成考试记录
6. 学员制定学习计划并持续更新完成天数
7. 管理员发布公告，前台公开列表展示

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 082-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 082-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
