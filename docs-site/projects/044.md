## 项目简介

本项目是基于 Spring Boot 3.2、MyBatis-Plus、Spring Security、JWT 的特色民宿预订平台后端，提供用户登录注册、民宿浏览搜索、房型查看、收藏、预订、房东民宿管理和房东订单确认等核心接口。

| 模块 | 说明 |

## 演示流程

1. 用户注册（可选游客/房东角色）→ 登录系统
2. 浏览首页热门民宿 → 点击进入详情页
3. 查看民宿介绍、设施、房型列表、用户评价
4. 选择房型 → 选择入住/离店日期 → 填写联系人信息
5. 确认预订 → 自动计算费用 → 生成订单
6. 我的订单 → 查看订单状态 → 可取消未确认订单
7. 订单完成后 → 评价民宿（评分+内容）
8. 收藏喜欢的民宿 → 我的收藏管理
9. 房东登录 → 民宿管理 → 添加/编辑民宿和房型
10. 房东确认订单 → 回复用户评价

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 044-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 044-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
