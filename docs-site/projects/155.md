## 项目简介

本系统面向社区基层党建治理场景，提供党组织维护、党员档案、党建活动、活动报名、签到记录、志愿任务、积分记录、积分兑换、组织关系和榜单激励的一体化后台管理能力。

- 形成“党组织维护-党员档案-党建活动-活动报名-签到记录-志愿任务-积分记录-积分兑换-组织关系-榜单统计-通知公告”完整业务闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和状态完成
- 首页看板展示党建活动、报名签到、积分发放和榜单发布的近7日趋势与状态分布

- 社区党建活动报名与积分激励平台类毕设项目
- 基层党建、社区治理、志愿服务、积分激励和组织关系管理课程设计

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 155-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 155-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
