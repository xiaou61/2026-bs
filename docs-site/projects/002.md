- MyBatis

### 前端

## 角色权限

系统支持 **3** 种角色，各角色权限如下：

### 学生

- 在线选课、退课
- 查看课表
- 查看成绩

### 教师

- 查看授课课程
- 查看选课学生
- 录入成绩

### 管理员

- 管理学生、教师信息
- 管理课程信息
- 发布系统公告
- 配置选课时间

| 模块 | 说明 |

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 002-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/course_selection_system
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 002-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
