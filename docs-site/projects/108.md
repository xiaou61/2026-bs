## 项目简介

云原生成本分析与资源优化平台面向企业云资源 FinOps 治理场景，围绕云账号、资源空间、账单明细、预算策略、成本分摊、闲置识别、优化建议、节省计划和异常事件构建闭环管理能力，帮助运维和成本团队完成成本透明、责任分摊、异常发现和优化执行。

- 查看节省计划和成本趋势

## 功能模块

| 模块 | 说明 |
|------|------|
| **账号权限** | 管理员/成本分析师/云运维工程师/成本主管四角色登录、JWT鉴权和Redis登录态 |
| **云账号** | 管理云厂商账号、账号编码、接入方式、月预算和负责人 |
| **资源命名空间** | 维护业务线、环境、资源空间、云账号归属和负责人 |
| **成本账单** | 管理账单编号、账期、币种、应付实付金额并支持确认、支付、关闭 |
| **成本明细** | 记录资源ID、资源类型、账期、使用量、成本金额和归属空间 |
| **预算策略** | 配置预算金额、告警比例、周期类型和启用归档状态 |
| **成本分摊** | 管理分摊编号、业务线、分摊金额并支持提交、通过、驳回 |
| **闲置资源** | 识别低利用资源、闲置天数、月成本并支持确认、忽略、清理 |
| **优化规则** | 配置资源类型、指标阈值、优化动作和优先级 |
| **优化建议** | 记录建议编号、建议类型、预计节省并支持采纳、完成、拒绝 |
| **节省计划** | 管理承诺消费、覆盖范围、预计节省和执行状态 |
| **成本异常** | 记录成本突增、预算超限等异常事件并支持确认、解决、关闭 |
| **报告快照** | 生成月度分析、优化和异常报告，支持发布和归档 |
| **操作日志** | 记录关键操作对象、动作、详情和IP地址 |

## 特色亮点

- 选题贴合 FinOps、云原生成本治理和企业降本增效方向，答辩辨识度高
- 形成“云账号-账单明细-预算策略-成本分摊-闲置识别-优化建议-报告复盘”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper，与 107 的 MyBatis-Plus 路线继续交替
- 账单确认支付、分摊审批、闲置资源清理、优化建议采纳、异常事件解决都有状态流转
- 首页看板展示云账号、账单、优化建议、异常事件、成本趋势和节省来源分布

## 演示流程

1. 管理员登录维护账号权限、云账号和操作日志
2. 成本分析师创建资源命名空间、导入成本账单和成本明细
3. 成本分析师配置预算策略并发起成本分摊审批
4. 云运维工程师维护优化规则，确认闲置资源并处理优化建议
5. 成本主管审批成本分摊，跟踪节省计划执行结果
6. 成本分析师处理成本异常事件并发布报告快照
7. 首页看板查看成本趋势、节省来源和核心统计指标

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 108-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 108-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
