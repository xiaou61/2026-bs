## 项目简介

本系统是一个基于 Spring Boot + Vue 3 的博物馆文物数字化管理平台，面向管理员、讲解员、研究员、游客四类角色，支持文物数字档案、展览活动、参观预约、讲解预约、研究成果审核、收藏和公告等功能。

本轮巡检后，项目默认使用 H2 内存数据库自举，直接启动即可演示；如需连接 MySQL，可切换到 `mysql` profile。

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 048-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 048-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
