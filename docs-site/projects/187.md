## 项目简介

本系统面向旅行社团建行程组织、成员报名确认、预算编制、费用分摊、缴费记录、通知同步和出行反馈场景，提供旅行社档案、团队档案、团建行程、行程报名、成员确认、费用预算、费用分摊、缴费记录、通知同步、出行反馈和发票记录的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 旅行社管理员

- 维护旅行社档案
- 团队档案
- 行程规则和协作配置

### - 行程策划

- 编排团建行程
- 预算项目和报名节奏

### - 财务人员

- 维护费用预算
- 费用分摊
- 缴费记录和发票记录

### - 团队领队

- 组织成员报名确认
- 通知同步和出行反馈

### - 团建成员

- 查看行程安排
- 确认报名
- 缴费和提交反馈

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 187-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 187-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
