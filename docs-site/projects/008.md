## 项目简介

智能菜谱推荐系统是一个前后端一体化的Spring Boot项目，提供食材管理、菜谱推荐、营养分析、购物清单、做菜打卡等功能。

## 技术栈

### 后端

- Spring Boot 3.2.0
- MyBatis-Plus 3.5.5
- MySQL 8.0
- JWT 4.4.0

### 前端

- jQuery 3.7.1
- Bootstrap 5.3.0
- ECharts 5.4.3
- Layer 3.5.1

## 角色权限

系统支持 **2** 种角色，各角色权限如下：

### 1. 用户

- 用户认证 - 注册/登录（JWT）
- 首页 - 今日推荐、热门菜谱、统计数据、营养分析图表
- 菜谱模块 - 列表、详情、搜索、收藏
- 智能推荐 - 今日推荐、基于食材推荐、新手推荐
- 食材管理 - 库存管理、过期提醒
- 营养分析 - 自动计算、可视化展示
- 购物清单 - 自动生成、一键加入库存
- 打卡记录 - 记录、统计、成就展示
- ...还有 1 项权限

### 2. 管理员

- 数据统计 - 用户、菜谱、食材、打卡统计
- 菜谱管理 - 查看、删除
- 食材管理 - 查看、删除
- 用户管理 - 查看、启用/禁用

## 功能模块

| 模块 | 说明 |
|------|------|
| **智能推荐** | 多维度算法推荐菜谱（食材匹配、口味偏好） |
| **菜谱管理** | 菜谱浏览、详情、搜索、收藏 |

| **营养分析** | 自动计算营养成分、可视化图表 |

| **打卡记录** | 做菜打卡、成就系统、统计分析 |
| **后台管理** | 菜谱/食材/用户管理 |

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 008-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/smart_recipe
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 008-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
