## 演示流程

1. 用户注册登录 → 进入首页
2. 浏览轮播图 → 查看推荐/热门/最新课程
3. 进入课程列表 → 按分类筛选、搜索课程
4. 查看课程详情 → 了解课程介绍、章节列表、教师信息
5. 开始学习课程 → 观看视频 → 系统自动保存进度
6. 切换章节 → 继续学习 → 完成课程
7. 发表评价 → 给课程评分和评论
8. 收藏课程 → 管理我的收藏列表
9. 个人中心 → 查看我的课程、学习记录
10. 管理员登录 → 管理课程、章节、视频、用户、评论

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 052-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 052-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
