## 项目简介

### 项目简介
哈尔滨文旅综合服务平台，聚焦冰雪之城哈尔滨的旅游文化推广。涵盖冰雪景点、历史建筑、特色美食、住宿推荐、旅游路线、活动赛事、游记分享等全方位文旅服务，为游客提供一站式哈尔滨旅游攻略与预订服务。

### 技术栈

## 角色权限

系统支持 **2** 种角色，各角色权限如下：

### - 管理员 (admin)

- 全平台管理权限
- 景点/酒店/餐厅/活动/游记审核
- 数据统计看板

### - 普通用户 (user)

- 浏览景点路线
- 门票预订
- 游记发布
- 评价互动
- 收藏

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户管理** | 注册登录、JWT认证、双角色权限（管理员/普通用户）、个人信息管理 |
| **景点管理** | 景点CRUD、分类筛选（冰雪乐园/历史文化/自然风光等）、景点详情、浏览量统计 |

| **酒店推荐** | 酒店列表、价格评分、联系预订 |
| **餐厅推荐** | 餐厅分类（东北菜/俄式西餐等）、推荐菜品、评分展示 |

| **游记分享** | 游记发布编辑、评论互动、浏览收藏 |
| **评价系统** | 景点评价、评分统计、评价展示 |
| **收藏功能** | 景点收藏、游记收藏、收藏管理 |
| **钱包系统** | 余额充值、订单支付、充值记录 |
| **公告管理** | 系统公告发布、置顶公告、公告列表 |
| **数据统计** | 管理看板、用户统计、订单统计、收入统计 |

## 演示流程

1. 用户注册登录 → JWT Token认证
2. 浏览首页 → 查看热门景点排行、最新公告、快捷入口
3. 浏览景点列表 → 分类筛选 → 查看景点详情
4. 景点详情页 → 查看门票列表 → 选择票种预订
5. 填写预订信息 → 选择日期数量 → 提交订单
6. 钱包充值 → 支付订单 → 订单状态变为已支付
7. 浏览旅游路线 → 查看行程安排 → 途经景点导航
8. 查看酒店餐厅推荐 → 联系预订
9. 浏览活动列表 → 报名参加文旅活动
10. 发布旅游游记 → 其他用户评论互动
11. 收藏喜欢的景点和游记 → 管理我的收藏
12. 管理员登录后台 → 查看运营数据 → 管理景点/订单/用户

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 072-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 072-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
