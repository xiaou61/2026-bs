## 项目简介

本项目为 `079 计算机学院校友网` 后端，提供校友信息、新闻公告、活动报名、捐赠、校友企业、招聘、论坛、轮播图、日志和统计接口。

## 角色权限

系统支持 **2** 种角色，各角色权限如下：

### - 管理员(admin)

- 系统全部权限
- 用户审核
- 内容管理
- 数据统计

### - 校友(alumni)

- 查看信息
- 参与活动
- 发帖交流
- 捐赠等

## 演示流程

1. 用户注册 → 填写校友信息 → 等待管理员审核
2. 审核通过 → 完善个人档案 → 浏览校友列表
3. 查看新闻公告 → 发表评论 → 参与讨论
4. 浏览活动列表 → 报名参加活动 → 活动当天签到
5. 查看捐赠项目 → 在线捐赠 → 查看捐赠记录
6. 登记校友企业 → 发布招聘信息 → 等待认证
7. 进入论坛社区 → 发布话题 → 参与讨论回复
8. 管理员后台 → 审核校友、管理活动、处理捐赠、维护论坛

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 079-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 079-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
