| **用户系统** | 微信授权登录、用户信息管理、积分等级体系（Lv1-Lv5）、角色权限（普通用户/教师/管理员） |

## 演示流程

1. 小程序登录 → 微信授权 → 获取用户信息 → 创建账户
2. 浏览首页 → 查看热门课程、问答、文章、代码片段
3. 进入课程模块 → 选择课程 → 浏览章节 → 开始学习
4. 学习课程 → 记录进度 → 完成课程 → 获得积分 → 提升等级
5. 进入问答社区 → 浏览问题 → 提出问题 → 设置悬赏积分
6. 回答问题 → 获得积分 → 答案被采纳 → 获得额外积分
7. 查看技术文章 → 点赞收藏 → 发表评论 → 发布自己的文章
8. 代码分享 → 浏览代码片段 → Fork代码 → 分享自己的代码
9. 每日打卡 → 记录学习时长 → 连续打卡 → 查看排行榜
10. 个人中心 → 查看积分等级 → 我的发布 → 我的收藏 → 学习统计
11. 全局搜索 → 搜索课程/问答/文章/代码 → 快速定位内容
12. 管理后台 → 用户管理 → 内容审核 → 数据统计

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |

### 后端启动

```bash
cd 037-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/programming_learning
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 小程序启动

1. 使用微信开发者工具打开 `037-miniprogram` 目录
2. 确认 AppID 配置正确
3. 修改后端接口地址（通常在 config.js 或 utils/request.js 中）
4. 点击编译运行

### 开发注意事项

1. 数据库配置：修改 `application.yml` 中的数据库连接信息（数据库名：`programming_learning`，默认使用 H2 内存数据库）
2. 后端启动前需先创建数据库并执行 SQL 初始化脚本
3. 后端默认端口为 `8037`，前端代理需配置对应端口
4. 本项目为微信小程序登录，无账号密码；用户敏感信息已在接口层脱敏返回
