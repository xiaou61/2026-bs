## 项目简介

本项目是贫困地区儿童资助管理平台，后端位于 `080-backend`，前端位于 `080-frontend`。系统包含用户登录、儿童档案、资助申请、捐赠管理、资金记录、项目管理、公告、成长记录、反馈和统计看板等模块。

## 演示流程

1. 志愿者登录 → 录入贫困儿童信息 → 填写家庭情况、教育状况
2. 儿童提交资助申请 → 填写申请理由、所需金额
3. 管理员审核申请 → 通过/拒绝 → 填写审核意见
4. 捐赠人浏览儿童信息 → 选择资助对象
5. 发起捐赠 → 选择捐赠类型（资金/物资）→ 填写金额、支付方式
6. 管理员确认捐赠 → 生成捐赠证书
7. 建立资助关系 → 选择资助类型（一次性/按月/按年）
8. 志愿者添加成长记录 → 记录学习、生活、健康情况
9. 儿童发送感谢信 → 捐赠人查看反馈
10. 管理员记录资金使用 → 收入支出明细
11. 查看数据统计 → 首页仪表盘、捐赠趋势、地区分布
12. 发布公告通知 → 新闻动态、政策文件

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 080-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 080-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
