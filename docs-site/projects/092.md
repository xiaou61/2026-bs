## 项目简介

### 项目简介
蓝天幼儿园管理系统面向幼儿园园务管理、教师日常执行和家长协同服务场景，支持管理员、教师、家长三角色围绕园区基础资料、幼儿档案、课程活动、考勤晨检、接送反馈、食谱公告和统计看板开展业务协同，形成完整的园务管理闭环。

## 特色亮点

- 幼儿园业务主题鲜明，适合做园务管理与家园协同类毕设答辩
- 管理员、教师、家长三角色边界清晰，演示流程完整
- 形成“档案-活动-食谱-考勤-晨检-反馈”的日常运营闭环
- MyBatis 原生 XML + PageHelper 便于展示 SQL 与分页实现细节
- 统计看板聚合账号、幼儿、活动、公告等核心指标

## 演示流程

1. 园长登录后台维护园区、年级、班级、学期和活动课程
2. 教师维护幼儿档案、发布活动安排和每周食谱
3. 教师登记入园考勤与健康晨检，沉淀每日在园数据
4. 家长登录查看孩子档案、活动安排和食谱，并提交家园反馈
5. 园方在公告中心和看板页查看园务通知与整体运营情况

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 092-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 092-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
