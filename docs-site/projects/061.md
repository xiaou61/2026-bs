## 项目简介

本项目是游戏交易系统后端服务，配套 `061-frontend` 使用。系统覆盖登录注册、游戏分类、交易商品、下单支付、卖家发货、买家确认、收藏评价、交易申诉、公告和运营看板等流程。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户与认证** | 注册登录、JWT鉴权、用户状态管理 |

## 演示流程

1. 管理员登录后维护游戏分类与平台公告
2. 卖家发布游戏商品并设置价格、库存、交易方式
3. 买家进入交易市场筛选商品并提交订单
4. 买家支付订单，系统自动扣库存并结算双方余额
5. 卖家/管理员发货，买家确认完成并提交评价
6. 发生争议时买家发起申诉，管理员后台回复处理

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 061-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 061-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
