## 功能模块

| 模块 | 说明 |
|------|------|
| **动物展示** | 待领养动物列表、筛选（品种/性别/年龄）、详情展示 |

| **救助中心** | 发布救助线索、救助地图标记、紧急救助 |
| **爱心社区** | 发帖交流、分享经验、评论互动、点赞 |
| **科普公告** | 养宠知识科普、平台活动公告 |
| **管理后台** | 动物管理、申请审核、内容管理、用户管理 |

## 特色亮点

- 规范领养流程：申请→审核→协议→回访，全流程规范化
- 救助地图功能：标记流浪动物出没地点，便于精准救助
- 爱心社区互动：建立用户间的交流圈子，分享养宠乐趣
- 严格审核机制：管理员审核领养申请，确保动物安全
- 前后端分离：Vue3 + Spring Boot主流架构

## 演示流程

1. 用户注册登录 → 浏览待领养动物列表
2. 查看动物详情 → 点击申请领养 → 填写申请表
3. 管理员后台审核申请 → 通过/驳回
4. 用户发布救助线索 → 上传照片和位置
5. 在爱心社区发帖 → 分享自家宠物照片
6. 管理员发布科普文章 → 用户阅读学习

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 024-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/db_024
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 024-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
