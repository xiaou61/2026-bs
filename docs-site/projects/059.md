## 项目简介

本项目是制造装备物联及生产管理 ERP 系统后端服务，配套 `059-frontend` 管理端使用。系统覆盖用户权限、设备台账、传感器数据、告警处理、产品档案、生产工单、物料库存、质量检测、维保计划、维保记录和数据看板等流程。

## 演示流程

1. 管理员登录 → 管理设备分类和设备台账 → 配置传感器
2. 操作员登录 → 查看设备状态 → 记录传感器数据 → 处理告警
3. 生产经理登录 → 管理产品信息 → 创建生产工单 → 跟踪进度
4. 质检员登录 → 执行质量检测 → 记录检测结果
5. 管理员管理物料 → 出入库操作 → 库存盘点
6. 管理员制定维保计划 → 执行维保 → 查看数据看板

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 059-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 059-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
