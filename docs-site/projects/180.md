## 项目简介

本系统面向社区物业报修、客服派单、维修上门、物料使用、费用登记、支付记录、满意评价和投诉回访场景，提供小区区域、业主档案、报修分类、报修工单、派单分配、上门处理、物料使用、费用登记、支付记录、满意评价和投诉回访的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 物业管理员

- 维护小区区域
- 业主档案
- 报修分类和服务规则

### - 业主住户

- 提交报修工单
- 查看上门处理
- 支付费用并评价服务

### - 派单员

- 接收报修工单
- 分派维修人员并跟踪处理进度

### - 维修人员

- 接收派单
- 登记上门处理
- 填写物料使用和处理结果

### - 财务客服

- 登记维修费用
- 核对支付记录并开展满意度回访

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 180-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 180-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
