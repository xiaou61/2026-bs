## 项目简介

本项目面向高校课程作业、论文初稿、实验报告和竞赛材料的学术诚信管理，提供文本提交、检测规则、AI 生成概率评估、人工复核、预警通知、整改跟踪和统计审计能力。系统采用 Spring Boot + MyBatis + PageHelper + MySQL + Redis + Vue3 技术栈，围绕“提交-检测-复核-预警-整改-归档”形成完整闭环。

## 功能模块

| **整改申诉** | 学生提交整改说明和申诉，教师或复核员审核闭环 |
| **统计审计** | 首页指标、ECharts 图表和操作日志审计 |

## 特色亮点

- 选题贴合高校 AI 生成文本治理、论文诚信检测和教学过程监管热点
- 形成“作业发布-文本提交-规则检测-人工复核-预警整改-申诉处理”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper，与 099 的 MyBatis-Plus 技术路线区分明显
- 检测服务根据命中规则、AI模板化表达、重复片段和引用说明计算风险指标
- 支持四角色演示路径，既能展示学生端提交，也能展示教师和复核员处理流程

## 演示流程

1. 管理员登录维护用户账号和检测规则
2. 教师维护课程、班级、学生档案并发布作业任务
3. 学生提交课程论文正文、引用说明和附件地址
4. 复核员创建检测任务并点击启动，系统生成AI概率、重复率、引用风险和诚信分
5. 复核员在检测结果页填写人工复核意见
6. 高风险提交生成诚信预警，教师或复核员处理预警并要求整改
7. 学生提交整改记录或申诉，复核员审核后在首页查看统计指标

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 100-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 100-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
