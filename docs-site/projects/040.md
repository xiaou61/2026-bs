## 项目简介

本项目是基于 Spring Boot 3.2、JPA、QueryDSL、JWT 的社区车辆违停处理系统后端，提供用户登录注册、违停类型、违停举报、举报审核等核心接口。

## 演示流程

1. 用户注册登录 → JWT Token认证
2. 绑定车辆 → 输入车牌号、车主姓名
3. 提交举报 → 上传照片 → 选择违停类型 → 填写位置描述
4. 查看我的举报 → 实时查看审核状态
5. 管理员登录 → 进入管理后台
6. 数据看板 → 查看总举报数、待审核数、通过率
7. 审核举报 → 查看证据照片 → 通过/拒绝 → 填写审核意见
8. 用户收到通知 → 查看审核结果
9. 如被拒绝 → 可提交申诉 → 上传补充证据
10. 查询积分 → 查看积分明细 → 历史记录

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 040-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 040-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
