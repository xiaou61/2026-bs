## 项目简介

本系统面向医院临床实习与医学教育管理场景，提供轮转科室、实习学生、带教老师、轮转安排、病例学习、学习记录、教学查房、技能训练、带教评分、出科考核和反馈整改的一体化后台管理能力。

### - 教学秘书

- 轮转计划
- 学生档案
- 带教安排和出科流程维护

### - 带教老师

- 病例学习
- 教学查房
- 技能训练和日常学习记录维护

### - 实习学生

- 轮转安排
- 病例学习
- 学习记录
- 评分反馈和考核进度查看

### - 考核评委

- 技能训练
- 出科考核和考核评分维护

### - 科室主任

- 轮转科室
- 带教老师
- 教学查房
- 评分复核和整改跟踪维护

| **学习记录** | 记录编号、学习病例、学习类型、实习学生、学习时间和完成状态维护 |
| **教学查房** | 查房编号、查房主题、查房类型、主持老师、查房时间和查房状态维护 |
| **技能训练** | 训练编号、训练项目、训练类型、指导老师、训练时间和训练状态维护 |

| **出科考核** | 考核编号、考核学生、考核类型、考核评委、考核时间和考核状态维护 |
| **反馈整改** | 反馈编号、反馈对象、反馈类型、负责人、反馈时间和整改状态维护 |

- 形成“轮转安排-病例学习-学习记录-教学查房-技能训练-带教评分-出科考核-反馈整改”完整业务闭环

- 首页看板展示核心指标、近7日趋势和状态分布

- MyBatis 注解 SQL + PageHelper 分页查询

- 通用业务台账、状态流转和统计看板联动

- 医学实习轮转考核与病例学习管理系统类毕设项目

- 多角色后台管理系统综合实战
- 需要展示完整业务闭环和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 163-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 163-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
