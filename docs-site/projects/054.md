## 项目简介

本项目是 `054 基于 SpringBoot 的农业生产技术管理系统` 的后端，提供用户登录、作物管理、农业知识库、生产计划与任务、病虫害预警、农资库存、技术咨询、专家预约、公告和统计接口。

- 全部功能权限

- 系统配置
- 数据统计

### EXPERT（专家）

- 回答咨询问题
- 发布农技知识
- 接受预约
- 发布病虫害预警

### TECHNICIAN（技术员）

- 发布农技知识
- 回答咨询问题
- 管理生产计划
- 记录病虫害防治

### FARMER（农户）

- 浏览知识库
- 提问咨询
- 预约专家
- 管理自己的生产计划

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户管理** | 三角色系统（管理员/专家/农户）、JWT认证、密码修改 |
| **作物管理** | 作物信息、分类管理、生长周期、适宜环境 |
| **知识库** | 农业技术知识发布、浏览、点赞、收藏 |
| **生产计划** | 种植计划制定、状态管理（计划中/执行中/已完成） |
| **生产任务** | 任务分配、任务类型、状态跟踪 |
| **病虫害管理** | 病虫害库、症状描述、防治方法 |
| **预警信息** | 病虫害预警发布、预警级别（轻度/中度/重度） |
| **防治记录** | 防治方法记录、效果评估 |
| **农资管理** | 农资信息、库存管理、入库出库、库存预警 |
| **技术咨询** | 咨询问题发布、专家回答、采纳答案 |
| **专家预约** | 预约专家指导、预约时间管理、状态流转 |
| **公告管理** | 系统公告发布、草稿/发布状态 |
| **数据统计** | 首页数据概览、用户/作物/计划/知识统计 |

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 054-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/agriculture_tech
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 054-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
