## 项目简介

校园心理咨询预约与危机干预管理系统面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

| **危机干预** | 干预编号、咨询主题、干预类型、干预对象、干预结果和状态维护 |
| **家校沟通** | 沟通编号、咨询主题、沟通主题、沟通方式、沟通时间和状态维护 |

## 特色亮点

- 选题贴合校园心理健康服务、咨询预约、风险评估、危机干预和家校沟通方向
- 形成“咨询个案-咨询室管理-学生档案-值班安排-预约申请-咨询记录-测评问卷-风险评估-危机干预-家校沟通-随访计划-通知公告”完整闭环
- 使用 MyBatis-Plus 分页查询与通用 CRUD 服务复用，与 148 的 MyBatis 注解 SQL 路线继续交替
- 预约申请、咨询记录、风险评估、危机干预、随访计划和通知公告都有状态流转动作
- 首页看板展示个案、预约、问卷、公告指标，以及咨询预约趋势和个案状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis-Plus 分页查询与通用 CRUD 服务复用
- JWT 登录态拦截与 Redis Token 校验
- 预约受理、风险评估、危机干预和随访通知状态机联动
- ECharts 展示咨询预约趋势和个案状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 校园心理咨询预约与危机干预管理系统类毕设项目
- 心理健康教育、学生事务、危机干预、家校协同和随访管理课程设计
- 多角色后台管理系统综合实战
- 需要展示心理咨询闭环、危机处置和跟踪随访能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 147-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 147-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
