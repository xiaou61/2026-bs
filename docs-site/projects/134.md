| 模块 | 说明 |
|------|------|
| **账号权限** | 管理员/科研人员/财务人员/审批领导四角色登录、JWT鉴权和Redis登录态 |

| **发票记录** | 发票编号、报销编号、发票类型、发票金额、开票单位和状态维护 |
| **审批任务** | 审批编号、报销编号、审批节点、审批人、审批意见和状态维护 |
| **支付记录** | 支付编号、报销编号、支付金额、支付时间、经办人和状态维护 |
| **科研成果** | 成果编号、项目编号、成果名称、成果类型、完成人和状态维护 |
| **论文成果** | 论文编号、项目编号、论文题目、期刊名称、发表时间和状态维护 |
| **专利成果** | 专利编号、项目编号、专利名称、申请人、授权时间和状态维护 |
| **绩效统计** | 统计编号、项目编号、统计月份、报销次数、成果数量和状态维护 |
| **风险预警** | 预警编号、项目编号、风险级别、触发原因、处理人和状态维护 |
| **操作日志** | 关键模块操作人、动作、对象、详情和结果记录 |

## 特色亮点

- 选题贴合科研项目管理、经费预算、报销审批、成果登记和绩效统计方向
- 形成“科研项目-预算科目-经费预算-报销申请-发票记录-审批任务-支付记录-科研成果-论文成果-专利成果-绩效统计-风险预警”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 133 的 MyBatis-Plus 路线继续交替
- 经费预算、报销申请、发票记录、审批任务、支付记录、成果登记和风险预警都有状态流转动作
- 首页看板展示项目、报销、成果、风险指标，以及经费报销趋势和成果类型分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 项目预算、报销申请、发票审批、支付登记、成果归集和风险预警状态机闭环
- ECharts 展示经费报销趋势和成果类型分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 科研项目经费报销与成果管理系统类毕设项目
- 科研管理、经费报销、成果登记、财务审批和绩效统计课程设计
- 多角色后台管理系统综合实战
- 需要展示科研项目、经费预算、报销审批、成果登记和绩效统计闭环的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 134-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 134-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
