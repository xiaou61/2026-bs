## 项目简介

无障碍出行路线规划与志愿协助平台面向无障碍出行服务协同场景，提供多角色登录、路线规划、协助预约、任务调度、统计看板和操作留痕能力。

## 特色亮点

- 选题贴合无障碍出行服务、路线规划、志愿协助、行程记录和消息通知方向
- 形成“无障碍路线-设施点位-出行用户-协助预约-志愿者档案-路线方案-协助任务-服务签到-评价反馈-应急联系人-行程记录-消息通知”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 143 的 MyBatis-Plus 路线继续交替
- 协助预约、路线方案、协助任务、服务签到、行程记录和消息通知都有状态流转动作
- 首页看板展示路线、志愿者、任务、行程指标，以及志愿协助趋势和状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 出行预约、路线协助、服务签到和消息通知状态机联动
- ECharts 展示志愿协助趋势和服务状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 无障碍出行路线规划与志愿协助平台类毕设项目
- 公共服务、无障碍设计、志愿协同、路线规划和社区治理课程设计
- 多角色后台管理系统综合实战
- 需要展示出行协助闭环、路线方案、行程记录和消息通知能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 144-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 144-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
