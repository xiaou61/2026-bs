## 项目简介

本系统面向社区儿童托管、入园签到、接送授权、接送核验、安全告警、健康晨检、活动安排和家长通知场景，提供托管班级、儿童档案、监护人档案、签到记录、接送授权、接送记录、安全告警、健康晨检、托管活动、家长通知和事件跟进的一体化后台管理能力。

### 前端

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 中心管理员

- 维护托管班级
- 儿童档案
- 接送规则和通知配置

### - 托管老师

- 执行儿童签到
- 接送核验
- 活动安排和日常记录

### - 家长

- 查看儿童档案
- 授权接送人员
- 接收通知并确认接送记录

### - 安全员

- 处理接送核验
- 安全告警和事件跟进

### - 保健老师

- 维护健康晨检
- 异常提醒和儿童健康记录

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 181-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 181-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
