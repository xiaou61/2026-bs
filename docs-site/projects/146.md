## 项目简介

食品安全抽检任务与结果公示平台面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

## 技术栈

### 前端

## 特色亮点

- 选题贴合食品安全监管、抽样任务、样品检测、风险预警和结果公示方向
- 形成“抽检计划-食品档案-商户档案-抽样任务-检测机构-样品信息-检测结果-复检申请-处置记录-结果公示-风险预警-通知公告”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 145 的 MyBatis-Plus 路线继续交替
- 抽样任务、检测结果、复检申请、处置记录、风险预警和通知公告都有状态流转动作
- 首页看板展示计划、机构、结果、预警指标，以及抽检趋势和检测状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 抽样任务、检测结果、风险预警和公示通知状态机联动
- ECharts 展示抽检趋势和检测状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 食品安全抽检任务与结果公示平台类毕设项目
- 食品监管、抽检流程、检测协同、风险预警和公开公示课程设计
- 多角色后台管理系统综合实战
- 需要展示抽检闭环、样品检测、风险处置和结果公示能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 146-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 146-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
