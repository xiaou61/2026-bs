## 项目简介

本系统面向化工、制造、能源等工厂危险作业场景，提供作业票申请、分级审批、安全交底、现场监护、气体检测和隐患整改闭环的一体化后台管理能力。

## 角色权限

系统支持 **5** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 安全主管

- 作业区域
- 风险源台账
- 作业票
- 隐患闭环
- 应急预案和统计看板维护

### - 审批负责人

- 作业票审批
- 审批链路流转和审批意见维护

### - 现场监护人

- 监护安排
- 监护记录
- 气体检测和隐患上报维护

### - 作业申请人

- 作业票申请
- 安全交底确认和整改反馈提交

## 特色亮点

- 选题贴合工业安全方向，覆盖作业票、审批链路、监护记录、隐患闭环
- 形成“风险辨识-作业票申请-分级审批-安全交底-现场监护-气体检测-隐患整改-闭环复盘”完整业务闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和状态完成，并按角色收口操作按钮
- 首页看板展示作业票、审批链路、监护记录、隐患闭环指标和状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 通用业务台账、状态流转和统计看板联动
- ECharts 展示业务趋势和状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 工厂危险作业审批与监护巡检管理系统类毕设项目
- 工业安全、业务审批、台账管理和统计分析课程设计
- 多角色后台管理系统综合实战
- 需要展示完整业务闭环和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 152-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 152-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
