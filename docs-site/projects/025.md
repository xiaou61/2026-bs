## 特色亮点

- 完整物业流程：覆盖缴费、报修、公告等核心物业业务
- 模拟支付功能：实现物业费、停车费的模拟支付流程
- 访客通行管理：规范外来人员访问，提升社区安全
- 车位资源管理：可视化管理社区车位资源
- 现代化UI设计：Element Plus构建的简洁美观界面

## 演示流程

1. 管理员录入业主信息 → 绑定房屋
2. 管理员生成物业费账单 → 推送给业主
3. 业主登录 → 查看待缴账单 → 点击支付
4. 业主提交报修申请 → 上传故障图片
5. 管理员指派维修 → 更新维修状态
6. 访客到达 → 登记信息 → 获得通行权限

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 025-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/db_025
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 025-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
