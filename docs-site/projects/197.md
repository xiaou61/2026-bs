## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 服务站管理员

- 维护服务站点
- 居民档案
- 人员档案和服务项目规则

### - 派单调度

- 处理服务预约
- 预约审核
- 人员排班和服务进度流转

### - 家政人员

- 查看排班任务
- 记录上门服务过程并反馈服务状态

### - 品控专员

- 跟踪信用评价
- 投诉处理和服务质量闭环

### - 社区居民

- 提交服务预约
- 查看上门记录
- 完成信用评价并发起投诉

- 社区家政服务预约与人员信用评价系统类毕设项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 197-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 197-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
