## 特色亮点

- 选题贴合智能制造、设备运维、备件库存、寿命预测和维修决策方向
- 形成“设备资产-备件目录-库存出入库-使用记录-运行指标-故障记录-寿命预测-维保计划-采购申请-风险预警”完整闭环
- 使用 MyBatis-Plus 分页查询和通用 CRUD 服务复用，与 118 的 MyBatis 注解 SQL 路线继续交替
- 备件库存、出库领用、故障记录、寿命预测、维保计划、采购申请和风险预警都有状态流转动作
- 首页看板展示备件、库存、故障、预测、维保指标，以及剩余寿命趋势和备件风险分布

## 演示流程

1. 管理员维护账号权限、设备资产和备件目录
2. 设备管理员登记备件库存、入库记录和出库领用
3. 维保员记录备件装机、运行指标和故障处理过程
4. 分析员基于运行指标和故障记录生成寿命预测
5. 系统依据预测结果创建维保计划和采购申请
6. 风险预警进入处置闭环并同步库存安全状态
7. 首页看板查看寿命趋势、风险分布和维保决策指标

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 119-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 119-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
