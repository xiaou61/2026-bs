## 特色亮点

- 选题贴合高校实验设备共享、借用审批、违规追踪和维护复核方向
- 形成“设备档案-实验室档案-设备分类-借用人档案-预约申请-借用记录-使用登记-违规记录-维修工单-归还确认-巡检计划-通知公告”完整闭环
- 使用 MyBatis-Plus 分页查询与通用 CRUD 服务复用，与 150 的 MyBatis 注解 SQL 路线继续交替
- 预约申请、借用记录、违规记录、维修工单、归还确认和巡检计划都有状态流转动作
- 首页看板展示设备、预约、登记、公告指标，以及设备预约趋势和设备状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis-Plus 分页查询与通用 CRUD 服务复用
- JWT 登录态拦截与 Redis Token 校验
- 设备预约、借用归还、违规追踪和巡检复核状态机联动
- ECharts 展示设备预约趋势和设备状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 高校实验设备共享预约与违规使用追踪系统类毕设项目
- 实验教学管理、设备共享、借用审批、违规追踪和巡检维护课程设计
- 多角色后台管理系统综合实战
- 需要展示实验设备共享闭环、违规追踪和维护管理能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 149-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 149-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
