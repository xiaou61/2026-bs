## 项目简介

本系统面向宠物医院门诊运营场景，提供宠主建档、宠物档案、医生排班、接诊预约、疫苗计划、接种记录、随访记录和费用结算的一体化后台管理能力。

## 技术栈

### 后端

- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 角色权限

系统支持 **5** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 前台接诊员

- 宠主档案
- 宠物档案
- 接诊预约
- 费用结算和统计看板维护

### - 宠物医生

- 医生档案
- 接诊记录
- 疫苗计划和随访记录维护

### - 护理接种员

- 疫苗计划
- 疫苗接种
- 药品库存和随访记录维护

### - 宠物主人

- 宠物档案
- 接诊预约
- 疫苗提醒
- 随访记录和费用结算查询

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 154-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 154-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
