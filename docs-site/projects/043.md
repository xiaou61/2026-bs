## 项目简介

本项目是基于 Spring Boot 3.2、MyBatis-Plus、JWT 的宠物寄养服务系统后端，提供用户登录注册、宠物档案、寄养服务商浏览、服务项目查询、寄养预约和订单状态管理等核心接口。

## 演示流程

1. 用户注册登录 → 进入首页仪表盘
2. 添加宠物 → 填写宠物名称、类型、年龄、健康状况等信息
3. 浏览寄养服务商 → 按评分排序、搜索筛选
4. 进入服务商详情 → 查看服务商信息和服务项目
5. 选择服务项目 → 点击预约按钮
6. 选择寄养宠物和日期 → 系统自动计算费用
7. 确认预约 → 生成预约订单
8. 我的预约 → 查看订单状态、取消待确认订单
9. 个人中心 → 管理个人信息

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 043-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 043-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
