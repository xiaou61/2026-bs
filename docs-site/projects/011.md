## 项目简介

### 1.1 项目背景
短视频已成为年轻人主要的内容消费和创作方式。校园场景下，学生希望记录和分享校园生活、才艺展示、学习经验等内容，但缺乏一个专属的校园短视频社区。本平台旨在打造一个校园版的短视频创作与分享平台，提供视频上传、社交互动、话题挑战、创作激励等功能。

## 特色亮点

- 类抖音沉浸式播放体验：全屏播放、上下滑动切换、双击点赞
- 智能推荐算法：热度分数 = 点赞×4 + 评论×6 + 转发×8 + 收藏×5 + 播放×0.5 + 时间衰减
- 完整的社交互动：点赞、评论、转发、收藏、关注、通知全流程
- 积分激励系统：发布视频+10、被点赞+2、被评论+3、自动升级
- 话题挑战机制：10个预置热门话题、话题视频聚合
- 实时消息通知：点赞、评论、关注自动推送
- 精美UI设计：Element Plus组件库、渐变背景、流畅动画
- 安全认证：JWT无状态认证、密码MD5加密、三层拦截

## 演示流程

1. 用户注册登录 → 进入推荐页
2. 上下滑动浏览视频 → 沉浸式播放体验
3. 点击发布 → 上传视频和封面 → 选择话题 → 发布成功
4. 点赞评论视频 → 创作者收到通知和积分
5. 关注喜欢的创作者 → 查看关注页视频流
6. 浏览话题页 → 查看话题下所有视频
7. 搜索感兴趣的内容 → 查看搜索结果
8. 个人中心 → 查看作品、喜欢、收藏、积分明细

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 011-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/campus_video
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 011-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
