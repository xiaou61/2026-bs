## 项目简介

本系统面向医院护理部、带教老师、考核老师与护理人员的培训管理场景，提供培训项目、护理人员档案、培训班次、培训签到、技能项目、技能考核、证书档案、续训提醒、实操记录、考核申诉和培训通知的一体化管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 护理培训管理员

- 维护培训项目
- 培训班次
- 技能项目和培训通知

### - 带教老师

- 组织班次培训
- 登记签到
- 维护实操记录和指导结果

### - 考核老师

- 维护技能考核
- 评分记录
- 申诉处理和考核结论

### - 护理人员

- 查看培训班次
- 完成签到
- 参与考核
- 查看证书和续训提醒

### - 护理人事

- 维护护理人员档案
- 证书档案和续训周期

- 选题贴合护理教育方向，覆盖培训项目、培训签到、技能考核、证书档案、续训提醒和考核申诉
- 形成“培训项目-护理人员档案-培训班次-培训签到-技能项目-技能考核-证书档案-续训提醒-实操记录-考核申诉-培训通知”完整业务闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和状态完成
- 首页看板展示培训项目、护理人员、培训签到和技能考核核心指标、近7日趋势和状态分布

- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 通用业务台账、状态流转和统计看板联动
- ECharts 展示业务趋势和状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 医疗护理培训签到考核与技能档案系统类毕设项目
- 护理教育、业务审批、台账管理和统计分析课程设计
- 多角色后台管理系统综合实战
- 需要展示完整业务闭环和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 188-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 188-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
