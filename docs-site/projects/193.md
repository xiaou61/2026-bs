## 项目简介

本系统面向高校创新实验班选拔、学院教务、评审专家、导师教师、辅导员与学生用户，提供实验班项目、学生档案、导师档案、选拔公告、报名选拔、资格评审、面试考核、导师匹配、培养计划、过程跟踪、成果档案和操作日志的一体化管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 教务管理员

- 维护实验班项目
- 学生档案
- 导师档案和选拔规则

### - 选拔评审

- 处理报名材料
- 资格评审和面试考核

### - 导师教师

- 维护导师档案
- 参与导师匹配
- 制定培养计划并跟踪过程

### - 辅导员

- 维护学生档案
- 跟进报名选拔和培养过程

### - 学生用户

- 提交报名申请
- 查看导师匹配
- 培养计划和过程反馈

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 193-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 193-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
