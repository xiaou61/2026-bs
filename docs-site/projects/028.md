## 角色权限

系统支持 **3** 种角色，各角色权限如下：

### 普通用户（学生/教职工）

- 用户特征：高校在校学生、教职工
- 核心需求：快速租车、便捷还车、费用透明
- 使用场景：宿舍-教学楼、食堂-图书馆等短途出行
- 痛点：找车困难、还车点满、费用不透明

### 系统管理员

- 用户特征：后勤管理人员、系统运营人员
- 核心需求：车辆管理、用户管理、数据统计
- 使用场景：日常运维、故障处理、数据分析
- 痛点：车辆调度困难、故障响应慢

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 028-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/db_028
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 028-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
