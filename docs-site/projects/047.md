## 项目简介

本系统是一个剧本杀创作与预约平台，支持玩家、店家、作者、管理员四类角色协同完成剧本创作、内容审核、店铺房间管理、玩家预约、店家确认完成、评价收藏和公告管理等流程。项目采用前后端分离架构，后端使用 Spring Boot 3 + MyBatis-Plus，前端使用 Vue 3 + TypeScript + Element Plus。

本轮巡检后，项目默认使用 H2 内存数据库自举，直接启动即可演示；如需连接 MySQL，可切换到 `mysql` profile。

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 047-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 047-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
