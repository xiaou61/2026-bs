## 项目简介

本系统面向实验动物饲养排班、伦理申请审批、实验登记、健康巡检、异常告警、兽医处置和耗材使用场景，提供动物房间、实验动物档案、饲养排班、饲养记录、伦理申请、伦理审批、实验登记、健康巡检、异常告警、兽医处置和耗材使用的一体化后台管理能力。

## 技术栈

### 后端

- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 实验室管理员

- 维护动物房间
- 动物档案
- 实验登记和耗材规则

### - 饲养员

- 查看饲养排班
- 登记饲养记录并上报异常告警

### - 课题研究员

- 提交伦理申请
- 登记实验过程并查看审批结果

### - 伦理委员

- 执行伦理审批
- 复核实验登记和审批留痕

### - 兽医

- 执行健康巡检
- 异常诊断和兽医处置

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 183-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 183-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
