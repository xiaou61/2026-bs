## 项目简介

本项目是 `053 基于 SpringBoot 的救灾物资调度与救援系统` 的后端，提供用户登录、灾情上报、物资分类、物资库存、仓库管理、物资调度、救援任务、公告和统计接口。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户管理** | 四角色系统（管理员/仓库管理员/救援人员/信息上报员）、JWT认证 |

| **公告管理** | 系统公告发布、草稿/发布状态 |
| **数据统计** | 首页数据概览、用户/灾情/仓库/调度统计 |

## 演示流程

1. 管理员登录 → 查看首页数据统计
2. 上报员登录 → 上报灾情信息（地点、类型、等级）
3. 管理员查看灾情 → 更新灾情状态为处理中
4. 仓库管理员 → 管理物资入库、维护库存
5. 管理员创建调度单 → 选择灾情、仓库、物资
6. 仓库管理员确认出库 → 更新调度状态为运输中
7. 管理员创建救援任务 → 分配救援人员
8. 救援人员查看任务 → 更新任务状态
9. 物资送达 → 更新调度状态为已送达
10. 灾情处理完毕 → 更新灾情状态为已结束

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 053-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 053-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
