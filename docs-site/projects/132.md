## 项目简介

医疗器械借用消毒与追踪管理系统面向毕业设计答辩场景，提供多角色登录、业务台账维护、状态流转、统计看板和操作留痕能力。

## 特色亮点

- 选题贴合医疗器械管理、借用归还、消毒追踪、二维码流转和风险预警方向
- 形成“器械档案-器械分类-科室信息-借用申请-借用记录-归还记录-消毒批次-消毒记录-二维码追踪-维护记录-巡检任务-风险预警”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 131 的 MyBatis-Plus 路线继续交替
- 借用申请、借用记录、归还记录、消毒批次、消毒记录、维护记录、巡检任务和风险预警都有状态流转动作
- 首页看板展示器械、借用、消毒、预警指标，以及器械借用趋势和风险等级分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 器械借用、归还、消毒、追踪、巡检和风险预警状态机闭环
- ECharts 展示器械借用趋势和风险等级分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 医疗器械借用消毒与追踪管理类毕设项目
- 医疗器械、医院供应室、二维码追踪、消毒记录和风险预警课程设计
- 多角色后台管理系统综合实战
- 需要展示借用归还、消毒追踪、巡检维护和风险预警闭环的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 132-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 132-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
