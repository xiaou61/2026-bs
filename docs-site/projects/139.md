|------|------|
| **账号权限** | 系统管理员/培训讲师/企业员工/部门主管四角色登录、JWT鉴权和Redis登录态 |

| **课程报名** | 报名编号、课程名称、学员姓名、报名时间、学习方式和状态维护 |
| **考核考试** | 考试编号、课程名称、考试方式、考试日期、监考讲师和状态维护 |

| **能力画像** | 画像编号、学员姓名、能力维度、当前等级、评估时间和状态维护 |
| **认证记录** | 认证编号、学员姓名、证书名称、认证机构、获取时间和状态维护 |

## 特色亮点

- 选题贴合企业培训管理、课程学习路径、能力画像、认证跟踪和学习提醒方向
- 形成“培训项目-课程目录-学员档案-学习路径-学习任务-课程报名-考核考试-考试成绩-技能标签-能力画像-认证记录-学习提醒”完整闭环
- 使用 MyBatis-Plus 分页查询与通用 CRUD 服务复用，与 140 的 MyBatis 注解 SQL 路线继续交替
- 学习路径、学习任务、课程报名、考核考试、考试成绩、能力画像和学习提醒都有状态流转动作
- 首页看板展示培训、报名、考试、认证指标，以及学习完成趋势和能力分布
- Spring Boot + Vue3 前后端分离
- MyBatis-Plus 分页查询与通用 CRUD 服务复用
- JWT 登录态拦截与 Redis Token 校验
- 学习任务、考试成绩、能力画像、认证记录和提醒状态机联动
- ECharts 展示学习完成趋势和能力等级分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 企业培训学习路径与能力画像系统类毕设项目
- 企业培训、人才发展、课程学习、岗位能力和认证管理课程设计
- 多角色后台管理系统综合实战
- 需要展示培训闭环、考试评估、能力画像和认证跟踪能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 139-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 139-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
