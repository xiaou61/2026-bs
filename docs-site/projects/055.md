## 项目简介

本项目是企业 OA 管理系统的 Spring Boot 后端，当前已调整为默认 H2 内存库自举，便于毕业设计合集在本机直接启动、测试和演示；如需连接 MySQL，可切换 `mysql` profile。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户管理** | 三角色系统（管理员/经理/员工）、JWT认证、个人中心、密码修改 |
| **部门管理** | 部门树形结构、部门负责人、子部门管理 |
| **考勤管理** | 签到签退、迟到早退检测（09:00-18:00）、考勤记录查询、月度统计 |
| **请假管理** | 请假申请、审批流程（经理审批）、多种假期类型（事假/病假/年假/婚假/产假） |
| **会议室管理** | 会议室CRUD、容量设备管理 |
| **会议管理** | 会议预约、参会人员选择、会议状态跟踪、时间冲突检测 |
| **公告通知** | 公告发布、置顶功能、草稿/发布状态、阅读标记 |
| **日程安排** | 个人日程管理、分类（工作/会议/其他）、提醒设置 |
| **文档管理** | 文档上传下载、分类管理、共享设置、下载统计 |
| **薪资管理** | 薪资录入、基本工资/奖金/补贴/扣款/社保、实发工资计算 |
| **工作日志** | 今日工作记录、明日计划、日志查看 |
| **数据统计** | 首页仪表盘、员工数/部门数/今日打卡/待审请假统计 |

## 演示流程

1. 员工登录 → 查看首页打卡状态和今日日程
2. 点击签到 → 系统记录签到时间 → 判断是否迟到
3. 发起请假申请 → 选择假期类型和时间 → 提交审批
4. 经理登录 → 查看待审请假 → 通过/拒绝
5. 预约会议 → 选择会议室和时间 → 邀请参会人员
6. 撰写工作日志 → 记录今日工作和明日计划
7. 查看公告通知 → 系统记录已读状态
8. 管理员录入薪资 → 员工查看本月工资明细
9. 下班签退 → 系统判断是否早退 → 完成当日考勤

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 055-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 055-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
