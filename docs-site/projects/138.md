## 项目简介

在线考试反作弊行为分析与证据管理系统面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

## 技术栈

### 后端

- Spring Boot 2.7.18

- PageHelper 1.4.7
- MySQL 8.0

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 功能模块

| 模块 | 说明 |
|------|------|
| **账号权限** | 系统管理员/监考老师/考生/复核员四角色登录、JWT鉴权和Redis登录态 |
| **考试计划** | 计划编号、考试名称、考试科目、开考时间、考试时长和状态维护 |
| **监考档案** | 监考编号、监考姓名、所属学院、负责考场、联系方式和状态维护 |
| **考生档案** | 考生编号、考生姓名、所属班级、考试终端、准考证号和状态维护 |
| **考试场次** | 场次编号、考试名称、考场名称、开考时间、监考老师和状态维护 |
| **异常行为** | 行为编号、场次编号、异常类型、触发时间、风险等级和状态维护 |
| **证据记录** | 证据编号、行为编号、证据类型、存证地址、采集时间和状态维护 |
| **复核任务** | 任务编号、行为编号、复核人、复核节点、下发时间和状态维护 |
| **复核结论** | 结论编号、任务编号、处理结论、处理意见、提交时间和状态维护 |
| **预警规则** | 规则编号、规则名称、识别策略、阈值配置、适用科目和状态维护 |
| **设备监测** | 监测编号、终端编号、网络状态、摄像状态、抓拍次数和状态维护 |
| **违规申诉** | 申诉编号、行为编号、申诉人、申诉理由、提交时间和状态维护 |
| **告警通知** | 通知编号、场次编号、告警类型、通知内容、接收人和状态维护 |
| **操作日志** | 关键模块操作人、动作、对象、详情和结果记录 |

## 特色亮点

- 选题贴合在线考试监管、异常行为识别、证据归档、人工复核和违规申诉方向
- 形成“考试计划-监考档案-考生档案-考试场次-异常行为-证据记录-复核任务-复核结论-预警规则-设备监测-违规申诉-告警通知”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 137 的 MyBatis-Plus 路线继续交替
- 异常行为、证据记录、复核任务、复核结论、设备监测、违规申诉和告警通知都有状态流转动作
- 首页看板展示考试、异常、证据、告警指标，以及异常告警趋势和异常类型分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 异常识别、证据留存、复核处理、申诉闭环和告警通知状态机联动
- ECharts 展示异常告警趋势和异常类型分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 在线考试反作弊行为分析与证据管理系统类毕设项目
- 教育安全、在线考试监管、行为识别、证据存档和复核申诉课程设计
- 多角色后台管理系统综合实战
- 需要展示考试监管、异常识别、证据管理和违规处理闭环的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 138-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 138-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
