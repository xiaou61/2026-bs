## 功能模块

| 模块 | 说明 |
|------|------|
| **用户系统** | 用户注册与登录、个人信息管理（身高、体重、年龄、性别等）、密码修改与找回 |
| **饮食记录** | 按餐次记录饮食（早餐/午餐/晚餐/加餐）、食物搜索与选择、食用量输入、自动计算营养摄入、历史记录查询 |
| **食物库管理** | 丰富的预置食物数据库、按分类浏览食物、食物搜索功能、自定义食物添加、食物营养成分查看 |
| **营养分析** | 每日营养摄入统计、营养素达标率分析、热量摄入趋势图、三大营养素比例图、营养报告生成 |
| **健康数据管理** | 体重记录与追踪、BMI指数计算、体脂率记录、血压心率记录、睡眠运动记录、健康趋势图表 |
| **营养目标设定** | 设定减重/增重/维持目标、目标体重设定、每日热量目标、营养素目标配比、目标达成进度追踪 |
| **食谱推荐** | 健康食谱浏览、按分类筛选食谱、食谱详情查看、食谱收藏功能、食谱热度排行 |
| **数据统计与报告** | 周/月营养统计、体重变化曲线、饮食习惯分析、健康报告生成、数据导出功能 |

## 演示流程

1. 注册登录 → 填写个人信息（身高、体重、年龄、性别）→ 系统自动计算BMI
2. 设定目标 → 选择目标类型（减重/增重/维持）→ 输入目标体重 → 系统智能推荐每日热量
3. 浏览食物库 → 搜索食物 → 查看营养成分 → 添加自定义食物
4. 记录饮食 → 选择餐次 → 搜索食物 → 输入食用量 → 系统自动计算营养摄入
5. 查看营养分析 → 每日摄入统计 → 营养素达标率 → 热量趋势图 → 营养比例图
6. 记录健康数据 → 输入体重、体脂率、血压等 → 系统自动计算BMI → 生成健康趋势图
7. 查看目标进度 → 目标达成率 → 体重变化曲线 → 营养目标对比
8. 浏览食谱 → 筛选食谱分类 → 查看食谱详情 → 收藏喜欢的食谱
9. 数据统计 → 周/月营养统计 → 饮食习惯分析 → 生成健康报告
10. 个人中心 → 修改个人信息 → 查看历史记录 → 导出数据

### 后端启动

```bash
cd 038-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 038-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
