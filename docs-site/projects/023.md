## 特色亮点

- 协同过滤推荐算法：基于用户行为相似性推荐课程
- 学习路径优化：Dijkstra算法规划最佳学习路径
- 知识掌握度预测：基于遗忘曲线预测知识点掌握情况
- 智能问答：集成NLP技术，提供智能学习辅导
- 数据可视化：ECharts展示学习进度和能力模型
- 前后端一体化：Thymeleaf服务端渲染，部署便捷

## 演示流程

1. 学生登录 → 完成能力评估测试 → 生成初始学习路径
2. 浏览推荐课程 → 基于协同过滤算法推荐
3. 开始学习 → 观看视频/阅读文档 → 记录学习行为
4. 遇到问题 → 提问智能助手 → 获得解答和引导
5. 查看学习仪表盘 → ECharts展示学习进度和能力雷达图
6. 系统根据学习情况 → 动态调整后续学习路径

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 023-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/db_023
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 023-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
