## 功能模块

| 模块 | 说明 |
|------|------|
| **账号权限** | 管理员/补货员/顾客三角色登录与权限控制 |
| **点位设备** | 点位信息、售货机编号、温区、负责人和在线状态维护 |
| **商品库存** | 商品分类、商品资料、货道容量、库存预警和状态管理 |
| **补货维护** | 补货员按设备货道登记补货并自动回写库存 |
| **订单支付** | 顾客选择设备货道下单、余额支付、查看订单与取货码 |
| **出货追踪** | 自动生成出货记录，跟踪成功、待处理与异常结果 |
| **反馈公告** | 顾客故障反馈提交、后台处理闭环与系统公告发布 |
| **经营看板** | 今日销售额、订单趋势、热销商品和设备销售排行分析 |

## 演示流程

1. 管理员维护点位、售货机、商品分类、商品资料和设备货道
2. 补货员登录后台查看货道库存，完成补货登记并同步设备库存
3. 顾客浏览在线设备和商品，选择货道创建订单
4. 顾客在钱包中心充值后完成余额支付，系统自动写入出货记录
5. 顾客遇到异常可提交故障反馈，后台处理后通过经营看板查看整体运营数据

## 快速启动

| Maven | 3.8+ |
| MySQL | 8.0+ |

### 后端启动

```bash
cd 093-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 093-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
