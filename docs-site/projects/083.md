## 项目简介

本项目面向体检中心、社区卫生服务机构和养老机构，提供老人档案、体检项目与套餐、预约登记、结果录入、异常预警、随访管理、公告和统计看板能力。

## 特色亮点

- 面向老年体检场景的完整业务闭环，覆盖建档到随访
- 异常指标自动触发预警，减少人工漏判
- Redis + JWT 支持高并发下的会话校验
- 统一 CRUD 页面模板，方便二开扩展新业务
- ECharts 展示预约趋势与异常等级分布

## 演示流程

1. 前台登录后新增老人档案并创建体检预约
2. 医生录入检查结果并自动生成异常预警
3. 护士在预警中心跟踪异常处理状态
4. 医生创建随访记录并形成健康管理闭环
5. 管理员发布通知并在看板查看核心指标

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 083-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 083-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
