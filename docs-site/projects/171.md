## 项目简介

本系统面向应急物资储备、库存盘点、申领调拨、审批出库、调度配送和库存预警场景，提供储备仓库、物资分类、物资台账、库存批次、库存盘点、盘点差异、申领工单、调拨审批、调度任务、出库记录和库存预警的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 仓储管理员

- 维护储备仓库
- 物资台账
- 库存批次和出库记录

### - 盘点人员

- 执行库存盘点
- 登记盘点差异并跟踪差异处理

### - 申领单位

- 提交申领工单并查看调拨审批
- 调度和出库进度

### - 调度人员

- 维护申领工单
- 调拨审批
- 调度任务和库存预警

### - 审核人员

- 审核调拨申请
- 复核盘点差异并查看应急储备统计

## 特色亮点

- 选题贴合应急管理、储备仓库、物资保障、库存盘点、调拨审批和应急调度方向
- 形成“储备仓库-物资分类-物资台账-库存批次-库存盘点-盘点差异-申领工单-调拨审批-调度任务-出库记录-库存预警”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与前序项目保持轻量可读的后端结构
- 库存盘点、盘点差异、申领工单、调拨审批、调度任务和出库记录都有状态流转动作
- 首页看板展示仓库、物资、库存批次和盘点指标，以及近7日盘点与调拨出库趋势和物资状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 储备台账、库存盘点、调拨审批和出库调度状态流转联动
- ECharts 展示物资盘点趋势和库存状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 应急物资储备盘点与调拨审批平台类毕设项目
- 应急管理、物资储备、库存盘点、调拨审批和出库调度课程设计
- 多角色后台管理系统综合实战
- 需要展示应急储备闭环、库存预警、审批调度和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 171-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 171-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
