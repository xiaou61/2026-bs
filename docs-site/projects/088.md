## 项目简介

### 项目简介
孩童收养信息管理系统面向儿童福利机构与社会申请人场景，提供儿童档案管理、收养申请、资格审核、家访评估、匹配审批、协议签订、回访跟踪和公告展示等功能，形成从信息发布到收养后回访的完整业务闭环。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户权限** | 管理员/审核员/申请人三角色登录与会话校验 |

| **回访管理** | 签约后回访记录、下次回访计划维护 |
| **公告看板** | 公告发布展示与业务统计可视化 |

## 特色亮点

- 同时提供机构后台与申请人门户，适合毕设演示完整双端流程
- 形成“儿童档案-在线申请-审核评估-匹配审批-签约回访”闭环
- MyBatis-Plus + JWT + Redis 组合实现高效率开发与登录态校验
- 儿童公开信息和申请进度分端展示，角色边界清晰
- 统计看板聚合开放儿童、申请总量、签约数量和回访情况

## 演示流程

1. 管理员维护儿童档案、健康信息与系统公告
2. 申请人注册登录后完善资料并浏览可申请儿童信息
3. 申请人提交收养申请并补充相关材料
4. 审核员完成材料初审、家访评估与匹配审批
5. 管理员登记协议并在后续回访模块持续跟踪

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 088-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 088-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
