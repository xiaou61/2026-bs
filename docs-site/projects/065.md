## 项目简介

本项目是名城小区物业管理系统后端服务，配套 `065-frontend` 使用。系统覆盖登录注册、用户管理、楼栋房屋、物业缴费、报修工单、投诉建议、访客登记、车位管理、公告和物业看板等流程。

### - ADMIN

- 系统管理员

### - STAFF

- 物业人员
- 负责收费
- 报修
- 投诉
- 访客等业务处理

- 业主
- 负责查看房屋

- 报修
- 投诉与访客申请

## 演示流程

1. 管理员维护楼栋、房屋、业主账号与车位信息
2. 物业人员按月生成物业费账单并跟踪缴费状态
3. 业主在线提交报修、投诉与访客登记申请
4. 物业人员处理报修工单并回复投诉与访客审批结果
5. 管理员在看板查看待缴费、待处理工单和收费趋势

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 065-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 065-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
