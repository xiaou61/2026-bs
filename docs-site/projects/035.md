## 演示流程

1. 农户注册登录 → 主角色为 FARMER
2. 农户添加地块 → 填写地块名称、面积、位置
3. 农户创建预约 → 选择地块、设置预约时间
4. 管理员或机手查看预约列表 → 选择合适的预约
5. 管理员指派设备 → 选择机手的收割机设备
6. 机手开始服务 → 点击开工 → 状态变为服务中
7. 服务完成后 → 点击完工 → 状态变为已完成
8. 农户查看预约历史 → 查看状态日志
9. 机手管理设备 → 添加/编辑/删除设备信息
10. 访问 Knife4j 文档 → `/swagger-ui/index.html` 测试接口

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 035-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 035-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
