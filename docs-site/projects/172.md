# 口腔门诊治疗预约与耗材计费管理系统

> 基于SpringBoot+Vue3的口腔门诊治疗预约与耗材计费管理系统

## 项目概览

| 属性 | 内容 |
|------|------|
| 项目编号 | 172 |
| 项目名称 | 口腔门诊治疗预约与耗材计费管理系统 |

## 项目简介

本系统面向口腔门诊预约、签到分诊、治疗记录、耗材库存、耗材使用和费用结算场景，提供诊室档案、医生档案、患者档案、治疗项目、预约诊疗、签到分诊、治疗记录、耗材目录、耗材库存、耗材使用和费用结算的一体化后台管理能力。

## 技术栈

### 后端

- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 前台导诊

- 维护患者档案
- 预约诊疗
- 签到分诊和诊室安排

### - 口腔医生

- 维护医生档案
- 治疗项目
- 治疗记录和耗材使用

### - 诊疗护士

- 协助签到分诊
- 耗材目录
- 耗材库存和耗材使用登记

### - 收费人员

- 维护治疗项目价格
- 耗材计费
- 费用结算和收费状态

### - 患者用户

- 查看预约诊疗
- 治疗记录
- 耗材计费和结算进度

- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 口腔门诊治疗预约与耗材计费管理系统类毕设项目
- 医疗门诊、口腔治疗、耗材库存、耗材计费和费用结算课程设计
- 多角色后台管理系统综合实战
- 需要展示门诊接诊闭环、治疗记录、耗材计费和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 172-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 172-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
