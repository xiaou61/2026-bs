## 特色亮点

- 选题贴合绿色低碳、家庭能源管理、设备能耗分析、节能建议和异常提醒方向
- 形成“家庭档案-成员-智能电表-用电设备-能耗读数-电费账单-设备用电-预算-节能建议-异常提醒-碳排统计-维护工单”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 125 的 MyBatis-Plus 路线继续交替
- 电费账单、设备用电、能耗预算、节能建议、异常提醒、碳排统计和维护工单都有状态流转动作
- 首页看板展示家庭、电表、设备、异常指标，以及能耗趋势和设备类型分布

## 演示流程

1. 管理员维护账号权限、家庭档案、家庭成员和智能电表
2. 住户登记用电设备并查看能耗读数和电费账单
3. 分析员配置能耗预算并生成设备用电分析
4. 系统根据高耗能设备生成节能建议和异常提醒
5. 维护员处理设备异常和电表维护工单
6. 系统汇总碳排统计并沉淀操作日志
7. 首页看板查看用电趋势、设备分布和节能异常指标

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 126-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 126-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
