## 特色亮点

- 形成“任务发布-学生评价-结果分析-教学改进”业务闭环
- 同一学生同一任务防重复提交，评价数据更可靠
- 指标启用权重总和限制，保障评分模型可解释性
- JWT + Redis 实现登录态双校验与失效控制
- ECharts 展示月度评价趋势与课程均分TOP5

## 演示流程

1. 管理员维护课程分类、数学课程与评价指标
2. 管理员发布课程评价任务并设置评价时间区间
3. 学生登录后选择任务提交多指标评分与建议
4. 教师/管理员查看任务评价记录与指标均分统计
5. 首页看板查看评价趋势与课程均分排行

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 085-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 085-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
