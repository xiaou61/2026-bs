## 演示流程

1. 用户登录系统，进入首页看板查看平台概览
2. 管理员维护工艺分类，设置分类排序和状态
3. 卖家发布手工艺作品，填写工艺名称、作坊、价格、库存、配送方式
4. 买家进入手作市集，按关键词和分类筛选商品并查看详情
5. 买家创建订单并支付，订单进入待发货状态
6. 卖家处理售出订单并发货，买家确认收货完成交易
7. 买家对已完成订单进行评价并可收藏喜欢作品
8. 若产生争议，买家发起申诉，管理员介入处理并回复
9. 管理员在后台查看订单趋势、分类分布与热销商品
10. 管理员发布公告，前台用户同步查看平台通知

## 快速启动

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |

### 后端启动

```bash
cd 074-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 074-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
