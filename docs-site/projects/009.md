## 项目简介

基于 Spring Boot 3 + MyBatis-Plus 开发的校园快递代收管理系统后端服务。

**前端：** 基于 Vue 3 + Element Plus 开发的校园快递代收管理系统前端项目。

## 特色亮点

- 自动化程度高：取件码生成、通知发送、超期计算全自动
- 定时任务：每日凌晨自动计算超期并发送提醒
- 数据可视化：折线图（趋势）、饼图（占比）、柱状图（对比）
- 批量处理：Excel批量导入导出，提升效率
- 超期管理：分阶梯收费（4-7天1元/天，8天+2元/天）
- 角色清晰：四种角色布局和权限完全独立
- 完整日志：所有操作可追溯审计

## 演示流程

1. 快递员批量导入快递 → 自动生成取件码
2. 系统自动发送通知给学生
3. 学生查看待取快递列表（卡片展示取件码）
4. 工作人员扫码核销取件
5. 定时任务自动计算超期并提醒
6. 管理员查看数据统计图表

### 后端启动

```bash
cd 009-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/express_system
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 009-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
