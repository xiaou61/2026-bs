## 项目简介

本系统面向物流园区车辆入场与月台调度场景，提供承运商预约、车辆档案、司机档案、时段计划、门岗核验、排队叫号、道口资源、道口分配、装卸任务和周转统计的一体化后台管理能力。

## 角色权限

系统支持 **5** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 调度主管

- 承运商档案
- 车辆档案
- 时段计划
- 道口资源和周转统计维护

### - 门岗核验员

- 门岗核验
- 车辆放行
- 排队叫号和异常入场处理

### - 场内调度员

- 时段计划
- 排队叫号
- 道口资源
- 道口分配和装卸任务维护

### - 承运商代表

- 入场预约
- 排队叫号
- 道口分配和装卸任务查看

- 物流园区车辆入场预约与道口调度平台类毕设项目
- 智慧物流、园区调度、月台道口管理、承运商协同和装卸周转课程设计
- 多角色后台管理系统综合实战

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 157-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 157-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
