## 项目简介

本系统面向医院病区、陪护协调员、护工人员、结算人员与患者家属，提供病区档案、患者档案、护工档案、陪护预约、预约审核、护工排班、服务派单、服务记录、家属沟通、护工评价、评价结算和操作日志的一体化管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 院区管理员

- 维护病区档案
- 患者档案
- 护工档案和陪护规则

### - 陪护协调员

- 处理陪护预约
- 预约审核
- 护工排班和服务派单

### - 护工人员

- 查看排班任务
- 执行陪护服务并登记服务记录

### - 结算人员

- 核对服务记录
- 护工评价和费用结算状态

### - 患者家属

- 提交陪护预约
- 查看服务记录
- 完成护工评价和结算确认

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 192-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 192-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
