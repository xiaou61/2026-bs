## 项目简介

本系统面向养老助餐、营养套餐配置、助餐订单、配送排线、配送任务、签收回执、饮食禁忌、营养分析、补贴记录和回访关怀场景，提供助餐站点、老人档案、营养套餐、助餐订单、配送排线、配送任务、签收回执、饮食禁忌、营养分析、补贴记录和回访关怀的一体化后台管理能力。

## 技术栈

### 后端

- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 助餐中心

- 维护助餐站点
- 老人档案
- 助餐订单和服务规则

### - 营养师

- 维护营养套餐
- 饮食禁忌和营养分析

### - 调度员

- 维护配送排线
- 分配配送任务并跟踪签收进度

### - 配送员

- 接收配送任务
- 登记配送过程并提交签收回执

### - 老人用户

- 查看套餐
- 提交助餐订单
- 确认签收并查看回访关怀

## 功能模块

| 模块 | 说明 |
|------|------|
| **账号权限** | ADMIN / CENTER / DIETITIAN / DISPATCH / COURIER / ELDER 六类角色登录、JWT鉴权和Redis登录态 |
| **助餐站点** | 站点编号、服务区域、负责人、供餐能力、联系电话和启停状态维护 |
| **老人档案** | 老人姓名、居住地址、联系人、年龄、补贴类型、健康标签和助餐状态维护 |
| **营养套餐** | 套餐名称、餐次类型、热量、蛋白质、适用人群、禁忌提示和上架状态配置 |
| **助餐订单** | 订单编号、老人信息、套餐、配送日期、餐次数量、补贴金额和订单状态管理 |
| **配送排线** | 线路编号、覆盖区域、司机/配送员、计划时间、预计单量和排线状态维护 |
| **配送任务** | 任务编号、关联订单、配送员、出餐时间、送达时间、异常说明和任务状态跟踪 |
| **签收回执** | 签收方式、签收人、签收时间、满意度、照片说明和回执结果登记 |
| **饮食禁忌** | 老人禁忌标签、忌口原因、提醒等级、医嘱说明和生效状态维护 |
| **营养分析** | 周期摄入、热量达标、蛋白质达标、风险提示和营养师建议记录 |
| **补贴记录** | 补贴类型、订单金额、政府补贴、自付金额、结算周期和核销状态维护 |
| **回访关怀** | 回访对象、回访方式、满意反馈、异常诉求、处理人和跟进状态管理 |
| **操作日志** | 关键模块操作人、动作、对象、详情和结果记录 |

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 182-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 182-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
