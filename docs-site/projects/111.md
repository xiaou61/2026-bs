## 项目简介

网络钓鱼邮件演练与安全意识培训平台面向企业安全运营场景，围绕员工档案、邮件模板、演练活动、目标名单、发送记录、点击追踪、培训课程、培训考试和风险评分构建完整闭环，帮助企业提升员工识别钓鱼邮件和保护账号凭据的能力。

## 角色权限

系统支持 **4** 种角色，各角色权限如下：

### - 管理员

- 维护账号
- 员工
- 部门和操作日志

### - 安全专员

- 维护邮件模板
- 演练活动
- 目标名单和点击追踪

### - 培训讲师

- 维护课程
- 考试
- 题目和考试记录

### - 审计员

- 查看发送记录
- 风险评分和演练结果

## 功能模块

| 模块 | 说明 |
|------|------|
| **账号权限** | 管理员/安全专员/培训讲师/审计员四角色登录、JWT鉴权和Redis登录态 |
| **员工档案** | 管理演练对象员工、邮箱、部门、岗位、风险等级和状态 |
| **部门分组** | 管理演练部门、业务线、负责人和员工规模 |
| **邮件模板** | 配置钓鱼邮件主题、发件人、诱导类型、落地页并支持发布、归档 |
| **演练活动** | 管理演练编号、模板、目标范围、计划时间并支持提交、启动、完成 |
| **目标名单** | 管理活动目标员工、部门、邮箱、风险标签和发送状态 |
| **发送记录** | 记录邮件发送编号、发送时间并支持已发送、已打开、失败 |
| **点击追踪** | 记录点击IP、点击时间、是否提交凭据并支持确认、培训、忽略 |
| **培训课程** | 管理安全意识课程、课程分类、讲师、课时并支持发布、归档 |
| **培训考试** | 管理安全考试、关联课程、及格分和考试状态 |
| **考试题目** | 管理题库题目、题型、答案、分值和启用状态 |
| **考试记录** | 记录员工考试成绩并支持通过、未通过、补训 |
| **风险评分** | 根据点击行为和考试成绩维护综合风险分、风险等级和整改状态 |
| **操作日志** | 记录关键操作对象、动作、详情和IP地址 |

## 特色亮点

- 选题贴合网络安全、安全意识培训和企业攻防演练方向，答辩场景清晰
- 形成“员工档案-邮件模板-演练活动-发送记录-点击追踪-课程考试-风险评分”完整闭环
- 使用 MyBatis-Plus BaseMapper + 通用 CRUD Service，与 110 的 MyBatis 路线继续交替
- 演练活动、邮件发送、点击追踪、课程考试、风险评分都有状态流转动作
- 首页看板展示员工数量、演练活动、点击追踪、风险评分、演练趋势和风险分布

## 演示流程

1. 管理员维护账号权限、员工档案和部门分组
2. 安全专员创建邮件模板和演练活动
3. 安全专员导入目标名单并记录邮件发送状态
4. 系统沉淀打开、点击和凭据提交追踪记录
5. 培训讲师发布安全意识课程和培训考试
6. 系统根据点击行为和考试成绩生成风险评分
7. 审计员查看演练趋势、风险分布和操作日志

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 111-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 111-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
