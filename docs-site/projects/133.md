## 功能模块

| 模块 | 说明 |
|------|------|
| **账号权限** | 管理员/库管员/实验教师/审批负责人四角色登录、JWT鉴权和Redis登录态 |
| **耗材目录** | 耗材编号、耗材名称、规格型号、计量单位、安全库存和状态维护 |
| **供应商档案** | 供应商编号、名称、联系人、联系电话、资质等级和状态维护 |
| **实验室** | 实验室编号、名称、楼栋位置、负责人、联系电话和状态维护 |
| **库存台账** | 库存编号、耗材名称、实验室、当前库存、锁定数量和状态维护 |
| **采购申请** | 申请编号、耗材名称、申请数量、申请人、申请时间和状态维护 |
| **采购审批** | 审批编号、申请编号、审批人、审批意见、审批时间和状态维护 |
| **采购订单** | 订单编号、供应商、耗材名称、订单金额、预计到货日期和状态维护 |
| **入库记录** | 入库编号、订单编号、耗材名称、入库数量、入库人和状态维护 |
| **领用记录** | 领用编号、耗材名称、领用实验室、领用数量、领用人和状态维护 |
| **库存盘点** | 盘点编号、实验室、耗材名称、账面数量、实盘数量和状态维护 |
| **预警规则** | 规则编号、耗材分类、最小库存、预警级别、通知对象和状态维护 |
| **库存预警** | 预警编号、耗材名称、当前库存、预警级别、处理人和状态维护 |
| **操作日志** | 关键模块操作人、动作、对象、详情和结果记录 |

## 特色亮点

- 选题贴合实验室管理、耗材采购审批、库存台账、出入库流转和预警提醒方向
- 形成“耗材目录-供应商档案-实验室-库存台账-采购申请-采购审批-采购订单-入库记录-领用记录-库存盘点-预警规则-库存预警”完整闭环
- 使用 MyBatis-Plus 分页查询与通用 CRUD 服务复用，与 132 的 MyBatis 注解 SQL 路线继续交替
- 采购申请、采购审批、采购订单、入库记录、领用记录、库存盘点和库存预警都有状态流转动作
- 首页看板展示耗材、库存、采购、预警指标，以及采购需求趋势和库存风险分布
- Spring Boot + Vue3 前后端分离
- MyBatis-Plus 分页查询与通用 CRUD 服务复用
- JWT 登录态拦截与 Redis Token 校验
- 耗材采购、审批、订单、入库、领用、盘点和预警状态机闭环
- ECharts 展示采购需求趋势和库存风险分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 实验室耗材采购审批与库存预警系统类毕设项目
- 实验室管理、耗材采购、库存预警、领用登记和供应商管理课程设计
- 多角色后台管理系统综合实战
- 需要展示耗材采购、审批订单、出入库、库存盘点和预警处置闭环的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 133-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 133-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
