## 项目简介

本系统面向高校宿舍能源管理场景，提供宿舍楼栋、宿舍房间、能耗表计、用电读数、能耗账单、预警策略、异常预警、节能任务、节能排行和巡查记录的一体化后台管理能力。

### 后端

- Spring Boot 2.7.18

- PageHelper 1.4.7

- JWT 认证

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 角色权限

系统支持 **5** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 后勤管理员

- 宿舍楼栋
- 宿舍房间
- 能耗账单
- 节能排行和通知公告维护

### - 宿舍辅导员

- 异常预警
- 巡查记录
- 节能任务和学生沟通维护

### - 能耗专员

- 能耗表计
- 用电读数
- 预警策略
- 异常预警和巡查记录维护

### - 学生代表

- 用电读数
- 能耗账单
- 节能任务和节能排行查看

- 选题贴合智慧校园与宿舍能源治理方向，覆盖楼栋房间、表计读数、账单核算、异常预警、节能任务和节能排行

- 使用 MyBatis 注解 SQL + PageHelper 分页查询

- 智慧校园、宿舍能源管理、异常用电预警、节能激励和后勤巡查课程设计
- 多角色后台管理系统综合实战
- 需要展示完整业务闭环和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 156-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 156-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
