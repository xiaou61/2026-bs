## 项目简介

AIGC 图片内容审核与版权存证平台面向 AI 图片生成、设计素材管理和版权确权场景，提供图片作品入库、内容审核、风险标签、版权登记、存证编号、授权记录、申诉处理和审计统计能力。系统帮助平台方对 AIGC 图片进行合规审核、版权归属登记和后续争议处理。

## 功能模块

| 模块 | 说明 |
|------|------|
| **账号权限** | 管理员/审核员/创作者三角色登录、JWT鉴权和操作权限控制 |
| **图片作品** | 维护作品标题、图片地址、提示词、模型来源、分类和审核状态 |
| **审核规则** | 配置内容安全、版权相似、肖像风险和商标风险关键词规则 |
| **审核任务** | 创建作品审核任务，执行规则匹配并生成评分、风险等级和建议 |
| **审核结果** | 查看命中规则、结论、建议并支持人工复核通过或驳回 |
| **风险标签** | 维护风险标签类型、颜色、说明和启停状态 |
| **版权登记** | 提交权利声明，审核登记申请并记录审核意见 |
| **电子存证** | 对已通过登记生成存证编号、哈希值和存证平台记录 |
| **授权申诉** | 管理授权使用、撤销授权、侵权线索和申诉处理 |
| **统计审计** | 首页指标、图表看板和操作日志审计 |

## 特色亮点

- 选题贴合 AIGC 内容治理、版权保护和生成式 AI 合规热点，适合 AI 治理方向答辩
- 形成“作品上传-规则审核-人工复核-版权登记-电子存证-授权使用-侵权申诉”完整业务闭环
- 审核任务可根据启用规则自动计算风险评分和风险等级，演示时动作链路清晰
- 存证模块生成存证编号和 SHA-256 哈希值，能体现版权证据固化和可信记录思路
- 前端覆盖 14 个管理页面，支持搜索、分页、弹窗编辑、任务执行、登记审核和线索处理

## 演示流程

1. 管理员登录后维护用户、审核规则、风险标签和操作日志
2. 创作者新增 AIGC 图片作品，填写图片地址、提示词、生成模型和作品描述
3. 审核员创建审核任务并点击执行，系统自动生成审核结果、评分和风险建议
4. 审核员在审核结果页进行人工复核，填写通过或驳回意见
5. 创作者提交版权登记申请，审核员通过登记后进入电子存证
6. 创作者输入登记ID生成存证编号和哈希凭证，并维护授权使用记录
7. 创作者提交侵权线索或申诉，审核员处理后在看板查看待办和风险数据

### 后端启动

```bash
cd 099-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 099-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
