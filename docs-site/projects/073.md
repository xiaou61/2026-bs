## 项目简介

### 项目简介
人事管理系统是一套面向企业的人力资源管理解决方案，涵盖员工信息管理、部门组织架构、考勤打卡、薪资核算、招聘管理、培训管理、绩效考核等核心功能，帮助企业实现人事管理的数字化、规范化。

## 演示流程

1. 管理员登录 → 配置部门和职位 → 设置薪资范围
2. HR发布招聘职位 → 设置职位要求和人数
3. 求职者投递简历 → HR筛选简历 → 安排面试
4. 面试通过 → 录用入职 → 创建员工档案
5. 签订劳动合同 → 合同信息录入系统
6. 员工每日打卡 → 系统记录考勤状态
7. 员工提交请假申请 → HR审批 → 更新考勤
8. 月底薪资核算 → 计算应发实发工资 → 发放确认
9. HR创建培训计划 → 员工报名参加 → 培训评估
10. 查看数据看板 → 员工统计、考勤概览、薪资分析
11. 员工登录 → 查看个人信息、考勤记录、薪资条
12. 合同到期提醒 → HR处理续签或离职

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 073-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 073-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
