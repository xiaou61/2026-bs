## 项目简介

本项目是家电售后维修系统，包含 Spring Boot 后端、Vue 管理端和 uni-app 小程序端。核心流程包括用户报修、管理员派单、技师处理、维修进度、用户支付评价、公告和统计看板。

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |

### 后端启动

```bash
cd 081-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 小程序启动

1. 使用微信开发者工具打开 `081-miniprogram` 目录
2. 确认 AppID 配置正确
3. 修改后端接口地址（通常在 config.js 或 utils/request.js 中）
4. 点击编译运行
