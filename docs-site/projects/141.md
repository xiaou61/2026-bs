## 项目简介

固定资产 RFID 盘点与借用归还系统面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

|------|------|
| **账号权限** | 系统管理员/资产管理员/借用员工/财务专员四角色登录、JWT鉴权和Redis登录态 |

| **盘点明细** | 明细编号、任务名称、资产名称、扫描时间、盘点结果和状态维护 |
| **借用申请** | 申请编号、资产名称、借用用途、借用日期、借用人和状态维护 |
| **借用归还** | 归还编号、资产名称、归还人、归还数量、归还时间和状态维护 |
| **维修记录** | 维修编号、资产名称、维修类型、责任部门、维修费用和状态维护 |
| **折旧记录** | 折旧编号、资产名称、资产原值、累计折旧、计提日期和状态维护 |
| **闲置处置** | 处置编号、资产名称、处置方式、处置原因、处置时间和状态维护 |
| **预警通知** | 通知编号、资产名称、预警类型、预警内容、发送时间和状态维护 |
| **操作日志** | 关键模块操作人、动作、对象、详情和结果记录 |

## 特色亮点

- 选题贴合固定资产管理、RFID 快速盘点、借用归还、维修折旧和预警提醒方向
- 形成“资产档案-资产分类-RFID标签-存放位置-盘点任务-盘点明细-借用申请-借用归还-维修记录-折旧记录-闲置处置-预警通知”完整闭环
- 使用 MyBatis-Plus 分页查询与通用 CRUD 服务复用，与 142 的 MyBatis 注解 SQL 路线继续交替
- 盘点任务、借用申请、借用归还、维修记录、折旧记录和闲置处置都有状态流转动作
- 首页看板展示资产、盘点、借用、预警指标，以及资产流转趋势和状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis-Plus 分页查询与通用 CRUD 服务复用
- JWT 登录态拦截与 Redis Token 校验
- RFID 盘点、借用归还、维修折旧和预警状态机联动
- ECharts 展示资产流转趋势和资产状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 固定资产 RFID 盘点与借用归还系统类毕设项目
- 资产管理、智能盘点、借用归还、折旧核算和预警管理课程设计
- 多角色后台管理系统综合实战
- 需要展示 RFID 盘点闭环、借还追踪、折旧处理和预警通知能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 141-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 141-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
