## 项目简介

本系统面向高校学生社团活动预算、报销票据与物资借还协同场景，提供社团档案、成员档案、活动立项、预算申请、预算明细、审批记录、报销申请、票据归档、物资台账、物资借用和归还验收的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 团委审核员

- 社团档案
- 活动立项
- 预算审批和经费使用复核

### - 社团负责人

- 活动立项
- 预算申请
- 报销申请和成员协同维护

### - 社团财务员

- 预算明细
- 报销申请
- 票据归档和费用核对维护

### - 物资管理员

- 物资台账
- 物资借用
- 归还验收和库存状态维护

### - 社团成员

- 活动申请
- 物资借用
- 归还反馈和报销进度查看

## 特色亮点

- 选题贴合高校社团财务与资产协同方向，覆盖活动立项、预算申请、报销票据、物资借用和归还验收
- 形成“社团档案-成员档案-活动立项-预算申请-预算明细-审批记录-报销申请-票据归档-物资台账-物资借用-归还验收”完整业务闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和归档完成
- 首页看板展示预算借用趋势、审批状态分布和社团财资核心指标
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 按团委审核员、社团负责人、社团财务员、物资管理员、社团成员进行菜单、路由和操作权限收口
- ECharts 展示预算报销趋势、物资借用分布和社团经费使用概览
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和预算借用状态动作交互
- 校园社团活动预算报销与物资借用系统类毕设项目
- 校园社团、活动预算、报销票据、物资借还和资产验收课程设计
- 多角色后台管理系统综合实战
- 需要展示完整业务闭环和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 160-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 160-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
