## 项目简介

本项目是基于 Spring Boot 3.2、Spring Data JDBC、Spring Security、JWT 的民歌民谣交流平台后端，提供用户注册登录、民歌分类、民歌发布审核、点赞收藏、评论、公告和后台管理等接口。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户系统** | 注册登录、JWT认证、个人信息管理、角色权限（普通用户/管理员） |
| **民歌分类** | 按地域分类（山歌/号子/小调/田歌/牧歌/渔歌/茶歌/秧歌）、分类浏览 |
| **民歌发布** | 民歌上传发布、标题描述、分类选择、多媒体上传（图片/音频/视频）、审核机制 |
| **民歌浏览** | 民歌列表展示、分类筛选、搜索功能、民歌详情查看、多媒体播放 |
| **互动功能** | 点赞收藏、评论互动、回复评论、互动统计 |
| **审核管理** | 民歌审核流程、三种状态（待审核/已通过/已拒绝）、审核理由 |
| **系统公告** | 公告发布、公告列表、公告详情、首页置顶 |
| **文件上传** | 图片/音频/视频文件上传、文件类型校验、文件大小限制 |
| **管理后台** | 用户管理、分类管理、民歌审核、评论管理、公告管理、数据统计 |

## 演示流程

1. 用户注册登录 → JWT Token认证
2. 浏览首页 → 查看推荐民歌、热门分类
3. 按分类浏览 → 选择山歌/号子等分类 → 查看分类下的民歌列表
4. 民歌详情 → 播放音频视频 → 查看详细介绍 → 点赞收藏
5. 发表评论 → 互动交流 → 回复其他用户评论
6. 发布民歌 → 上传图片音频视频 → 填写标题描述 → 选择分类 → 提交审核
7. 管理员登录 → 审核待审核民歌 → 通过或拒绝并填写理由
8. 审核通过后 → 民歌展示在列表中 → 其他用户可浏览
9. 管理分类 → 添加编辑民歌分类
10. 发布公告 → 查看系统公告 → 首页展示
11. 查看数据统计 → 用户数量、民歌数量、评论数量等

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 039-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/db_039
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 039-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
