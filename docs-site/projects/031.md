## 演示流程

1. 用户注册登录 → 完善个人信息
2. 浏览赛事列表 → 筛选感兴趣的比赛
3. 进入赛事详情 → 查看球队、场馆、时间信息
4. 点击购票 → 选择座位区域 → 查看座位图
5. 选择座位（可多选） → 确认选择信息
6. 生成订单 → 选择支付方式 → 余额支付
7. 支付成功 → 生成电子票 → 获取二维码
8. 比赛当天 → 到场馆入口 → 扫码验票入场
9. 管理员后台 → 管理赛事、场馆、订单
10. 查看统计报表 → 分析销售数据

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 031-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 031-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
