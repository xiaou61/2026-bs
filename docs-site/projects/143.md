## 项目简介

社区公益时间银行互助服务平台面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

## 特色亮点

- 选题贴合社区公益互助、时间银行时长存取、志愿服务协同和互助兑换方向
- 形成“服务项目-服务分类-居民档案-志愿者档案-服务预约-服务签到-时长账户-互助兑换-评价反馈-公益活动-积分规则-站内通知”完整闭环
- 使用 MyBatis-Plus 分页查询与通用 CRUD 服务复用，与 144 的 MyBatis 注解 SQL 路线继续交替
- 服务预约、服务签到、时长账户、互助兑换、公益活动和站内通知都有状态流转动作
- 首页看板展示项目、预约、账户、通知指标，以及志愿服务趋势和状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis-Plus 分页查询与通用 CRUD 服务复用
- JWT 登录态拦截与 Redis Token 校验
- 志愿服务预约、签到、时长兑换和站内通知状态机联动
- ECharts 展示志愿服务趋势和服务状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 社区公益时间银行互助服务平台类毕设项目
- 社区治理、公益互助、志愿服务、积分激励和居民服务课程设计
- 多角色后台管理系统综合实战
- 需要展示时长存取闭环、志愿协同、积分兑换和活动管理能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 143-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 143-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
