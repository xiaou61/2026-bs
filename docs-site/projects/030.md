## 功能模块

| 模块 | 说明 |
|------|------|
| **用户管理** | 注册登录、角色权限（患者/医生/管理员）、个人信息管理 |
| **健康档案** | 完整的个人健康档案（病史/家族史/过敏史/生活习惯） |
| **健康数据** | 数据记录（血压/血糖/体重/心率）、数据可视化、趋势分析 |
| **在线咨询** | 患者发起咨询、医生回答、咨询历史、评价反馈 |
| **医生管理** | 医生认证、信息管理、工作台、排班管理 |
| **健康评估** | 健康指数计算、疾病风险评估、异常数据预警 |
| **健康知识** | 知识库、科普文章、视频教程 |
| **管理后台** | 医生审核、用户管理、数据统计 |

## 演示流程

1. 用户注册登录 → 选择角色（患者/医生）
2. 患者完善健康档案 → 填写病史、过敏史等信息
3. 记录健康数据 → 输入血压、血糖、体重等
4. 查看健康概览 → 数据趋势图表展示
5. 发起在线咨询 → 选择医生、描述问题
6. 医生查看待处理咨询 → 回复患者问题
7. 患者查看医生回复 → 进行评价反馈
8. 管理员审核医生认证 → 查看系统统计数据

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 030-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/db_030
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 030-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
