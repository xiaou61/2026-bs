## 项目简介

基于Spring Boot的志愿活动管理与积分激励平台，通过积分激励机制鼓励用户参与志愿活动，提供活动发布、报名、签到、积分兑换等功能。

## 技术栈

### 后端

- Spring Boot 3.x
- MyBatis-Plus
- MySQL 8.0
- JWT 认证

### 前端

- Vue 3
- Element Plus
- Axios

## 角色权限

系统支持 **2** 种角色，各角色权限如下：

- 活动管理管理
- 报名系统管理

### 志愿者

- 签到打卡
- 奖励商城
- 数据统计

## 功能模块

| 模块 | 说明 |
|------|------|

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 007-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/volunteer_db
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 007-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
