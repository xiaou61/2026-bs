## 项目简介

本系统面向非遗工坊、非遗讲师、展陈运营、展销财务和学员访客，提供工坊档案、传承人档案、课程目录、工坊排期、课程预约、预约审核、课程签到、作品档案、作品展销、展销订单、展销结算和操作日志的一体化管理能力。

## 技术栈

### 后端

- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 工坊管理员

- 维护工坊档案
- 传承人档案
- 课程目录和工坊排期

### - 非遗讲师

- 维护课程内容
- 处理课程预约
- 预约审核和课程签到

### - 展陈运营

- 维护作品档案
- 作品展销和展陈状态

### - 展销财务

- 核对展销订单和展销结算闭环

### - 学员访客

- 查看课程排期
- 提交课程预约
- 查看作品展销和订单记录

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 200-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 200-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
