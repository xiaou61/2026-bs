## 项目简介

本系统面向城市共享充电宝运营企业、点位商户、巡检人员、财务结算人员和合作商户，提供投放点位、设备柜档案、充电宝档案、点位投放、设备巡检、故障维修、异常回收、租借订单、商户收益、收益结算、库存调拨和操作日志的一体化管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 运营管理员

- 维护投放点位
- 设备柜档案
- 充电宝档案和点位投放计划

### - 点位商户

- 查看设备柜
- 充电宝状态
- 巡检维修和点位运营记录

### - 巡检人员

- 处理设备巡检
- 故障维修
- 异常回收和库存调拨

### - 财务结算

- 核对租借订单
- 商户收益和收益结算闭环

### - 合作商户

- 查看点位收益
- 结算记录和订单分成

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 198-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 198-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
