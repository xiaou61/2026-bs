## 项目简介

本项目是电影订票及评论网站后端服务，配套 `060-frontend` 使用。系统覆盖用户登录注册、电影浏览、影院影厅、排片、在线订票、收藏、评论、公告和数据看板等流程。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户管理** | 管理员/普通用户双角色、注册登录 |
| **电影分类** | 动作/喜剧/科幻/爱情/恐怖/动画/纪录片等 |
| **电影管理** | 电影CRUD、上下架、分类筛选 |
| **影院管理** | 影院信息、启用/停业 |
| **影厅管理** | 影厅CRUD、座位行列、厅型（普通/IMAX/巨幕/VIP） |
| **排片管理** | 排片CRUD、影院+影厅+电影关联、余座计算 |
| **在线订票** | 选座下单、订单状态流转（待支付→已支付→已完成/已取消） |
| **影评系统** | 用户评分+评论、电影评分自动计算 |
| **收藏功能** | 收藏/取消收藏电影 |
| **公告管理** | 系统公告发布与查看 |
| **数据看板** | 分类统计饼图、订单趋势折线图、票房排行柱状图 |

## 演示流程

1. 管理员登录 → 管理电影分类和电影信息 → 管理影院和影厅
2. 管理员创建排片 → 设置电影+影院+影厅+时间+票价
3. 用户注册登录 → 浏览电影列表 → 查看电影详情
4. 用户选择场次 → 选座购票 → 支付订单 → 查看我的订单
5. 用户发表影评 → 收藏电影 → 查看系统公告
6. 管理员查看数据看板 → 分类统计 → 订单趋势 → 票房排行

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 060-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 060-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
