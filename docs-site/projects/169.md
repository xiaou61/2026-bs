## 项目简介

本系统面向中小学校车乘车预约、学生上下车核验、异常处置与家校通知场景，提供校车线路、站点、车辆、司机、学生、家长、乘车预约、上车核验、下车核验、异常告警和家长通知的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 校方管理员

- 校车线路
- 站点
- 学生
- 家长和安全台账维护

### - 调度管理员

- 车辆
- 司机
- 乘车预约
- 上下车核验和异常调度维护

### - 校车司机

- 查看线路站点
- 维护上车核验
- 下车核验和异常上报

### - 跟车老师

- 维护学生乘车核验
- 异常处置和家长通知

### - 学生家长

- 提交乘车预约
- 查看学生上下车核验
- 异常告警和通知

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 169-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 169-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
