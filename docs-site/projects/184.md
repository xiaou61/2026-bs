## 项目简介

本系统面向停车场月租合同续费、车辆绑定、车位巡检、异常占位、违规处理和客服协同场景，提供停车场档案、车位档案、月租车主、月租合同、续费提醒、续费缴费、车辆绑定、占位巡检、异常占位、违规处理和客服工单的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 停车场管理员

- 维护停车场档案
- 车位档案
- 月租合同和车辆绑定

### - 收费财务

- 维护续费提醒
- 续费缴费
- 费用核对和合同续期状态

### - 巡检员

- 执行占位巡检
- 上报异常占位并协同违规处理

### - 月租车主

- 查看合同
- 续费缴费
- 绑定车辆并提交客服工单

### - 客服专员

- 处理续费提醒
- 异常占位沟通和客服工单闭环

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 184-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 184-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
