## 项目简介

社区养老服务派单与健康随访管理系统面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

## 特色亮点

- 选题贴合社区养老照护、服务派单、健康评估、家属回访和预警提醒方向
- 形成“服务套餐-老人档案-护理档案-服务工单-服务团队-上门签到-服务记录-健康评估-用药提醒-家属回访-预警事件-通知公告”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 147 的 MyBatis-Plus 路线继续交替
- 服务工单、上门签到、服务记录、健康评估、预警事件和通知公告都有状态流转动作
- 首页看板展示套餐、团队、记录、预警指标，以及养老服务趋势和服务状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 服务派单、上门签到、健康随访和预警提醒状态机联动
- ECharts 展示养老服务趋势和服务状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 社区养老服务派单与健康随访管理系统类毕设项目
- 养老服务、居家照护、健康随访、预警管理和家属协同课程设计
- 多角色后台管理系统综合实战
- 需要展示养老服务闭环、健康跟踪和家属回访能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 148-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 148-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
