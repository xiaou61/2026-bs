> 基于SpringBoot+Vue3的在线职业培训证书考试与续证提醒系统

| 属性 | 内容 |

| 项目编号 | 168 |

## 项目简介

本系统面向职业培训机构与持证人员继续教育场景，提供培训课程、学员档案、讲师档案、课程报名、学习进度、考试安排、考试成绩、证书发放、证书台账、续证申请和提醒通知的一体化后台管理能力。

## 技术栈

### 后端

- Spring Boot 2.7.18
- MyBatis 2.3.1 注解 SQL
- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 培训管理员

- 培训课程
- 学员档案
- 讲师档案
- 课程报名和学习进度维护

### - 授课讲师

- 课程授课
- 学习进度
- 考试安排和考试成绩维护

### - 参训学员

- 课程报名
- 学习进度
- 考试成绩
- 证书台账和续证申请查看

### - 考务管理员

- 考试安排
- 考试成绩
- 发证复核和考试流程维护

### - 证书管理员

- 证书发放
- 证书台账
- 续证申请和提醒通知维护

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 168-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 168-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
