## 项目简介

本项目面向高校和培训机构的教学资源管理场景，提供教学资料上传、分类、审核、检索、下载、收藏、学习清单、公告和统计看板能力。

## 特色亮点

- 教学资料业务闭环：上传→审核→发布→下载→复习追踪
- MyBatis原生XML + PageHelper，便于答辩讲解SQL细节
- Redis结合JWT实现会话校验与失效控制
- 资料收藏自动加入学习清单，支持进度管理
- ECharts展示下载趋势与热门资料TOP5

## 演示流程

1. 教师登录后台，上传教学资料并提交审核
2. 管理员在审核中心进行通过或驳回处理
3. 审核通过后资料自动上架并可下载
4. 学生收藏资料后在学习清单更新学习进度
5. 管理员发布公告并在看板查看运营指标

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 084-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 084-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
