## 项目简介

本系统面向校外培训机构教务与财务协同场景，提供校区档案、课程产品、学员档案、教师档案、班级台账、排课计划、上课考勤、课消记录、退费申请、退费审批和财务流水的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 校区校长

- 校区档案
- 课程产品
- 班级台账
- 课消统计和财务总览维护

### - 教务主管

- 排课计划
- 上课考勤
- 课消记录和调课补课处理

### - 任课老师

- 上课考勤
- 课堂反馈
- 课消确认和学员沟通维护

### - 财务审核员

- 退费审批
- 财务流水
- 收入核对和退款结算维护

### - 学员家长

- 课消记录
- 退费申请
- 剩余课时和财务流水查看

## 特色亮点

- 选题贴合教培机构合规运营方向，覆盖课程售卖、排课签到、课消确认、家长退费和财务核销
- 形成“校区档案-课程产品-学员档案-教师档案-班级台账-排课计划-上课考勤-课消记录-退费申请-退费审批-财务流水”完整业务闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和结清归档
- 首页看板展示课消趋势、退费状态分布和教培经营核心指标
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 按校区校长、教务主管、任课老师、财务审核员、学员家长进行菜单、路由和操作权限收口
- ECharts 展示课消确认趋势、退费审批分布和财务流水概览
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和课消退费状态动作交互
- 校外培训机构课消统计与退费审批系统类毕设项目
- 教培管理、课消统计、退费审批、财务流水和家校协同课程设计
- 多角色后台管理系统综合实战
- 需要展示完整业务闭环和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 158-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 158-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
