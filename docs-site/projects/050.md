## 项目简介

这是一个基于微信小程序的课堂考勤签到系统，支持多种签到方式（普通签到、定位签到、二维码签到、数字签到），包含学生端和教师端功能，实现了完整的课堂考勤管理流程。

## 技术栈

### 后端

- Spring Boot 3.x
- MyBatis-Plus
- MySQL 8.0
- JWT认证

### 前端

- 微信小程序原生开发
- WXML + WXSS + JavaScript

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |

### 后端启动

```bash
cd 050-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 小程序启动

1. 使用微信开发者工具打开 `050-miniprogram` 目录
2. 确认 AppID 配置正确
3. 修改后端接口地址（通常在 config.js 或 utils/request.js 中）
4. 点击编译运行
