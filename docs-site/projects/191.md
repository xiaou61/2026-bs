## 项目简介

本系统面向社区助残服务站、器具管理人员、康复治疗师、随访志愿者与居民用户，提供服务站点、居民档案、助残器具、器具借用、借用审核、器具交付、康复计划、康复训练、随访记录、回收提醒、器具维护和操作日志的一体化管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 社区服务员

- 维护服务站点
- 居民档案
- 借用审核和服务规则

### - 器具管理员

- 维护助残器具
- 器具交付
- 回收提醒和器具维护

### - 康复治疗师

- 制定康复计划
- 记录康复训练并参与随访评估

### - 随访志愿者

- 协助器具交付
- 随访记录和回收提醒跟进

### - 居民用户

- 提交器具借用申请
- 查看康复计划和随访进度

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 191-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 191-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
