## 项目简介

本系统面向高校毕业去向填报、就业材料审核、岗位推荐、就业帮扶、协议归档和跟踪回访场景，提供学院专业、毕业生档案、用人单位、招聘岗位、去向填报、协议归档、材料审核、帮扶记录、岗位推荐、跟踪回访和就业统计的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 就业中心

- 维护去向填报
- 材料审核
- 就业统计和就业服务规则

### - 学院就业老师

- 维护学院专业
- 毕业生档案
- 协议归档和统计汇总

### - 辅导员

- 维护毕业生档案
- 帮扶记录
- 岗位推荐和跟踪回访

### - 毕业生

- 提交去向填报
- 查看岗位推荐
- 协议归档和帮扶进度

### - 用人单位

- 维护单位档案
- 发布招聘岗位并协同就业推荐

## 特色亮点

- 选题贴合高校就业服务方向，覆盖毕业生建档、去向填报、材料审核、协议归档、岗位推荐和帮扶跟踪
- 形成“学院专业-毕业生档案-用人单位-招聘岗位-去向填报-协议归档-材料审核-帮扶记录-岗位推荐-跟踪回访-就业统计”完整业务闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询
- 各业务模块支持列表查询、新增编辑、删除、状态处理和状态完成
- 首页看板展示毕业去向、岗位推荐、帮扶记录、就业统计、近7日趋势和去向状态分布
- Spring Boot + Vue3 前后端分离
- MyBatis 注解 SQL + PageHelper 分页查询
- JWT 登录态拦截与 Redis Token 校验
- 六类就业服务角色的菜单、路由和后端接口权限收口
- ECharts 展示业务趋势和状态分布
- Element Plus 通用数据页复用，统一搜索、表格、分页、弹窗和状态动作交互
- 高校毕业去向填报与就业帮扶跟踪系统类毕设项目
- 就业服务、毕业生管理、材料审核、岗位推荐、帮扶跟踪和统计分析课程设计
- 多角色后台管理系统综合实战
- 需要展示高校就业闭环、帮扶过程跟踪和可视化看板能力的答辩项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 173-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 173-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
