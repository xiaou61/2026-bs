## 项目简介

本项目是短文写作竞赛管理小程序的后端服务，配套 `056-frontend` 管理端和 `056-miniapp` 小程序端使用。系统包含竞赛分类、竞赛发布、公告、参赛作品投稿、作品审核、评委分配、评分标准、评委评分、获奖名单与统计概览等能力。

## 功能模块

| 模块 | 说明 |
|------|------|
| **竞赛管理** | 竞赛创建发布、状态管理（草稿/进行中/已结束/已下架）、字数限制设置、截止时间管理 |
| **分类管理** | 竞赛分类CRUD、分类层级 |
| **作品管理** | 作品提交审核、通过/退回/撤回状态流转、字数统计、作品详情 |

| **评委管理** | 评委分配、取消分配、评委列表 |
| **评分管理** | 多维度评分、评语填写、最终得分计算、评分记录 |
| **公告通知** | 系统公告、竞赛公告、置顶功能 |
| **获奖管理** | 获奖名单生成、奖项等级、获奖公示 |
| **用户管理** | 三角色系统（管理员/评委/参赛者）、个人信息 |
| **数据统计** | 首页仪表盘、竞赛统计、作品统计、评分统计 |

## 演示流程

1. 管理员登录后台 → 创建竞赛分类 → 设置评分标准
2. 创建竞赛 → 设置字数限制和截止时间 → 发布竞赛
3. 分配评委 → 选择评委负责该竞赛评审
4. 参赛者小程序登录 → 浏览竞赛 → 提交作品
5. 管理员审核作品 → 通过/退回并填写原因
6. 评委登录 → 查看待评作品 → 按维度打分填写评语
7. 竞赛结束 → 生成获奖名单 → 发布公告
8. 参赛者查看获奖结果 → 获奖展示

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |

### 后端启动

```bash
cd 056-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 小程序启动

1. 使用微信开发者工具打开 `056-miniprogram` 目录
2. 确认 AppID 配置正确
3. 修改后端接口地址（通常在 config.js 或 utils/request.js 中）
4. 点击编译运行
