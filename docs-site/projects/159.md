## 项目简介

本系统面向医疗废弃物分类收集、转运交接与处置监管场景，提供医废来源、废物类别、包装赋码、收集预约、称重记录、暂存交接、转运联单、运输轨迹、处置确认、异常追溯和监管抽查的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 医院交接员

- 医废来源
- 包装赋码
- 暂存交接和院内出库维护

### - 收运调度员

- 收集预约
- 称重记录
- 暂存交接和转运联单调度

### - 转运司机

- 转运联单
- 运输轨迹
- 车辆交接和异常上报维护

### - 处置厂审核员

- 处置确认
- 入厂复核
- 处置批次和闭环回执维护

### - 监管人员

- 监管抽查
- 异常追溯
- 联单链路和闭环报表查看

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 159-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 159-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
