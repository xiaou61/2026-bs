> 基于SpringBoot+Vue3的校园图书漂流借阅与读书打卡平台

## 项目简介

本系统面向校园图书漂流、学生借阅、读书打卡、笔记沉淀、积分勋章和阅读活动场景，提供漂流书架、漂流图书、读者档案、图书捐赠、借阅记录、归还流转、读书打卡、读书笔记、积分勋章、阅读活动和消息通知的一体化后台管理能力。

## 技术栈

### 后端

- Spring Boot 2.7.18

- PageHelper 1.4.7
- MySQL 8.0
- Redis 登录态缓存
- JWT 认证

### 前端

- Vue 3.4.0
- Element Plus 2.4.4
- Vite 5
- Pinia 2.1.7
- Axios 1.6.2
- ECharts 5.4.3

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 图书馆管理员

- 维护漂流书架
- 漂流图书
- 借阅记录和统计规则

### - 漂流负责人

- 维护图书捐赠
- 借阅记录
- 归还流转和消息通知

### - 学生读者

- 参与图书捐赠
- 图书借阅
- 归还流转
- 读书打卡和读书笔记

### - 读书社团

- 组织阅读活动
- 协同打卡运营和积分勋章评定

### - 阅读指导老师

- 审核读书笔记
- 指导阅读活动
- 评定积分勋章和读者成长

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 175-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 175-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
