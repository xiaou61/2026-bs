## 项目简介

本项目是大学生一体化服务平台后端服务，配套 `067-frontend` 使用。系统覆盖统一登录、用户管理、课程选课、活动报名、实习岗位投递、失物招领、公告中心和运营看板等校园服务流程。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户与认证** | 登录注册、JWT鉴权、个人资料维护、密码修改 |
| **用户管理** | 管理员维护账号、角色、状态与基础档案 |
| **课程中心** | 教师/管理员发布课程、维护学分与上限人数 |
| **选课管理** | 学生在线选课，教师/管理员维护选课状态与成绩 |
| **活动中心** | 校园活动发布、时间地点配置、报名名额控制 |
| **活动报名** | 学生报名/取消/签到，教师端统一管理报名记录 |
| **实习岗位** | 教师/管理员发布岗位，支持截止日期与状态控制 |
| **岗位投递** | 学生在线投递简历，教师/管理员审核通过或拒绝 |
| **失物招领** | 寻物与招领双类型发布、状态流转与归属权限控制 |
| **公告中心** | 公告发布停用、全员可查的通知信息 |
| **运营看板** | 用户、课程、活动、投递等核心指标与7日趋势图 |

## 演示流程

1. 管理员维护用户账号并分配教师/学生角色
2. 教师发布课程、活动与实习岗位，配置时间、人数与状态
3. 学生登录后完成选课、活动报名和岗位投递
4. 教师审核学生投递结果，维护选课与报名记录状态
5. 学生与教师发布失物招领信息，管理员进行治理
6. 管理员/教师在看板查看整体运营指标与趋势变化

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 067-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 067-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
