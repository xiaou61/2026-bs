## 特色亮点

- 知识付费机制：优质资源可设置积分下载，激励内容创作者
- 完善积分体系：上传资源+10分、下载消耗积分、回答被采纳+20分、评价+2分
- 协作学习小组：小组讨论区、小组资源共享、促进学生交流合作
- 智能题库系统：随机组卷算法、错题本管理、帮助学生高效复习
- 悬赏答疑机制：付费提问、积分悬赏、快速获得优质答案
- Markdown笔记：mavon-editor编辑器、实时预览、代码高亮、便于记录分享
- 资源评价体系：5星评价、评价标签、影响资源推荐
- 多格式支持：文档、视频、代码、压缩包多种资源类型

## 演示流程

1. 学生注册登录 → 获得初始100积分
2. 浏览资源广场 → 按分类筛选（课件/教材/视频/代码）
3. 查看资源详情 → 评分评价、下载次数
4. 下载资源 → 消耗积分 → 上传者获得收益
5. 上传自己的资源 → 设置下载所需积分 → 获得上传奖励+10分
6. 创建学习小组 → 邀请同学加入 → 小组内分享资源
7. 浏览题库 → 选择科目 → 随机刷题
8. 生成模拟试卷 → 完成答题 → 错题自动加入错题本
9. 发起悬赏提问 → 设置悬赏积分 → 等待回答
10. 其他用户回答问题 → 提问者采纳最佳答案 → 回答者获得悬赏积分
11. 创建Markdown笔记 → 记录学习心得 → 分享给其他用户

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 020-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 020-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
