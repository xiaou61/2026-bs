## 角色权限

### - 学生

- 浏览岗位
- 投递简历
- 预约面试
- 分享经验

### - 企业HR

- 发布岗位
- 查看简历
- 安排面试

### - 管理员

- 平台管理

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户系统** | 学生、企业HR、管理员三种角色 |
| **企业管理** | 企业入驻认证、企业信息管理、企业主页展示 |
| **岗位管理** | 岗位发布编辑、搜索筛选、智能推荐 |
| **简历管理** | 在线简历创建、简历信息完善、简历预览 |
| **投递系统** | 在线投递、进度跟踪（5个状态）、投递历史 |
| **面试管理** | 面试邀约、时间预约、线上/线下面试、日程管理 |
| **经验分享** | 面经/笔试题发布、浏览搜索、点赞收藏 |
| **内推广场** | 校友发布内推、内推信息展示、内推码管理 |

## 特色亮点

- 智能推荐算法：基于专业、技能、学历三维度匹配岗位
- 完整投递流程：pending→screening→interview→offer→rejected五状态流转
- 双端架构：学生端（求职）+ 企业端（招聘），界面独立设计
- 面试管理：支持线上/线下面试类型，面试反馈记录
- 经验社区：面经和笔试题分享，点赞浏览统计
- 内推机制：校友发布内推机会，内推码管理系统
- 现代化UI：Element Plus组件库，渐变色主题，流畅交互
- 前后端分离：RESTful API设计，JWT无状态认证

## 演示流程

1. 学生注册登录 → 完善个人简历信息
2. 浏览岗位列表 → 多条件搜索筛选（类型/地点/关键词）
3. 查看智能推荐岗位 → 基于专业技能匹配
4. 点击岗位详情 → 投递简历 → 选择个人简历
5. 查看投递记录 → 跟踪投递状态（待筛选→面试邀约）
6. 收到面试邀约 → 查看面试时间地点
7. 浏览经验广场 → 阅读面经笔试题 → 点赞收藏
8. 查看内推广场 → 获取校友内推信息
9. 企业HR登录 → 发布岗位（填写岗位详情、薪资、要求）
10. 查看收到的简历 → 更新投递状态 → 安排面试

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 018-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/campus_recruitment
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 018-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
