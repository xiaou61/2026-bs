## 项目简介

### 项目简介
高清壁纸社区网站面向桌面端与移动端壁纸浏览下载场景，支持游客浏览、用户注册收藏与投稿、管理员审核运营，实现从内容上传、审核发布到浏览下载和统计分析的完整业务闭环。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户权限** | 管理员/普通用户双角色登录注册与会话校验 |

| **运营模块** | 轮播图管理、公告发布与首页数据看板 |

## 特色亮点

- 前台社区浏览与后台审核管理一体化，适合演示内容平台业务闭环
- 支持分类、标签、分辨率、色系等多维筛选，视觉站点展示效果好
- 用户投稿后进入审核流，体现内容平台常见运营机制
- 收藏与下载记录联动统计，可直观看到内容热度变化
- ECharts 看板展示壁纸数量、下载量和收藏量

## 演示流程

1. 管理员维护壁纸分类、标签和首页轮播图
2. 普通用户注册登录后浏览公开壁纸并查看详情
3. 用户上传壁纸投稿，进入后台审核列表
4. 管理员审核通过后壁纸上架展示
5. 用户收藏或下载壁纸，后台看板更新统计数据

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 086-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 086-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
