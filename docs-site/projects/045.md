## 项目简介

基于 Spring Boot 3.2、MyBatis-Plus 3.5.5、Vue 3、TypeScript 和 Element Plus 的养老院管理系统，覆盖管理员、护工、家属三类角色。

## 技术栈

### 后端

- Spring Boot 3.2
- MyBatis-Plus 3.5.5
- MySQL 8.0
- JWT认证

### 前端

- Vue 3 + TypeScript
- Element Plus
- Vite
- Pinia

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 045-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 045-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
