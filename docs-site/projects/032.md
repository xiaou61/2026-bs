## 特色亮点

- 专注老年人群体：针对社区老年人健康管理的专业化系统
- 完整健康档案：涵盖基本信息、健康数据、病史、用药等多维度信息
- 健康数据可视化：ECharts图表展示健康数据趋势，直观了解健康状况
- 智能预警机制：血压、血糖等异常数据自动预警，及时发现健康风险
- 预约随访服务：支持上门随访预约，完整的服务流程管理
- 多维度提醒：服药、随访、体检等多种提醒，贴心关怀老年人
- 前后端分离：Vue3 + Spring Boot现代化架构，响应式设计

## 演示流程

1. 管理员/社工登录 → 创建老人档案 → 录入基本信息和健康档案
2. 记录健康测量数据 → 输入血压、血糖、体温等数据
3. 系统自动生成健康趋势图 → 异常数据触发预警
4. 制定随访计划 → 预约上门服务时间
5. 社工完成随访 → 记录随访内容和老人状态
6. 设置用药提醒 → 系统自动推送服药通知
7. 进行健康评估 → 生成综合健康报告和建议
8. 查看统计报表 → ECharts可视化数据分析

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 032-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/senior_health
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 032-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
