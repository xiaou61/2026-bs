## 特色亮点

- 信用评价体系：5星评价+信用分机制，建立可信交易环境
- 多维度搜索：支持关键词、分类、成色、价格等多条件筛选
- 在线沟通系统：内置聊天功能，支持价格议价和交易细节沟通
- 安全交易流程：完整的订单状态管理，保障买卖双方权益
- 商品状态管理：在售/已售/下架状态切换，灵活控制商品展示
- 浏览统计功能：记录商品浏览量和收藏数，为热门推荐提供数据
- 响应式设计：适配PC和移动端，随时随地发布和浏览商品
- JWT无状态认证：安全的用户身份验证和授权机制

## 演示流程

1. 学生注册登录 → 完善个人信息 → 获得初始信用分
2. 发布二手商品 → 选择分类 → 上传商品图片 → 设置价格和成色
3. 商品进入审核 → 审核通过后正式上架展示
4. 其他用户浏览商品 → 搜索筛选 → 查看商品详情
5. 感兴趣可收藏商品 → 与卖家在线沟通议价
6. 确认购买后下单 → 生成订单记录 → 卖家确认交易
7. 双方线下完成交易 → 确认收货 → 进行相互评价
8. 评价完成 → 信用分更新 → 交易记录保存
9. 管理员监控商品信息 → 处理违规商品 → 维护平台秩序

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 021-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/campus_market
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 021-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
