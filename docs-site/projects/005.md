## 角色权限

系统支持 **2** 种角色，各角色权限如下：

### 管理员

- 可以创建
- 编辑
- 发布
- 删除问卷
- 查看统计数据

### 普通用户

- 只能查看已发布的问卷列表
- 可以查看统计（但不能创建
- 编辑
- 发布
- 删除）

## 功能模块

| 模块 | 说明 |
|------|------|
| **问卷设计** | 可视化问卷创建、多种题型支持 |
| **问卷发布** | 问卷链接生成、二维码分享 |
| **答卷收集** | 在线填写、自动保存 |
| **数据分析** | 自动统计、图表可视化（饼图、柱状图） |
| **权限管理** | 问卷权限控制、结果查看 |
| **导出功能** | 数据导出Excel、PDF报告 |

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 005-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/survey_db
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 005-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
