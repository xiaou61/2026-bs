## 项目简介

### 项目简介
课程管理系统面向高校教务与教学场景，支持管理员、教师、学生三角色协同完成组织管理、课程开设、排课选课、课程资源、考勤登记、成绩录入、评教反馈和统计分析，形成完整的教学管理闭环。

## 功能模块

| 模块 | 说明 |
|------|------|

## 特色亮点

- 覆盖教务管理从基础组织到教学结果分析的完整业务链条
- 三角色权限分工明确，适合毕设答辩展示角色差异
- 排课、选课、资源、考勤、成绩和评教模块串联成教学闭环
- MyBatis + PageHelper 适合展示原生 SQL 与分页查询能力
- ECharts 看板可展示课程数、排课数、选课数等核心指标

## 演示流程

1. 管理员维护院系、专业、班级、学期和课程信息
2. 管理员创建排课计划并开放学生选课
3. 学生登录后查看课程列表并完成在线选课
4. 教师上传课程资源、登记考勤并录入成绩
5. 学生提交评教后，后台查看统计看板和反馈结果

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 087-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 087-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
