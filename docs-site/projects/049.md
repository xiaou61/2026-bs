## 项目简介

本系统是一个面向考研学生的学习平台，基于微信小程序前端 + Spring Boot后端开发。系统提供课程学习、题库刷题、错题本、学习计划、每日打卡、笔记管理、考研社区等功能，帮助考研学生高效备考。

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |

### 后端启动

```bash
cd 049-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 小程序启动

1. 使用微信开发者工具打开 `049-miniprogram` 目录
2. 确认 AppID 配置正确
3. 修改后端接口地址（通常在 config.js 或 utils/request.js 中）
4. 点击编译运行
