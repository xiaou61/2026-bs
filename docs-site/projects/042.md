## 项目简介

本项目是基于 Spring Boot 3.2、MyBatis-Plus、JWT 的房屋租赁管理系统后端，提供用户登录注册、房源浏览与发布、收藏、预约看房、合同签署、租金账单、报修和消息通知等核心接口。

## 演示流程

1. 房东注册登录 → 发布房源 → 上传房源图片和详细信息
2. 管理员审核房源 → 审核通过后房源上架
3. 租客浏览房源 → 多条件筛选 → 查看房源详情
4. 租客收藏心仪房源 → 预约看房 → 选择看房时间
5. 房东确认预约 → 双方线下看房
6. 租客看房满意 → 在线签署租赁合同
7. 房东确认合同 → 合同生效 → 租客支付押金和首期租金
8. 系统自动生成后续租金账单 → 按月提醒租客缴费
9. 租客缴纳账单 → 房东查看收入统计
10. 租客提交报修申请 → 房东安排维修 → 更新维修进度
11. 合同到期 → 双方互相评价 → 完成租赁流程
12. 管理员后台 → 查看平台数据统计 → 管理用户和房源

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 042-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 042-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
