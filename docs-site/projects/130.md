## 特色亮点

- 选题贴合智慧农业、温室物联网、环境监测、远程控制、病虫害预警和采收维护方向
- 形成“温室档案-作物批次-环境传感器-环境读数-灌溉任务-施肥计划-虫害预警-病害诊断-控制设备-远程指令-采收记录-维护工单”完整闭环
- 使用 MyBatis 注解 SQL + PageHelper 分页查询，与 129 的 MyBatis-Plus 路线继续交替
- 环境读数、灌溉任务、虫害预警、病害诊断、远程指令、采收记录和维护工单都有状态流转动作
- 首页看板展示温室、传感器、虫害预警、远程指令指标，以及环境监测趋势和作物类型分布

## 演示流程

1. 管理员维护账号权限、温室档案、作物批次和环境传感器
2. 设备技术员采集环境读数并管理控制设备
3. 种植员制定灌溉任务和施肥计划
4. 系统根据环境和作物状态生成虫害预警
5. 专家处理病害诊断并给出建议方案
6. 园区经理下发远程指令、跟踪采收记录和维护工单
7. 首页看板查看环境趋势、作物分布和病虫害处置指标

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 130-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 130-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
