## 项目简介

本项目是高校招生管理系统后端服务，配套 `057-frontend` 管理端使用。系统覆盖院系、专业、招生计划、考生、报名申请、成绩、录取、分数线、公告和统计分析等招生管理流程。

## 功能模块

| 模块 | 说明 |
|------|------|
| **系统管理** | 管理员登录/退出、密码修改、个人信息 |
| **院系管理** | 院系信息维护、排序管理 |
| **专业管理** | 专业信息、学制学费、报考要求 |
| **招生计划** | 年度招生计划、计划发布/关闭、录取人数统计 |

| **报名管理** | 志愿填报、服从调剂、审核流程 |
| **成绩管理** | 成绩录入、语数英综合成绩、总分自动计算 |
| **录取管理** | 录取操作、录取确认、录取编号生成 |
| **分数线管理** | 各专业分省份分数线设置 |
| **通知公告** | 公告发布、置顶、草稿/发布状态 |
| **统计分析** | 首页仪表盘、专业招生统计、生源地分布(ECharts) |

## 演示流程

1. 管理员登录 → 创建院系与专业 → 设置招生计划
2. 发布招生计划 → 设置各省份分数线
3. 录入考生信息 → 审核考生资质
4. 考生报名申请 → 审核通过/拒绝
5. 录入考生成绩 → 语数英综合自动计算总分
6. 执行录取操作 → 生成录取编号 → 考生状态变更
7. 确认录取 → 查看统计报表 → 发布录取公告

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 057-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 057-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
