## 项目简介

本系统面向乡镇农机服务站、农户、农机机手、调度员与维修人员的农机作业服务场景，提供农机站点、农机档案、农户档案、机手档案、作业预约、机手调度、作业派单、作业回执、维修保养、维修跟踪和季节统计的一体化管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 农机站管理员

- 维护农机站点
- 农机档案
- 机手档案和服务规则

### - 调度员

- 处理作业预约
- 机手调度
- 作业派单和季节统计

### - 农机机手

- 接收作业派单
- 登记作业回执并上报维修需求

### - 维修人员

- 维护维修保养
- 维修跟踪和农机状态

### - 农户

- 提交作业预约
- 查看派单并确认作业完成

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 189-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 189-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
