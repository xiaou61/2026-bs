## 项目简介

本系统面向高校课堂考勤、异常申诉、申诉审核、课堂巡查、问题整改和反馈复核场景，提供教学班级、学生档案、教师档案、课程排课、考勤记录、异常申诉、申诉审核、课堂巡查、巡查问题、整改任务和整改反馈的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 教务管理员

- 维护教学班级
- 教师档案
- 课程排课和考勤规则

### - 任课教师

- 登记考勤记录
- 处理异常申诉并配合课堂巡查

### - 课堂巡查员

- 发起课堂巡查
- 记录巡查问题并跟踪整改任务

### - 辅导员

- 查看学生考勤
- 协助申诉审核和整改反馈

### - 学生

- 查看个人考勤记录
- 提交异常申诉并补充整改反馈

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 179-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 179-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
