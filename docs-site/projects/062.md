## 项目简介

本项目是反欺诈平台后端服务，配套 `062-frontend` 使用。系统覆盖登录注册、黑名单、风控规则、风险事件上报、自动风险评分、预警指派与处置、案件、申诉、公告、操作日志和风险看板等流程。

## 功能模块

| 模块 | 说明 |
|------|------|
| **用户与认证** | 登录注册、JWT鉴权、个人资料维护、密码修改 |
| **用户管理** | 管理员维护账号、角色、状态，支持禁用/启用 |

| **风险看板** | 总览统计、7日趋势、风险等级分布、规则命中排行、高风险用户排行 |
| **操作审计** | 核心业务操作日志记录、管理员按条件追溯查询与CSV导出 |

## 演示流程

1. 管理员维护黑名单和风控规则并发布公告
2. 用户提交风险事件，系统自动计算风险分并产出风险等级
3. 高/中风险事件自动生成预警，管理员指派分析员处理
4. 分析员处置预警并输出处理结论
5. 必要时由预警创建案件，跟踪调查后关闭案件
6. 用户对处理结果发起申诉，平台复核并反馈最终结论

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 062-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 062-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
