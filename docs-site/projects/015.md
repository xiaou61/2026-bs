## 角色权限

系统支持 **2** 种角色，各角色权限如下：

### 用户

- 用户注册/登录（实名认证）
- 匿名发帖（表白/吐槽/求助/树洞/校园/闲聊）
- 话题广场（分类浏览、热门/最新排序）
- 帖子详情（浏览、点赞、评论）
- 评论互动（楼中楼）
- 敏感词自动过滤
- 私信功能（匿名对话）
- 举报功能（举报帖子/评论）
- ...还有 3 项权限

### 管理

- 数据概览统计
- 用户管理（禁言/封号）
- 实名认证审核
- 帖子内容审核
- 举报处理
- 敏感词管理
- 帖子置顶/删除

## 功能模块

| 模块 | 说明 |

| **用户系统** | 注册登录、实名认证、个人中心、积分等级 |

## 特色亮点

- 前后端一体化：Bootstrap+jQuery集成，单jar包部署
- 完整匿名保护：每个帖子/评论使用不同匿名身份，真实身份隔离
- 智能敏感词过滤：DFA算法高效过滤，三级敏感词库（轻度/中度/重度）
- 私信匿名聊天：双方匿名对话，5秒自动刷新消息
- 举报审核机制：用户举报、管理员审核、分级处罚（警告/禁言/封号）
- 积分激励系统：多维度积分获取、等级升级、违规扣分
- 热度排序算法：点赞×3 + 评论×2 + 浏览×0.1，时间衰减
- 美观UI设计：粉色渐变主题、卡片式布局、悬浮动画

## 演示流程

1. 用户注册 → 填写学号姓名 → 等待实名认证
2. 管理员审核认证 → 用户获得+10积分
3. 浏览话题广场 → 分类切换（表白/吐槽等6种）
4. 点击发帖 → 选择分类 → 输入内容 → 系统生成匿名身份
5. 敏感词检测 → 自动过滤或进入审核 → 发布成功+5积分
6. 查看帖子详情 → 评论互动 → 点赞收藏
7. 发起私信 → 双方匿名聊天 → 消息自动刷新
8. 举报违规内容 → 管理员审核 → 处理结果通知
9. 个人中心 → 查看我的发帖/评论/举报/积分
10. 管理后台 → 数据概览、用户管理、内容审核、敏感词管理

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 015-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/campus_confession
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 015-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
