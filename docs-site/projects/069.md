## 项目简介

本项目是科任教师考评系统后端服务，配套 `069-frontend` 使用。系统覆盖统一登录、用户权限、科目/班级/教师档案、评价指标、考评任务、学生评教、教师申诉、公告和统计看板等流程。

## 演示流程

1. 管理员维护科目、班级、教师档案和评价指标
2. 管理员创建并开启考评任务，指定班级与科任教师
3. 学生在任务期内提交五维评分与评语
4. 教师查看被评记录，对争议评分发起申诉
5. 管理员在申诉中心处理并回复申诉结果
6. 各角色在看板查看与自身相关的统计和趋势

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 069-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 069-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
