## 项目简介

本项目是精简博客系统后端服务，配套 `066-frontend` 使用。系统覆盖登录注册、用户管理、文章分类、标签、文章发布与置顶、评论审核、公告和后台看板等博客核心流程。

## 演示流程

1. 管理员登录后维护分类、标签与系统公告
2. 用户撰写文章并保存草稿或直接发布
3. 博客首页按关键词和分类浏览文章列表
4. 进入文章详情阅读内容并提交评论
5. 管理员审核评论、回复反馈并在看板查看运营数据

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 066-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 066-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
