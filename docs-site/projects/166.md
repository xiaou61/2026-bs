## 项目简介

本系统面向农贸市场日常监管与食品安全追溯场景，提供市场区域、摊位档案、商户档案、商品溯源、摊位巡检、问题整改、抽检记录、检测结果、进货台账、处置记录和风险预警的一体化后台管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 市场管理员

- 市场区域
- 摊位档案
- 商户档案
- 巡检计划和台账汇总维护

### - 巡检员

- 摊位巡检
- 问题发现
- 整改跟踪
- 处置记录和风险预警维护

### - 摊位商户

- 商品溯源
- 进货台账
- 整改反馈和检测结果查看维护

### - 抽检员

- 抽检记录
- 检测结果
- 样品流转和风险线索维护

### - 监管审核员

- 问题整改复核
- 检测结果审核
- 风险预警和监管闭环维护

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 166-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 166-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
