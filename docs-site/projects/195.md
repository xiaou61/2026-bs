## 项目简介

本系统面向便民服务中心、办事窗口、材料审核人员、督办人员和办事群众，提供事项档案、窗口档案、人员排班、事项预约、窗口叫号、材料审核、办理进度、通知提醒、满意评价、投诉处理、绩效统计和操作日志的一体化服务管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 中心管理员

- 维护事项档案
- 窗口档案
- 排班规则和窗口服务规则

### - 窗口经办

- 处理窗口叫号
- 材料流转
- 办理进度和通知提醒

### - 材料审核

- 审核预约材料
- 反馈审核意见并推动办理流转

### - 督办人员

- 查看满意评价
- 投诉处理和绩效统计

### - 办事群众

- 提交事项预约
- 查看叫号进度
- 办理状态并完成评价

- 便民服务中心事项预约与窗口评价平台类毕设项目

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 195-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 195-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
