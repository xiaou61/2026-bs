## 项目简介

本系统面向运动康复中心、体测评估师、康复教练、康复治疗师和训练会员，提供康复中心、会员档案、教练档案、体测项目、体测评估、风险提醒、训练计划、训练安排、训练打卡、康复反馈、复评记录和操作日志的一体化管理能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 中心管理员

- 维护康复中心
- 会员档案
- 教练档案和体测项目

### - 体测评估师

- 处理体测评估
- 风险提醒和复评记录

### - 康复教练

- 制定训练计划
- 维护训练安排并跟进训练打卡

### - 康复治疗师

- 查看评估结果
- 维护康复反馈并参与复评

### - 训练会员

- 查看个人体测
- 训练计划
- 训练打卡和康复反馈

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 199-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 199-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
