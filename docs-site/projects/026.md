## 特色亮点

- 完整交易闭环：从需求发布→画师接单→作品交付→评价，全流程管理
- 订单状态机：待支付定金→制作中→草图提交→修改中→待付尾款→已完成，8个状态流转
- 资金安全保障：定金30% + 尾款70%分阶段支付，平台担保交易
- 修改机制：支持有限次数修改（默认2次），平衡双方利益
- 双向评价体系：客户评价画师（质量/态度/速度），画师评价客户（沟通/配合度）
- 画师认证系统：申请→审核→认证通过，确保画师质量
- 作品集展示：精美的作品展示，支持分类、标签、精选
- 平台抽成机制：8%平台手续费，自动计算画师收益
- 灵活接稿模式：支持指定画师委托和公开招标两种方式
- 余额系统：充值、消费、提现全流程管理

## 演示流程

1. 用户注册登录 → 充值余额
2. 普通用户申请成为画师 → 提交资质信息
3. 管理员审核画师申请 → 通过后用户角色升级为ARTIST
4. 画师上传作品到作品集 → 展示个人作品
5. 客户浏览画师广场 → 查看画师详情和作品
6. 客户发布需求 → 填写需求详情、预算、截止日期
7. 画师浏览需求广场 → 选择合适需求
8. 双方协商后创建订单 → 客户支付定金（30%）
9. 画师开始创作 → 提交草图
10. 客户查看草图 → 确认或要求修改
11. 画师提交最终成品 → 客户支付尾款（70%）
12. 平台扣除8%手续费 → 转账给画师
13. 双方互相评价 → 订单完成

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 026-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/db_026
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 026-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
