## 项目简介

本系统面向工业园区管委会、危废暂存仓库、转运单位、环保巡查人员和产废企业，提供园区企业、危废类别、暂存设施、入库登记、暂存巡检、转运联单、车辆调度、称重记录、处置交接、风险预警、台账审计和操作日志的一体化监管能力。

## 角色权限

系统支持 **6** 种角色，各角色权限如下：

### - 系统管理员

- 账号权限
- 基础配置和全部数据维护

### - 园区监管

- 维护园区企业
- 危废类别
- 暂存设施和监管台账

### - 暂存管理员

- 登记入库
- 执行暂存巡检
- 配合称重和风险预警处置

### - 转运调度

- 维护转运联单
- 车辆调度
- 称重记录和处置交接

### - 环保巡查

- 核查暂存巡检
- 风险预警
- 处置交接和台账审计

### - 产废企业

- 维护企业相关危废信息
- 提交入库登记并查看转运进度

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 194-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 194-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
