## 项目简介

本项目是社区鲜牛奶订购配送管理系统后端服务，配套 `058-frontend` 管理端和用户端页面使用。系统覆盖牛奶分类、产品展示、订阅计划、收货地址、订单、配送任务、投诉反馈、通知和后台统计等流程。

## 演示流程

1. 管理员登录 → 管理牛奶分类和产品 → 设置配送区域和路线
2. 用户注册登录 → 浏览牛奶产品 → 添加收货地址
3. 用户创建订阅 → 选择产品/地址/类型/时段 → 系统生成订单
4. 配送员登录 → 查看今日配送任务 → 确认完成或标记异常
5. 用户查看订单状态 → 提交投诉反馈 → 管理员回复处理
6. 管理员查看仪表盘 → 订单统计 → 配送统计

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 8+ / 11+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 058-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/Shanghai
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 058-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
