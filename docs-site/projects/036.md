## 演示流程

1. 用户注册登录 → 进入首页浏览最新公益项目
2. 浏览项目广场 → 按分类/状态筛选项目
3. 查看项目详情 → 了解项目介绍、目标金额、已筹集进度
4. 点击立即捐赠 → 选择捐赠金额、填写留言、选择支付方式
5. 勾选匿名捐赠 → 确认捐赠 → 支付成功
6. 项目进度实时更新 → 捐赠人数+1、已筹金额增加
7. 查看我的捐赠记录 → 个人中心查看历史捐赠
8. 组织用户登录 → 创建公益项目 → 填写项目信息
9. 发布项目进度 → 上传图文 → 捐赠者可查看
10. 管理员后台 → 审核项目、用户管理、数据统计

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| MySQL | 8.0+ |
| Node.js | 18+ |

### 后端启动

```bash
cd 036-backend
# 1. 创建数据库并执行 init.sql
# 2. 修改 application.yml 数据库配置
# spring.datasource.url=jdbc:mysql://localhost:3306/db_036
# spring.datasource.username=root
# spring.datasource.password=你的密码
mvn spring-boot:run
```

### 前端启动

```bash
cd 036-frontend
npm install
# 修改后端接口地址（通常在 .env.development 或 src/utils/request.js 中）
npm run dev
```
