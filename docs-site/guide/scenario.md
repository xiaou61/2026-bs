# 按功能场景筛选

> 不同项目包含不同的功能场景。如果你需要实现特定功能（如工作流审批、数据可视化、预约排期等），可以使用此页面快速找到包含该功能的项目。

## 常见功能场景说明

| 场景 | 说明 | 常见项目类型 |
|------|------|-------------|
| **用户权限管理** | 多角色登录、权限控制、菜单管理 | 几乎所有项目 |
| **CRUD 台账管理** | 标准增删改查、列表分页、状态流转 | 几乎所有项目 |
| **工作流审批** | 申请-审核-驳回流程、状态机驱动 | 请假、报修、预约类项目 |
| **数据可视化** | ECharts 图表、统计看板、数据分析 | 大部分项目首页 |
| **文件上传下载** | 文件管理、附件上传、图片预览 | 资料管理、作业类项目 |
| **消息通知** | 系统通知、消息推送、站内信 | 社区、校园类项目 |
| **预约排期** | 预约管理、排期调度、时间选择 | 医疗、课程、场馆类项目 |
| **订单交易** | 下单支付、订单管理、结算流程 | 电商、票务类项目 |
| **JWT 认证** | Token 登录、Redis 缓存、拦截鉴权 | 几乎所有前后端分离项目 |
