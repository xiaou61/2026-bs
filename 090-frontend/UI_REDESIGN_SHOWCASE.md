# 090-frontend UI 美化展示

## 项目信息
- **项目名称**: 戏曲文化苑系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 戏曲红
- **美化完成时间**: 2026-05-30

## 设计理念

采用"戏曲红"设计理念，以红色和金色为主色调，体现戏曲文化的传统、典雅和艺术感。通过脸谱、音符等装饰元素，营造出戏曲传承、文化自信的传统艺术氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放旋转 |
| 面板顶部条 | operaGradient | 流光渐变 |
| 账号卡片 | hover | 悬浮上移 |
