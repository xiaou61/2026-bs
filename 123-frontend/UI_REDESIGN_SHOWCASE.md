# 123-frontend UI 美化展示

## 项目信息
- **项目名称**: 城市内涝监测与应急调度平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 防汛蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"防汛蓝"设计理念，以深蓝色为主色调，体现防汛预警的紧迫性、专业性和可靠性。通过海浪、雨滴等装饰元素，营造出防汛预警、安全度汛的应急管理氛围。
