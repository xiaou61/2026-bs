# 093-frontend UI 美化展示

## 项目信息
- **项目名称**: 自助售货管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 售货紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"售货紫"设计理念，以紫色和蓝色为主色调，体现自助售货的科技感、便捷性和现代感。通过商店、购物车等装饰元素，营造出24小时无人售货的智能零售氛围。
