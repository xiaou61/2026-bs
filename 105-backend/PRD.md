# API 接口文档生成与测试用例管理平台

## 项目概述
本系统面向软件研发和测试协作场景，提供接口项目、接口分组、接口定义、请求参数、响应字段、Mock规则、测试用例、执行记录和文档快照的一体化管理。

## 功能需求
1. 账号权限：管理员、产品经理、测试工程师、开发人员多角色登录。
2. 接口项目：维护项目编码、基础地址、鉴权方式和版本号。
3. 接口分组：按项目管理接口目录结构。
4. 接口定义：记录接口方法、路径、内容类型、维护人和发布状态。
5. 请求参数：维护参数位置、数据类型、是否必填和示例。
6. 响应字段：维护字段层级、类型、示例和说明。
7. Mock规则：配置匹配表达式、响应码、响应体和延迟。
8. 测试用例：维护正向、异常、边界等用例。
9. 用例步骤：配置请求数据、断言表达式和变量提取。
10. 环境配置：维护开发、测试、预发环境变量。
11. 执行记录：记录执行批次、环境、执行人和状态。
12. 结果明细：记录用例结果、状态码、耗时和断言信息。
13. 文档快照：生成接口文档版本并支持发布归档。
14. 操作日志：记录关键操作审计。
