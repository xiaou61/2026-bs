# 195-frontend UI 美化展示

## 项目信息
- **项目名称**: 便民服务中心事项预约与窗口评价平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 政务蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"政务蓝"设计理念，以蓝色为主色调，体现政务服务的专业性、权威性和可信赖感。通过建筑、表格等装饰元素，营造出便民服务、人民至上的政务服务氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | govGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
