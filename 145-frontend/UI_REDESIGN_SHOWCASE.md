# 145-frontend UI 美化展示

## 项目信息
- **项目名称**: 城市噪声投诉监测与执法协同平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 执法橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"执法橙"设计理念，以橙色和金色为主色调，体现城市执法的权威性、专业性和活力感。通过喇叭、建筑等装饰元素，营造出降低噪音、安静生活的城市治理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 执法橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |

## 设计特点

### 登录页面
- 橙色渐变背景
- 噪音装饰元素浮动
- 喇叭🔊Logo震动动画
- 流光渐变顶部条
- 测试账号标签
- 降低噪音标语

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | vibrate | 左右震动 |
| 顶部条 | noiseGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

145-frontend采用"执法橙"设计理念，通过橙色和金色的搭配，营造出权威、专业的城市噪声治理氛围。喇叭震动动画突出了噪音投诉监测的特色。
