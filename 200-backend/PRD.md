# 非遗工坊课程预约与作品展销平台 PRD

## 项目概述
本系统面向非遗工坊、非遗讲师、展陈运营、展销财务和学员访客，提供工坊档案、传承人档案、课程目录、工坊排期、课程预约、预约审核、课程签到、作品档案、作品展销、展销订单、展销结算和操作日志的一体化管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 工坊管理员：维护工坊档案、传承人档案、课程目录和工坊排期
- 非遗讲师：维护课程内容、处理课程预约、预约审核和课程签到
- 展陈运营：维护作品档案、作品展销和展陈状态
- 展销财务：核对展销订单和展销结算闭环
- 学员访客：查看课程排期、提交课程预约、查看作品展销和订单记录

## 功能模块
- 工坊档案：工坊编号、工坊名称、非遗门类、负责人、建档时间和工坊状态维护
- 传承人档案：传承人编号、传承人姓名、技艺方向、管理人员、建档时间和档案状态维护
- 课程目录：课程编号、课程名称、课程类型、授课讲师、启用时间和课程状态维护
- 工坊排期：排期编号、课程名称、排期类型、授课讲师、开课时间和排期状态维护
- 课程预约：预约编号、课程名称、预约类型、预约学员、预约时间和预约状态维护
- 预约审核：审核编号、课程预约、审核类型、审核人员、审核时间和审核状态维护
- 课程签到：签到编号、课程排期、签到类型、记录人员、签到时间和签到状态维护
- 作品档案：作品编号、作品名称、作品类型、创作人员、入库时间和作品状态维护
- 作品展销：展销编号、作品名称、展销类型、展陈人员、上架时间和展销状态维护
- 展销订单：订单编号、作品名称、订单类型、下单用户、下单时间和订单状态维护
- 展销结算：结算编号、结算周期、结算类型、财务人员、结算时间和结算状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

## 技术栈
- 后端：Spring Boot 2.7.18、MyBatis 注解 SQL、PageHelper、MySQL、Redis、JWT
- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
- 数据库：heritage_workshop_200
