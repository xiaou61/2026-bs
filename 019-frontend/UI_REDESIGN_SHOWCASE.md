# 019-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园运动健康管理平台
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 活力红
- **美化完成时间**: 2026-05-29

## 设计理念

采用"活力红"设计理念，以红色和橙色为主色调，体现运动的活力和激情。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 活力红 | #E53935 | 主色调 |
| 能量橙 | #FF6D00 | 辅助色 |
| 活力绿 | #43A047 | 强调色 |
| 背景色 | #F5F5F5 | 页面背景 |

## 设计特点

### 登录页面
- 红色渐变背景
- 跑步、足球、健身装饰动画
- 毛玻璃效果登录框

## 总结

019-frontend采用"活力红"设计理念，通过红色和橙色的搭配，营造出充满活力和激情的运动氛围。运动图标装饰元素突出了运动健康管理的功能特色。
