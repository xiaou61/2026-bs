# 111-frontend UI 美化展示

## 项目信息
- **项目名称**: 网络钓鱼邮件演练与安全意识培训平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 警示红
- **美化完成时间**: 2026-05-30

## 设计理念

采用"警示红"设计理念，以红色为主色调，体现网络安全的警示性、紧迫性和重要性。通过鱼钩、邮件等装饰元素，营造出识别钓鱼、安全上网的安全培训氛围。
