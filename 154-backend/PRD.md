# 宠物医院接诊排班与疫苗随访管理系统 PRD

## 项目概述
本系统面向宠物医院门诊运营场景，提供宠主建档、宠物档案、医生排班、接诊预约、疫苗计划、接种记录、随访记录和费用结算的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 前台接诊员：宠主档案、宠物档案、接诊预约、费用结算和统计看板维护
- 宠物医生：医生档案、接诊记录、疫苗计划和随访记录维护
- 护理接种员：疫苗计划、疫苗接种、药品库存和随访记录维护
- 宠物主人：宠物档案、接诊预约、疫苗提醒、随访记录和费用结算查询

## 功能模块
- 宠主档案：宠主编号、宠主姓名、联系方式、常住区域、建档时间和账号状态维护
- 宠物档案：宠物编号、宠物昵称、宠物类型、宠主姓名、出生日期和健康状态维护
- 医生档案：医生工号、医生姓名、擅长科室、所属诊室、坐诊时间和在岗状态维护
- 接诊排班：排班编号、坐诊医生、诊室区域、班次类型、排班时间和排班状态维护
- 接诊预约：预约编号、宠物名称、接诊类型、预约人、预约时间和预约状态维护
- 接诊记录：接诊编号、关联预约、疾病类型、接诊医生、接诊时间和接诊状态维护
- 疫苗计划：计划编号、宠物名称、疫苗类型、负责护士、计划时间和提醒状态维护
- 疫苗接种：接种编号、疫苗计划、疫苗批次、接种护士、接种时间和接种状态维护
- 随访记录：随访编号、关联接诊、随访方式、随访人、随访时间和随访状态维护
- 药品库存：药品编号、药品名称、药品类型、管理人、有效日期和库存状态维护
- 费用结算：结算编号、关联接诊、费用类型、收费员、结算时间和结算状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
