# 052-frontend UI 美化展示

## 项目信息
- **项目名称**: 在线课程学习平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 学习橙
- **美化完成时间**: 2026-05-29

## 设计理念

采用"学习橙"设计理念，以橙色和金色为主色调，体现学习的热情、活力和进取精神。通过书本、毕业帽等装饰元素，营造出积极向上的学习氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 学习橙 | #FF6F00 | 主色调 |
| 活力橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 深橙 | #E65100 | 文字色 |
| 亮金 | #FFB300 | 装饰色 |

## 设计特点

- 橙色渐变背景

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳效果 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

052-frontend采用"学习橙"设计理念，通过橙色和金色的搭配，营造出积极向上的学习氛围。书本和毕业帽装饰元素突出了在线学习的特色。
