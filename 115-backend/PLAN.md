# 实现计划

## 问题陈述

构建一个围绕跨境订单、清关申报和汇率结算的前后端分离系统，覆盖商家店铺、客户档案、商品 SKU、订单履约、税费记录、支付结算、物流跟踪和对账管理。

## 当前状态

`115-backend` 与 `115-frontend` 已完成跨境电商结算平台的基础实现，当前进入精修阶段，重点核对认证安全、订单结算、清关流程和文档准确性。

## 实施方案

### 第一阶段：后端开发
1. Spring Boot 2.7.18 + MyBatis-Plus 3.5.5 + Redis + JWT
2. 创建 `cross_border_115` 数据库和 14 张业务表
3. 实现统一返回、异常处理、JWT 拦截和 Redis Token 校验
4. 实现 Entity、Mapper、Service 和 Controller
5. 实现订单推进、清关申报、汇率结算、支付核销和对账处理
6. 实现首页统计和操作日志

### 第二阶段：前端开发
1. Vue3 + Vite + Element Plus + Pinia + Axios + ECharts
2. 登录页、布局页、首页看板
3. 商家店铺、客户档案、商品 SKU、跨境订单、清关申报、税费记录、汇率牌价、结算账单、物流跟踪、订单对账和日志页面
4. 业务按钮支持申报、结算、支付、对账和查看物流

### 第三阶段：文档更新
1. 更新 `readme.md`
2. 更新 `readme_simple.md`
3. 更新候选清单

## 完成标准

1. 后端结构完整。
2. 前端结构完整。
3. SQL 包含演示数据。
4. 认证返回不暴露敏感字段。
5. 订单、清关、税费、结算和对账口径一致。
6. 文档描述与实际实现保持一致。
7. 不执行编译验证。
