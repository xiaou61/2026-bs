# 运动康复训练计划与体测评估管理系统 PRD

## 项目概述
本系统面向运动康复中心、体测评估师、康复教练、康复治疗师和训练会员，提供康复中心、会员档案、教练档案、体测项目、体测评估、风险提醒、训练计划、训练安排、训练打卡、康复反馈、复评记录和操作日志的一体化管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 中心管理员：维护康复中心、会员档案、教练档案和体测项目
- 体测评估师：处理体测评估、风险提醒和复评记录
- 康复教练：制定训练计划、维护训练安排并跟进训练打卡
- 康复治疗师：查看评估结果、维护康复反馈并参与复评
- 训练会员：查看个人体测、训练计划、训练打卡和康复反馈

## 功能模块
- 康复中心：中心编号、中心名称、服务方向、负责人、建档时间和中心状态维护
- 会员档案：会员编号、会员姓名、康复目标、负责教练、建档时间和会员状态维护
- 教练档案：教练编号、教练姓名、擅长项目、管理人员、入职时间和教练状态维护
- 体测项目：项目编号、项目名称、评估类型、维护人员、启用时间和项目状态维护
- 体测评估：评估编号、会员姓名、评估类型、评估人员、评估时间和评估状态维护
- 风险提醒：提醒编号、会员姓名、风险类型、评估人员、提醒时间和处理状态维护
- 训练计划：计划编号、会员姓名、训练目标、制定教练、计划时间和计划状态维护
- 训练安排：安排编号、训练计划、训练类型、指导教练、训练时间和训练状态维护
- 训练打卡：打卡编号、训练安排、打卡类型、记录人员、打卡时间和打卡状态维护
- 康复反馈：反馈编号、训练安排、反馈类型、康复治疗师、反馈时间和反馈状态维护
- 复评记录：复评编号、会员姓名、复评类型、评估人员、复评时间和复评状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
