# 147-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园心理咨询预约与危机干预管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 心理绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"心理绿"设计理念，以绿色为主色调，体现心理健康的自然、和谐和治愈感。通过大脑、爱心等装饰元素，营造出关爱心灵、拥抱阳光的心理咨询氛围。
