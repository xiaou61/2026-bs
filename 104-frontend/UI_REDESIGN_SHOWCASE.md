# 104-frontend UI 美化展示

## 项目信息
- **项目名称**: 开源许可证合规扫描与项目台账系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 合规蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"合规蓝"设计理念，以蓝色为主色调，体现合规管理的专业性、严谨性和可靠性。通过扫描、检查等装饰元素，营造出开源合规、安全可控的管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | complianceGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
