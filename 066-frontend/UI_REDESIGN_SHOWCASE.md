# 066-frontend UI 美化展示

## 项目信息
- **项目名称**: 精简博客系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 文艺紫
- **美化完成时间**: 2026-05-29

## 设计理念

采用"文艺紫"设计理念，以紫色和粉色为主色调，体现博客的文艺、浪漫和创意。通过笔、书本等装饰元素，营造出记录生活、分享精彩的博客氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 文艺紫 | #764ba2 | 主色调 |
| 靛蓝 | #667eea | 辅助色 |
| 粉紫 | #f093fb | 强调色 |
| 玫红 | #f5576c | 注册色 |
| 暗紫 | #5b21b6 | 文字色 |

## 设计特点

### 登录页面
- 紫粉渐变背景
- 文艺装饰元素浮动
- 笔记📝Logo弹跳动画
- 流光渐变顶部条

- 博客名言展示

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳效果 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

066-frontend采用"文艺紫"设计理念，通过紫色和粉色的搭配，营造出文艺、浪漫的博客氛围。笔和书本装饰元素突出了博客的创作特色。
