# 081-frontend UI 美化展示

## 项目信息
- **项目名称**: 电器维修管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 维修橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"维修橙"设计理念，以橙色和金色为主色调，体现电器维修的专业性、活力和可靠感。通过插头、闪电等装饰元素，营造出专业维修、安心使用的服务氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 维修橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |
| 深橙 | #E65100 | 文字色 |

## 设计特点

### 登录页面
- 橙色渐变背景

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | repairGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

081-frontend采用"维修橙"设计理念，通过橙色和金色的搭配，营造出专业、活力的电器维修管理氛围。电器装饰元素突出了电器维修的行业特色。
