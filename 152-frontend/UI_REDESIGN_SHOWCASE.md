# 152-frontend UI 美化展示

## 项目信息
- **项目名称**: 工厂危险作业审批与监护巡检管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 安全红
- **美化完成时间**: 2026-05-30

## 设计理念

采用"安全红"设计理念，以红色为主色调，体现危险作业的警示性、紧迫性和重要性。通过警告、工厂等装饰元素，营造出安全第一、预防为主的工厂安全管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|

| 浅红 | #E53935 | 装饰色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 快速脉冲缩放 |
| 顶部条 | safetyGradient | 快速流光渐变 |
| 按钮 | hover | 悬浮上移 |
