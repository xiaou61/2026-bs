# 博物馆文物数字化管理平台

基于 SpringBoot + Vue3 的博物馆文物数字化管理平台

## 功能模块

### 管理员端
- 用户管理：管理系统用户、讲解员、研究员
- 文物管理：文物信息录入、编辑、上下架
- 分类管理：文物分类维护
- 朝代管理：朝代信息维护
- 展厅管理：展厅信息管理
- 展览管理：展览活动发布与管理
- 研究管理：研究成果审核
- 预约管理：参观预约审核确认
- 公告管理：系统公告、展览通知发布

### 讲解员端
- 预约管理：查看和处理讲解预约订单

### 研究员端
- 研究成果：提交、编辑研究成果

### 游客端
- 文物浏览：查看文物信息、点赞、收藏
- 展览预约：查看展览并在线预约
- 讲解预约：预约讲解服务
- 研究成果：查看已发布的研究成果
- 我的预约：管理个人预约记录
- 收藏管理：查看收藏的文物
- 公告中心：查看系统公告

## 技术栈

### 后端
- SpringBoot 2.7
- MyBatis-Plus
- MySQL 8.0
- JWT 认证

### 前端
- Vue 3
- Element Plus
- Vite
- Pinia
- Vue Router

## 运行说明

### 后端
1. 创建数据库 `museum_relic`
2. 执行 `sql/init.sql` 初始化表结构和数据
3. 修改 `application.yml` 数据库配置
4. 运行 `MuseumRelicApplication`

### 前端
```bash
cd 048-frontend
npm install
npm run dev
```
