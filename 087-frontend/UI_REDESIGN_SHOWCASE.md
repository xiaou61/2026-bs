# 087-frontend UI 美化展示

## 项目信息
- **项目名称**: 高校教务管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 教务紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"教务紫"设计理念，以紫色和蓝色为主色调，体现高校教务的学术氛围、专业性和权威感。通过毕业帽、书本等装饰元素，营造出智慧教学、高效管理的教务管理氛围。
