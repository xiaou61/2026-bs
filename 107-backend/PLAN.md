# 实现计划

## 问题陈述

构建一个围绕云资源监控、告警处置和运维工单闭环的前后端分离系统，覆盖云区域、主机资产、指标采集、告警通知、工单处置、维护窗口和容量规划。

## 当前状态

`107-backend` 与 `107-frontend` 已完成云监控系统的基础实现，当前进入精修阶段，重点核对认证安全、告警流转、看板口径和文档准确性。

## 实施方案

### 第一阶段：后端开发
1. Spring Boot 2.7.18 + MyBatis-Plus 3.5.5 + Redis + JWT
2. 创建 `cloud_monitor_107` 数据库和 14 张业务表
3. 实现统一返回、异常处理、JWT 拦截和 Redis Token 校验
4. 实现 Entity、Mapper、Service 和 Controller
5. 实现告警处置、工单闭环、维护窗口和容量分析
6. 实现首页统计和操作日志

### 第二阶段：前端开发
1. Vue3 + Vite + Element Plus + Pinia + Axios + ECharts
2. 登录页、布局页、首页看板
3. 云区域、主机资产、指标、告警、通知、工单、维护窗口、容量规划和日志页面
4. 业务按钮支持确认、派单、关闭、维护和查看趋势

### 第三阶段：文档更新
1. 更新 `readme.md`
2. 更新 `readme_simple.md`
3. 更新候选清单

## 完成标准

- 后端结构完整
- 前端结构完整
- SQL 包含演示数据
- 认证返回不暴露敏感字段
- 告警、通知、工单和容量分析口径一致
- 文档描述与实际实现保持一致
- 不执行编译验证
