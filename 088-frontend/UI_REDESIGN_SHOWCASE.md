# 088-frontend UI 美化展示

## 项目信息
- **项目名称**: 孩童收养信息平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 爱心粉
- **美化完成时间**: 2026-05-30

## 设计理念

采用"爱心粉"设计理念，以粉色和玫红色为主色调，体现孩童收养的温暖、关爱和希望。通过婴儿、爱心等装饰元素，营造出用爱守护、温暖成长的公益氛围。

## 设计特点

### 登录页面
- 粉色渐变背景

- 流光渐变顶部条
- 渐变登录按钮

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | heartbeat | 心跳效果 |
| 顶部条 | loveGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

088-frontend采用"爱心粉"设计理念，通过粉色和玫红色的搭配，营造出温暖、关爱的孩童收养信息氛围。婴儿心跳动画突出了孩童收养的公益特色。
