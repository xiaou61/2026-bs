# 169-frontend UI 美化展示

## 项目信息
- **项目名称**: 校车乘车预约与学生上下车核验系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 校车橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"校车橙"设计理念，以橙色和金色为主色调，体现校车服务的温暖、安全和活力。通过校车、书包等装饰元素，营造出安全校车、守护上学路的校园交通氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 校车橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |
| 深橙 | #E65100 | 文字色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | drive | 横向行驶 |
| 顶部条 | busGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

169-frontend采用"校车橙"设计理念，通过橙色和金色的搭配，营造出温暖、安全的校车管理氛围。校车行驶动画突出了校车乘车预约的特色。
