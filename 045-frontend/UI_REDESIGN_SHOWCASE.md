# 045-frontend UI 美化展示

## 项目信息
- **项目名称**: 养老院管理系统
- **技术栈**: Vue 3 + TypeScript + Element Plus
- **设计主题**: 关爱粉
- **美化完成时间**: 2026-05-29

## 设计理念

采用"关爱粉"设计理念，以粉色和玫红色为主色调，体现对老人的关爱、温暖和尊重。通过爱心、房屋等装饰元素，营造出温馨、有爱的养老院管理氛围。

| 颜色 | 色值 | 用途 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
