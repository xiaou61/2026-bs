# 007-frontend UI 美化展示

## 项目信息
- **项目名称**: 志愿活动管理与积分激励平台
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 公益绿
- **美化完成时间**: 2026-05-29

## 设计理念

采用"公益绿"设计理念，以绿色和棕色为主色调，体现志愿公益的温暖感和自然气息。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 公益绿 | #2E7D32 | 主色调 |
| 暖棕色 | #795548 | 辅助色 |
| 暖橙色 | #FF8F00 | 强调色 |
| 背景色 | #FAFAFA | 页面背景 |
