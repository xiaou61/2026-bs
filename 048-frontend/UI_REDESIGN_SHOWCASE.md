# 048-frontend UI 美化展示

## 项目信息
- **项目名称**: 博物馆文物数字化管理平台
- **技术栈**: Vue 3 + TypeScript + Element Plus
- **设计主题**: 古典棕
- **美化完成时间**: 2026-05-29

## 设计理念

采用"古典棕"设计理念，以棕色和古铜色为主色调，体现博物馆的历史厚重感和文化底蕴。通过博物馆、陶器等装饰元素，营造出古典、典雅的文物管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|

| 古铜 | #CD853F | 强调色 |
| 暗棕 | #3E2723 | 文字色 |
| 米色 | #DEB887 | 装饰色 |

## 设计特点

### 登录页面
- 棕色渐变背景
- 博物馆文物装饰浮动
- 博物馆🏛️Logo脉冲动画
- 流光渐变顶部条
- 渐变登录按钮
- 文化传承标语

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

048-frontend采用"古典棕"设计理念，通过棕色和古铜色的搭配，营造出古典、典雅的博物馆氛围。文物装饰元素突出了博物馆的文化底蕴。
