# 118-frontend UI 美化展示

## 项目信息
- **项目名称**: 智能仓储AGV任务调度与库位优化系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 仓储蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"仓储蓝"设计理念，以蓝色为主色调，体现智能仓储的科技感、专业性和效率感。通过机器人、仓库等装饰元素，营造出智能仓储、AGV调度的现代化物流氛围。

## 设计特点

### 登录页面
- 蓝色渐变背景
- 智能仓储装饰浮动
- 机器人🤖Logo移动动画
- 流光渐变顶部条
- 测试账号标签
- AGV调度标语

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | moveRobot | 横向移动 |
| 顶部条 | agvGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

118-frontend采用"仓储蓝"设计理念，通过蓝色的主色调，营造出科技、专业的智能仓储氛围。机器人移动动画突出了AGV自动导引车的特色。
