# 158-frontend UI 美化展示

## 项目信息
- **项目名称**: 校外培训机构课消统计与退费审批系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 培训绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"培训绿"设计理念，以绿色为主色调，体现培训教育的生机、成长和希望。通过书本、毕业帽等装饰元素，营造出学无止境、培训提升的教育管理氛围。
