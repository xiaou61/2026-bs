# 091-frontend UI 美化展示

## 项目信息
- **项目名称**: 电影院会员管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 影院红
- **美化完成时间**: 2026-05-30

## 设计理念

采用"影院红"设计理念，以红色和暖色为主色调，体现电影院的热情、精彩和娱乐感。通过电影、爆米花等装饰元素，营造出光影世界的会员管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 影院红 | #e94560 | 主色调 |
| 暖红 | #ff6b6b | 辅助色 |
| 金色 | #ffd93d | 强调色 |
| 暗蓝 | #1a1a2e | 背景色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放旋转 |
| 顶部条 | cinemaGradient | 彩虹流光 |
| 按钮 | hover | 悬浮上移 |

## 总结

091-frontend采用"影院红"设计理念，通过红色和暖色的搭配，营造出精彩的电影院会员管理氛围。电影装饰元素突出了电影院的娱乐特色。
