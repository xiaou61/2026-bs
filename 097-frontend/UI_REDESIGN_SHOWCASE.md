# 097-frontend UI 美化展示

## 项目信息
- **项目名称**: 大模型提示词资产管理与效果评测平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: AI青
- **美化完成时间**: 2026-05-30

## 设计理念

采用"AI青"设计理念，以青色和深蓝色为主色调，体现AI平台的科技感、专业性和未来感。通过机器人、灯泡等装饰元素，营造出AI赋能、智能未来的科技氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 背景光晕 | glow | 呼吸效果 |
| 顶部条 | aiGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
