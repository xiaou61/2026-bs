# 102-frontend UI 美化展示

## 项目信息
- **项目名称**: 法律咨询案件进度与智能文书管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 法律紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"法律紫"设计理念，以紫色和深蓝色为主色调，体现法律行业的庄重、权威和专业感。通过天平、法典等装饰元素，营造出公平正义、法治社会的法律管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | legalGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
