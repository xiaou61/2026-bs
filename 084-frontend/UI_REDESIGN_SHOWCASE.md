# 084-frontend UI 美化展示

## 项目信息
- **项目名称**: 教学资料管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 教学绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"教学绿"设计理念，以绿色为主色调，体现教学资料管理的生机、希望和成长感。通过书本、毕业帽等装饰元素，营造出知识共享、教学相长的教育氛围。

| 颜色 | 色值 | 用途 |

## 设计特点

### 登录页面
- 绿色渐变背景
- 教学装饰元素浮动
- 书本📖Logo弹跳动画
- 流光渐变顶部条
- 教学资料标语

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳缩放 |
| 顶部条 | teachingGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

084-frontend采用"教学绿"设计理念，通过绿色的主色调，营造出生机、希望的教学资料管理氛围。书本装饰元素突出了教学资料管理的教育特色。
