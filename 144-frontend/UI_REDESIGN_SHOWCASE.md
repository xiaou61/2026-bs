# 144-frontend UI 美化展示

## 项目信息
- **项目名称**: 无障碍出行路线规划与志愿协助平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 助残粉
- **美化完成时间**: 2026-05-30

## 设计理念

采用"助残粉"设计理念，以粉色和玫红色为主色调，体现无障碍出行的温暖、关爱和包容。通过轮椅、爱心等装饰元素，营造出无障碍出行、温暖同行的助残服务氛围。
