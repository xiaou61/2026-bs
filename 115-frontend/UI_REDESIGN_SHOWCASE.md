# 115-frontend UI 美化展示

## 项目信息
- **项目名称**: 跨境电商清关订单与汇率结算平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 跨境橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"跨境橙"设计理念，以橙色和金色为主色调，体现跨境电商的热情、活力和国际化。通过地球、飞机等装饰元素，营造出买全球、卖全球的跨境电商氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 跨境橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |
| 深橙 | #E65100 | 文字色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | spin | 持续旋转 |
| 顶部条 | crossBorderGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

115-frontend采用"跨境橙"设计理念，通过橙色和金色的搭配，营造出热情、国际化的跨境电商氛围。地球旋转动画突出了全球贸易的特色。
