# 098-frontend UI 美化展示

## 项目信息
- **项目名称**: 企业知识库智能问答与文档权限管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 知识青
- **美化完成时间**: 2026-05-30

## 设计理念

采用"知识青"设计理念，以青色和蓝绿色为主色调，体现知识管理的专业性、智慧和成长感。通过书本、灯泡等装饰元素，营造出知识就是力量的学习氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 背景光晕 | glow | 呼吸效果 |
| 顶部条 | knowledgeGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
