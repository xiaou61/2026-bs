# 社区家政服务预约与人员信用评价系统 PRD

## 项目定位
社区家政服务预约与人员信用评价系统面向毕业设计答辩场景，围绕家政服务方向，提供多角色登录、业务台账维护、状态流转、统计看板和操作留痕能力。

## 用户角色
系统管理员、审核人员、运营人员、服务对象

## 功能模块
- 服务预约：服务预约编号、服务预约名称、业务类型、负责人、计划时间和状态维护
- 人员排班：人员排班编号、人员排班名称、业务类型、负责人、计划时间和状态维护
- 信用评价：信用评价编号、信用评价名称、业务类型、负责人、计划时间和状态维护
- 投诉处理：投诉处理编号、投诉处理名称、业务类型、负责人、计划时间和状态维护
- 基础档案：基础档案编号、基础档案名称、业务类型、负责人、计划时间和状态维护
- 人员档案：人员档案编号、人员档案名称、业务类型、负责人、计划时间和状态维护
- 资源台账：资源台账编号、资源台账名称、业务类型、负责人、计划时间和状态维护
- 服务申请：服务申请编号、服务申请名称、业务类型、负责人、计划时间和状态维护
- 审批记录：审批记录编号、审批记录名称、业务类型、负责人、计划时间和状态维护
- 执行记录：执行记录编号、执行记录名称、业务类型、负责人、计划时间和状态维护
- 统计分析：统计分析编号、统计分析名称、业务类型、负责人、计划时间和状态维护
- 通知公告：通知公告编号、通知公告名称、业务类型、负责人、计划时间和状态维护
- 操作日志：关键模块操作人、动作、对象、详情和结果记录

## 技术要求
- 后端：Spring Boot 2.7.18、MyBatis 2.3.1 注解 SQL、PageHelper 1.4.7、MySQL、Redis、JWT
- 前端：Vue3、Vite、Element Plus、Pinia、Axios、ECharts
- 数据库：project_197
- 默认账号密码：各角色账号均使用 `123456`

## 特色亮点
- 选题贴合家政服务方向，核心能力覆盖服务预约、人员排班、信用评价、投诉处理
- 支持列表查询、新增编辑、删除、状态处理、状态完成和统计展示
- 通用数据页复用统一的搜索、表格、分页、弹窗和状态动作交互
