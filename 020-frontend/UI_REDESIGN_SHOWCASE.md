# 020-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园学习资源共享平台
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 知识蓝
- **美化完成时间**: 2026-05-29

## 设计理念

采用"知识蓝"设计理念，以蓝色和紫色为主色调，体现知识的深度和智慧。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 知识蓝 | #0288D1 | 主色调 |
| 智慧紫 | #7B1FA2 | 辅助色 |
| 创新绿 | #00897B | 强调色 |
| 背景色 | #F5F7FA | 页面背景 |

020-frontend采用"知识蓝"设计理念，通过蓝色和紫色的搭配，营造出深邃、智慧的学习氛围。书籍和灯泡装饰元素突出了学习资源共享的功能特色。
