# 151-frontend UI 美化展示

## 项目信息
- **项目名称**: 文旅场馆讲解预约与票务核销管理平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 文旅棕
- **美化完成时间**: 2026-05-30

## 设计理念

采用"文旅棕"设计理念，以棕色和古铜色为主色调，体现文旅场馆的文化底蕴、历史感和典雅。通过博物馆、门票等装饰元素，营造出文化传承、智慧文旅的文旅管理氛围。
