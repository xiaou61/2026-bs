# 117-frontend UI 美化展示

## 项目信息
- **项目名称**: 本地生活服务券核销与商户结算系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 生活橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"生活橙"设计理念，以橙色和金色为主色调，体现本地生活服务的热情、活力和优惠感。通过优惠券、商店等装饰元素，营造出本地生活、优惠共享的服务氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | localGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

117-frontend采用"生活橙"设计理念，通过橙色和金色的搭配，营造出热情、优惠的本地生活服务氛围。优惠券装饰元素突出了服务券核销的特色。
