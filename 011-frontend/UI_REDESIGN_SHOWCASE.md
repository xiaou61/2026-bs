# 011-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园短视频创作与分享平台
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 视频紫
- **美化完成时间**: 2026-05-29

## 设计理念

采用"视频紫"设计理念，以紫色和珊瑚粉为主色调，体现短视频创作的创意和活力。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 视频紫 | #7C4DFF | 主色调 |
| 珊瑚粉 | #FF6B6B | 辅助色 |
| 青柠绿 | #69F0AE | 强调色 |
| 背景色 | #F5F5F5 | 页面背景 |
