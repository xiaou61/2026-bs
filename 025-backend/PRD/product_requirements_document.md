# 智慧社区综合服务管理平台 - 产品需求文档 (PRD)

## 1. 项目概述
**项目名称**：智慧社区综合服务管理平台
**项目背景**：随着城市化进程加快，社区管理面临着信息不对称、服务效率低、缴费不便等问题。本平台旨在通过数字化手段，提升物业管理效率，为业主提供便捷的社区服务。
**技术栈**：
- 后端：Spring Boot, MyBatis, MySQL
- 前端：Vue 3, Element Plus (或类似UI库)

## 2. 用户角色
1.  **管理员 (物业人员)**：拥有系统所有权限，负责信息录入、审核、管理。
2.  **业主 (普通用户)**：查看公告、报修、缴费、登记访客等。

## 3. 功能模块详解

### 3.1 业主信息管理 (Owner Management)
**功能描述**：管理社区内业主的基本信息。
- **管理员**：
    - 录入业主信息（姓名、电话、楼栋号、单元号、房号、入住时间）。
    - 查询、编辑、删除业主信息。
    - 导入/导出业主数据（可选）。
- **业主**：
    - 查看个人信息。
    - 修改个人密码/联系方式。

### 3.2 物业缴费 (Property Fee Payment)
**功能描述**：模拟物业费用的生成与缴纳流程。
- **管理员**：
    - 生成账单（按月/年，输入金额）。
    - 查看缴费记录（已缴/未缴）。
    - 催缴提醒（模拟发送通知）。
- **业主**：
    - 查看待缴账单。
    - 模拟支付（点击“支付”按钮，状态变更为“已缴”）。
    - 查看历史缴费记录。

### 3.3 在线报修 (Online Repair)
**功能描述**：业主提交维修申请，物业进行处理。
- **业主**：
    - 提交报修申请（填写报修内容、上传图片、期望上门时间）。
    - 查看报修进度（待处理、处理中、已完成）。
    - 对维修服务进行评价。
- **管理员**：
    - 查看报修列表。
    - 指派维修人员（或直接接单）。
    - 更新维修状态（处理中 -> 已完成）。

### 3.4 社区公告 (Community Announcements)
**功能描述**：发布社区通知、新闻、温馨提示。
- **管理员**：
    - 发布新公告（标题、内容、发布时间）。
    - 编辑/删除公告。
    - 置顶重要公告。
- **业主**：
    - 浏览公告列表。
    - 查看公告详情。

### 3.5 访客登记 (Visitor Registration)
**功能描述**：管理外来访客，提升社区安全。
- **业主**：
    - 预约访客（填写访客姓名、预计到达时间、车牌号）。
    - 生成访客码（模拟）。
- **管理员/门岗**：
    - 查看今日访客列表。
    - 登记临时访客信息。
    - 确认访客离场。

### 3.6 停车位管理 (Parking Space Management)
**功能描述**：管理社区停车位资源。
- **管理员**：
    - 车位信息录入（车位号、区域、状态：空闲/已售/已租）。
    - 车位分配/租赁登记（绑定业主）。
- **业主**：
    - 查看我的车位。
    - 查询空闲车位。

## 4. 非功能需求
- **安全性**：用户密码加密存储，接口鉴权。
- **易用性**：界面简洁清晰，操作流程符合直觉。
- **性能**：页面加载速度快，响应时间在合理范围内。

## 5. 数据字典 (简略)
- **User (用户表)**: id, username, password, role, ...
- **Owner (业主表)**: id, user_id, name, phone, building, unit, room, ...
- **Fee (费用表)**: id, owner_id, amount, type, status, create_time, pay_time, ...
- **Repair (报修表)**: id, owner_id, content, image, status, ...
- **Notice (公告表)**: id, title, content, create_time, ...
- **Visitor (访客表)**: id, owner_id, visitor_name, visit_time, ...
- **Parking (车位表)**: id, spot_number, status, owner_id, ...
