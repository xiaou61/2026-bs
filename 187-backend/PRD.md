# 旅行社团建行程报名与费用分摊管理平台 PRD

## 项目概述
本系统面向旅行社团建行程组织、成员报名确认、预算编制、费用分摊、缴费记录、通知同步和出行反馈场景，提供旅行社档案、团队档案、团建行程、行程报名、成员确认、费用预算、费用分摊、缴费记录、通知同步、出行反馈和发票记录的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 旅行社管理员：维护旅行社档案、团队档案、行程规则和协作配置
- 行程策划：编排团建行程、预算项目和报名节奏
- 财务人员：维护费用预算、费用分摊、缴费记录和发票记录
- 团队领队：组织成员报名确认、通知同步和出行反馈
- 团建成员：查看行程安排、确认报名、缴费和提交反馈

## 功能模块
- 旅行社档案：旅行社编号、旅行社名称、服务类型、负责人员、合作时间和旅行社状态维护
- 团队档案：团队编号、团队名称、团队类型、领队人员、建档时间和团队状态维护
- 团建行程：行程编号、行程名称、行程类型、策划人员、出行时间和行程状态维护
- 行程报名：报名编号、关联行程、报名类型、报名人员、报名时间和报名状态维护
- 成员确认：确认编号、报名记录、确认类型、确认人员、确认时间和确认状态维护
- 费用预算：预算编号、预算项目、预算类型、编制人员、编制时间和预算状态维护
- 费用分摊：分摊编号、费用项目、分摊类型、财务人员、分摊时间和分摊状态维护
- 缴费记录：缴费编号、分摊费用、缴费类型、缴费人员、缴费时间和缴费状态维护
- 通知同步：通知编号、通知对象、通知类型、发布人员、发布时间和通知状态维护
- 出行反馈：反馈编号、关联行程、反馈类型、反馈人员、反馈时间和反馈状态维护
- 发票记录：发票编号、费用项目、发票类型、财务人员、开票时间和发票状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护
