# 183-frontend UI 美化展示

## 项目信息
- **项目名称**: 实验动物饲养排班与伦理审批管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 实验蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"实验蓝"设计理念，以蓝色为主色调，体现实验室的专业性、严谨性和科学感。通过老鼠、试管等装饰元素，营造出保护动物、科学实验的实验室管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | squeak | 左右摇摆 |
| 顶部条 | labGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
