# 实现计划

## 问题陈述

构建一个围绕家庭用电分析、设备能耗和节能建议的前后端分离系统，覆盖家庭档案、家庭成员、智能电表、用电设备、用电读数、电费账单、能耗预算和低碳管理。

## 当前状态

`126-backend` 与 `126-frontend` 已完成家庭能源平台的基础实现，当前进入精修阶段，重点核对能耗统计、节能建议、认证安全和文档准确性。

## 实施方案

### 第一阶段：后端开发
1. Spring Boot 2.7.18 + MyBatis 2.3.1 + PageHelper + Redis + JWT
2. 创建 `home_energy_126` 数据库和 14 张业务表
3. 实现统一返回、异常处理、JWT 拦截、Redis 配置和分页查询
4. 实现 Entity、Mapper、Service 和 Controller
5. 实现用电分析、账单统计、预算预警、设备能耗和节能建议
6. 实现首页统计和操作日志

### 第二阶段：前端开发
1. Vue3 + Vite + Element Plus + Pinia + Axios + ECharts
2. 登录页、布局页、首页看板
3. 家庭档案、家庭成员、智能电表、用电设备、用电读数、电费账单、设备用电、能耗预算和日志页面
4. 业务按钮支持录入、分析、预警确认和查看节能建议

### 第三阶段：文档更新
1. 更新 `readme.md`
2. 更新 `readme_simple.md`
3. 更新候选清单

## 完成标准

1. 后端结构完整。
2. 前端结构完整。
3. SQL 包含演示数据。
4. 认证返回不暴露敏感字段。
5. 用电读数、账单统计、预算预警和节能建议口径一致。
6. 文档描述与实际实现保持一致。
7. 不执行编译验证。
