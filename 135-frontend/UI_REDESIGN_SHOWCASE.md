# 135-frontend UI 美化展示

## 项目信息
- **项目名称**: 学术会议投稿评审与日程管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 学术紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"学术紫"设计理念，以紫色为主色调，体现学术会议的庄重、专业和创新感。通过毕业帽、书本等装饰元素，营造出学术交流、知识共享的学术会议氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳效果 |
| 顶部条 | conferenceGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
