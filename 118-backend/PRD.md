# 智能仓储 AGV 任务调度与库位优化系统 PRD

## 项目背景
面向自动化仓库、AGV调度中心、仓管员和维保人员，覆盖仓库区域、库位档案、AGV车辆、任务调度、路径规划、库位推荐、库存出入库、设备维保和异常告警。

## 用户角色
1. 管理员：维护账号权限、仓库基础资料和系统配置。
2. 调度员：派发AGV任务、选择路径并处理异常告警。
3. 仓管员：维护库存物料、入库订单、出库订单和库位状态。
4. 维保员：处理AGV车辆、充电站和传感器维保任务。

## 功能模块
1. 账号权限：管理员、调度员、仓管员、维保员四角色账号和启停状态管理
2. 仓库区域：仓库区域、温区类型、巷道数量、负责人和启用状态管理
3. 库位档案：库位编码、所属区域、承重、体积、热度等级和占用状态管理
4. AGV车辆：AGV车辆编号、型号、当前区域、电量、任务数量和运行状态管理
5. 充电站点：充电站编号、所属区域、端口数量、空闲端口和运行状态管理
6. 库存物料：物料SKU、批次、库位、数量、周转等级和库存状态管理
7. 入库订单：入库单号、供应商、物料、目标库位、预约时间和入库状态管理
8. 出库订单：出库单号、客户、物料、拣选库位、优先级和出库状态管理
9. AGV任务：AGV任务单、任务类型、车辆、起止库位、优先级和执行状态管理
10. 路径规划：任务路径、起止点、里程、预计耗时、拥堵等级和路径状态管理
11. 库位推荐：推荐单、物料、推荐库位、匹配分、推荐理由和采纳状态管理
12. 设备维保：AGV或充电站维保计划、故障类型、处理人、计划时间和维保状态管理
13. 异常告警：任务、车辆、库位、告警等级、告警内容、处理人和告警状态管理
14. 操作日志：关键模块操作人、动作、对象、详情、IP地址和执行状态审计

## 核心流程
1. 管理员维护仓库区域、库位档案、AGV车辆和充电站点。
2. 仓管员创建入库或出库订单，系统沉淀库存物料和目标库位。
3. 调度员依据订单生成AGV任务并选择路径规划方案。
4. 系统根据物料周转、承重、温区和热度生成库位推荐。
5. AGV执行搬运任务，异常告警进入处理闭环。
6. 维保员处理设备维保，首页看板展示任务趋势和库区分布。
