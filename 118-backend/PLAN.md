# 实现计划

## 问题陈述

构建一个围绕 AGV 调度、库位优化和仓储异常处置的前后端分离系统，覆盖仓库区域、库位档案、AGV 车辆、入出库订单、路径规划、库位推荐、设备维保和异常告警。

## 当前状态

`118-backend` 与 `118-frontend` 已完成智能仓储系统的基础实现，当前进入精修阶段，重点核对调度闭环、认证安全、库位优化和文档准确性。

## 实施方案

### 第一阶段：后端开发
1. Spring Boot 2.7.18 + MyBatis 2.3.1 + PageHelper + Redis + JWT
2. 创建 `smart_warehouse_agv_118` 数据库和 14 张业务表
3. 实现统一返回、异常处理、JWT 拦截、Redis 配置和分页查询
4. 实现 Entity、Mapper、Service 和 Controller
5. 实现 AGV 任务调度、路径规划、库位推荐、维保登记和异常告警处理
6. 实现首页统计和操作日志

### 第二阶段：前端开发
1. Vue3 + Vite + Element Plus + Pinia + Axios + ECharts
2. 登录页、布局页、首页看板
3. 仓库区域、库位档案、AGV 车辆、充电站点、库存物料、入库订单、出库订单、AGV 任务、路径规划、库位推荐、设备维保、异常告警和日志页面
4. 业务按钮支持调度、推荐、告警确认、维保登记和任务完结

### 第三阶段：文档更新
1. 更新 `readme.md`
2. 更新 `readme_simple.md`
3. 更新候选清单

## 完成标准

1. 后端结构完整。
2. 前端结构完整。
3. SQL 包含演示数据。
4. 认证返回不暴露敏感字段。
5. 调度、库位优化、维保和异常处置口径一致。
6. 文档描述与实际实现保持一致。
7. 不执行编译验证。
