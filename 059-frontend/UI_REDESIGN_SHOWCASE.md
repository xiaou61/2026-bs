# 059-frontend UI 美化展示

## 项目信息
- **项目名称**: 制造装备物联及生产管理ERP系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 工业蓝
- **美化完成时间**: 2026-05-29

## 设计理念

采用"工业蓝"设计理念，以深蓝色为主色调，体现制造业的专业性、可靠性和科技感。通过齿轮、工厂等装饰元素，营造出智能制造、高效生产的工业管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
