# 168-frontend UI 美化展示

## 项目信息
- **项目名称**: 在线职业培训证书考试与续证提醒系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 培训蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"培训蓝"设计理念，以蓝色为主色调，体现职业培训的专业性、权威性和成长感。通过证书、毕业帽等装饰元素，营造出学有所成、证有所依的职业培训氛围。

| 元素 | 动画 | 说明 |
|------|------|------|

| 顶部条 | certGradient | 流光渐变 |

168-frontend采用"培训蓝"设计理念，通过蓝色的主色调，营造出专业、权威的职业培训氛围。证书弹跳动画突出了在线职业培训证书考试的特色。
