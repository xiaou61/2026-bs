# 文旅场馆讲解预约与票务核销管理平台 PRD

## 项目概述
本系统面向博物馆、非遗馆、展览馆等文旅场馆，提供票务预约、讲解排期、扫码核销、客流统计、游客评价和公告发布的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 场馆主管：场馆档案、票种、排期、活动、公告和统计看板维护
- 讲解员：讲解员档案、讲解排期、讲解预约和游客评价处理
- 票务核销员：票务预约、扫码核销和客流统计维护
- 预约游客：票务预约、讲解预约和评价反馈提交

## 功能模块
- 场馆档案：场馆编码、场馆名称、开放区域、负责人、开放时间和状态维护
- 票种产品：票种编码、票种名称、适用人群、配置人、可售时段和上架状态维护
- 票务预约：预约单号、参观场馆、票种类型、预约人、参观时间和预约状态维护
- 讲解员档案：讲解员工号、姓名、语种等级、所属团队、可服务时间和在岗状态维护
- 讲解排期：排期编号、讲解路线、场次类型、讲解员、场次时间和排期状态维护
- 讲解预约：讲解预约单、讲解主题、团队类型、联系人、预约时间和确认状态维护
- 扫码核销：核销码、关联订单、核销渠道、核销员、核销时间和核销状态维护
- 客流统计：统计编号、场馆区域、客流类型、统计员、统计时段和预警状态维护
- 游客评价：评价编号、评价对象、评价类型、游客姓名、评价时间和处理状态维护
- 文旅活动：活动编号、活动名称、活动类型、策划人、活动时间和发布状态维护
- 场馆公告：公告编号、公告标题、公告类型、发布人、发布时间和公告状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
