# 167-frontend UI 美化展示

## 项目信息
- **项目名称**: 社区垃圾分类督导与积分兑换平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 环保绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"环保绿"设计理念，以绿色为主色调，体现垃圾分类的环保、可持续和责任感。通过回收符号、绿叶等装饰元素，营造出垃圾分类、从我做起的环保管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
