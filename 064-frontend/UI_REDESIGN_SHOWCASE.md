# 064-frontend UI 美化展示

## 项目信息
- **项目名称**: 经方药食两用服务平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 养生绿
- **美化完成时间**: 2026-05-29

## 设计理念

采用"养生绿"设计理念，以深绿色为主色调，体现中医药文化的自然、和谐和健康。通过草药、太极等装饰元素，营造出药食同源、健康养生的中医养生氛围。

- 深绿渐变背景

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放旋转 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
