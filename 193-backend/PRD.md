# 校园创新实验班选拔与导师跟踪管理系统 PRD

## 项目概述
本系统面向高校创新实验班选拔、学院教务、评审专家、导师教师、辅导员与学生用户，提供实验班项目、学生档案、导师档案、选拔公告、报名选拔、资格评审、面试考核、导师匹配、培养计划、过程跟踪、成果档案和操作日志的一体化管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 教务管理员：维护实验班项目、学生档案、导师档案和选拔规则
- 选拔评审：处理报名材料、资格评审和面试考核
- 导师教师：维护导师档案、参与导师匹配、制定培养计划并跟踪过程
- 辅导员：维护学生档案、跟进报名选拔和培养过程
- 学生用户：提交报名申请、查看导师匹配、培养计划和过程反馈

## 功能模块
- 实验班项目：项目编号、项目名称、培养方向、项目负责人、建档时间和项目状态维护
- 学生档案：档案编号、学生姓名、专业年级、辅导员、入学时间和培养状态维护
- 导师档案：导师编号、导师姓名、研究方向、所属学院、入库时间和导师状态维护
- 选拔公告：公告编号、公告名称、选拔批次、发布人员、发布时间和公告状态维护
- 报名选拔：报名编号、报名学生、申报方向、申请人员、报名时间和报名状态维护
- 资格评审：评审编号、报名申请、评审类型、评审人员、评审时间和评审状态维护
- 面试考核：考核编号、报名学生、考核项目、评委人员、考核时间和考核状态维护
- 导师匹配：匹配编号、入选学生、匹配方向、导师教师、匹配时间和匹配状态维护
- 培养计划：计划编号、培养学生、培养阶段、指导导师、计划时间和计划状态维护
- 过程跟踪：跟踪编号、培养计划、跟踪类型、跟进人员、跟踪时间和跟踪状态维护
- 成果档案：成果编号、培养学生、成果类型、指导人员、归档时间和成果状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护
