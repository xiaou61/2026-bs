# 114-frontend UI 美化展示

## 项目信息
- **项目名称**: 供应链冷链温控追踪与异常预警平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 冷链蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"冷链蓝"设计理念，以天蓝色为主色调，体现冷链物流的清新、可靠和专业感。通过雪花、温度计等装饰元素，营造出冷链保鲜、安全送达的物流管理氛围。

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |

114-frontend采用"冷链蓝"设计理念，通过天蓝的主色调，营造出清新、可靠的冷链物流氛围。雪花装饰元素突出了冷链温控追踪的特色。
