# 198-frontend UI 美化展示

## 项目信息
- **项目名称**: 城市共享充电宝投放巡检与收益结算系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 充电紫
- **美化完成时间**: 2026-05-30

## 设计理念

采用"充电紫"设计理念，以紫色和蓝色为主色调，体现共享充电的科技感、便捷和活力。通过电池、闪电等装饰元素，营造出随时随地、电量满满的共享充电氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |

| 顶部条 | chargeGradient | 彩虹流光 |
| 按钮 | hover | 悬浮上移 |

198-frontend采用"充电紫"设计理念，通过紫色和蓝色的搭配，营造出科技、便捷的共享充电氛围。电池充电动画突出了共享充电宝的特色。
