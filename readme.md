## 📖 使用说明

### 环境要求
- JDK 17+
- MySQL 8.0+
- Maven 3.6+
- Node.js 16+ (前端项目)
- Redis (004项目需要)

### 快速启动
1. 导入数据库脚本（每个项目的 `sql` 文件夹）
2. 修改 `application.yml` 配置数据库连接
3. 后端：`mvn spring-boot:run` 或直接运行主类
4. 前端：`npm install && npm run dev`（Vue项目）
5. 访问项目地址（默认8080端口）

### 测试账户
每个项目都包含测试账户，详见各项目的 `ACCOUNTS.md` 或 `README.md`
