## 🚀 快速开始

### 环境要求

- JDK 1.8 / 11 / 17（根据具体项目 `pom.xml` 而定）
- MySQL 5.7 / 8.0
- Node.js 16+（Vue 前端项目）
- Maven 3.6+
- Redis（部分项目需要）

### 启动任意项目（通用步骤）

1. 进入对应项目目录，如 `001-backend` / `001-frontend`
2. 执行数据库脚本（通常在 `sql/init.sql` 或 `src/main/resources/sql/init.sql`）
3. 修改 `application.yml` 中的数据库连接配置
4. 后端：`mvn spring-boot:run` 或运行主类
5. 前端：`npm install && npm run dev`

> 每个项目的具体启动方式，请参考对应目录下的 `README.md`。
