# 068-frontend UI 美化展示

## 项目信息
- **项目名称**: 周边游平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 旅行紫
- **美化完成时间**: 2026-05-29

## 设计理念

采用"旅行紫"设计理念，以紫色和蓝色为主色调，体现旅行的自由、浪漫和探索精神。通过海滩、山脉等装饰元素，营造出说走就走、畅游世界的旅行氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳缩放 |
| 顶部条 | travelGradient | 彩虹流光 |
| 按钮 | hover | 悬浮上移 |
