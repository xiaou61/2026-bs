# 060-frontend UI 美化展示

## 项目信息
- **项目名称**: 电影订票及评论网站
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 电影红
- **美化完成时间**: 2026-05-29

## 设计理念

采用"电影红"设计理念，以红色和暖色为主色调，体现电影的热情、精彩和娱乐感。通过电影、爆米花等装饰元素，营造出光影世界的观影氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 电影红 | #e94560 | 主色调 |
| 暖红 | #ff6b6b | 辅助色 |
| 金色 | #ffd93d | 强调色 |
| 暗蓝 | #1a1a2e | 背景色 |
| 绿色 | #6bcb77 | 注册色 |

## 设计特点

### 登录页面
- 暗蓝渐变背景
- 电影装饰元素浮动
- 电影🎬Logo脉冲动画
- 流光渐变顶部条
- 登录/注册标签页切换
- 电影名言展示

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放旋转 |
| 顶部条 | cinemaGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

060-frontend采用"电影红"设计理念，通过红色和暖色的搭配，营造出精彩的电影观影氛围。电影装饰元素突出了电影订票的娱乐特色。
