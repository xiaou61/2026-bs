# 城市公厕巡检保洁与耗材补给调度系统 PRD

## 项目概述
本系统面向城市公厕点位巡检、保洁任务执行、耗材库存补给、投诉反馈和问题整改场景，提供公厕点位、保洁路线、保洁任务、保洁记录、巡检计划、巡检记录、耗材库存、补给申请、补给调度、投诉反馈和问题整改的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 环卫管理员：维护公厕点位、保洁路线、巡检计划和调度规则
- 保洁员：接收保洁任务、填写保洁记录并提交补给申请
- 补给员：维护耗材库存、处理补给申请并执行补给调度
- 巡检员：执行巡检记录、发现问题并跟踪整改闭环
- 市民监督：提交投诉反馈并查看处理进度

## 功能模块
- 公厕点位：点位编号、点位名称、区域类型、负责人员、启用时间和点位状态维护
- 保洁路线：路线编号、覆盖区域、路线类型、保洁人员、计划时间和路线状态维护
- 保洁任务：任务编号、公厕点位、任务类型、保洁人员、派发时间和任务状态维护
- 保洁记录：记录编号、保洁任务、记录类型、保洁人员、完成时间和记录状态维护
- 巡检计划：计划编号、巡检区域、巡检类型、巡检人员、计划时间和计划状态维护
- 巡检记录：记录编号、公厕点位、巡检类型、巡检人员、巡检时间和巡检状态维护
- 耗材库存：库存编号、耗材名称、耗材类型、保管人员、入库时间和库存状态维护
- 补给申请：申请编号、公厕点位、补给类型、申请人员、申请时间和申请状态维护
- 补给调度：调度编号、补给申请、调度类型、补给人员、调度时间和调度状态维护
- 投诉反馈：反馈编号、公厕点位、反馈类型、反馈人员、反馈时间和反馈状态维护
- 问题整改：整改编号、巡检问题、整改类型、责任人员、整改时间和整改状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护
