# 137-frontend UI 美化展示

## 项目信息
- **项目名称**: 大学生创新创业项目孵化管理平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 创业橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"创业橙"设计理念，以橙色和金色为主色调，体现创新创业的热情、活力和梦想。通过灯泡、火箭等装饰元素，营造出创新创业、梦想起航的孵化管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|

| 亮金 | #FFB300 | 装饰色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | glow | 发光脉冲 |
| 顶部条 | innovationGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
