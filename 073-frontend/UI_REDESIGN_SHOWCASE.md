# 073-frontend UI 美化展示

## 项目信息
- **项目名称**: 人事管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 人事蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"人事蓝"设计理念，以蓝色为主色调，体现人事管理的专业性、可靠性和规范性。通过团队、办公等装饰元素，营造出以人为本、高效管理的企业人事氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | gradientMove | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
