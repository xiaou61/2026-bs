# 实现计划

## 问题陈述

构建一个围绕法律咨询、案件进度和智能文书管理的前后端分离系统，覆盖委托人、律师、案件、进度、咨询、模板、文书、预约、证据、费用和审计统计。

## 当前状态

`102-backend` 和 `102-frontend` 目录需要按仓库规则补齐 PRD、PLAN、后端、前端和合集 README。

## 实施方案

### 第一阶段：后端开发
1. Spring Boot 2.7.18 + MyBatis 2.3.1 + PageHelper + Redis + JWT
2. 创建 `legal_case_102` 数据库和 13 张表
3. 实现统一返回、异常处理、JWT 拦截、Redis 配置和 PageHelper 分页
4. 实现 Entity、Mapper、Service 和 Controller
5. 实现案件推进、节点完成、模板启停、文书生成复核、预约状态、证据核验和费用支付
6. 实现首页统计和操作日志

### 第二阶段：前端开发
1. Vue3 + Vite + Element Plus + Pinia + Axios + ECharts
2. 登录页、布局页、首页看板
3. 用户、委托人、律师、案件、进度、咨询、模板、文书、版本、预约、证据、费用、日志页面
4. 业务按钮支持推进、结案、生成、复核、确认、核验和支付

### 第三阶段：文档更新
1. 更新 `readme.md`
2. 更新 `readme_simple.md`
3. 更新候选清单

## 完成标准
