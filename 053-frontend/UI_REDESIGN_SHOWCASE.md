# 053-frontend UI 美化展示

## 项目信息
- **项目名称**: 救灾物资调度与救援系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 救援红
- **美化完成时间**: 2026-05-29

## 设计理念

采用"救援红"设计理念，以红色为主色调，体现紧急救援的紧迫感和重要性。通过救护车、急救等装饰元素，营造出紧急、高效的救灾物资调度氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 救援红 | #F44336 | 主色调 |
| 深红 | #D32F2F | 辅助色 |

| 浅红 | #E57373 | 装饰色 |
| 紧急红 | #FF5252 | 装饰色 |

## 设计特点

### 登录页面
- 红色渐变背景
- 救援装饰元素浮动
- 求救🆘Logo脉冲动画
- 流光渐变顶部条
- 渐变登录按钮
- 紧急救援标语

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 快速脉冲缩放 |
| 顶部条 | gradientMove | 快速流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

053-frontend采用"救援红"设计理念，通过红色的主色调，营造出紧急、高效的救灾物资调度氛围。快速脉冲动画和紧急标语突出了救援系统的紧迫性。
