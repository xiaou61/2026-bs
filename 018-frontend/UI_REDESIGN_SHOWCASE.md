# 018-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园实习招聘与求职平台
- **技术栈**: Vue 3 + Element Plus + Vite
- **设计主题**: 职场蓝
- **美化完成时间**: 2026-05-29

## 设计理念

采用"职场蓝"设计理念，以蓝色和金色为主色调，体现职场的专业感和成就感。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 职场蓝 | #1976D2 | 主色调 |
| 成就金 | #FFB300 | 辅助色 |
| 活力绿 | #43A047 | 强调色 |
| 背景色 | #F5F7FA | 页面背景 |
