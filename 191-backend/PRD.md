# 社区助残器具借用与康复随访平台 PRD

## 项目概述
本系统面向社区助残服务站、器具管理人员、康复治疗师、随访志愿者与居民用户，提供服务站点、居民档案、助残器具、器具借用、借用审核、器具交付、康复计划、康复训练、随访记录、回收提醒、器具维护和操作日志的一体化管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 社区服务员：维护服务站点、居民档案、借用审核和服务规则
- 器具管理员：维护助残器具、器具交付、回收提醒和器具维护
- 康复治疗师：制定康复计划、记录康复训练并参与随访评估
- 随访志愿者：协助器具交付、随访记录和回收提醒跟进
- 居民用户：提交器具借用申请、查看康复计划和随访进度

## 功能模块
- 服务站点：站点编号、站点名称、服务区域、站点负责人、建档时间和站点状态维护
- 居民档案：档案编号、居民姓名、需求类别、联系人、登记时间和服务状态维护
- 助残器具：器具编号、器具名称、器具类型、保管人员、入库时间和器具状态维护
- 器具借用：借用编号、借用居民、器具类型、申请人员、申请时间和借用状态维护
- 借用审核：审核编号、借用申请、审核类型、审核人员、审核时间和审核状态维护
- 器具交付：交付编号、借用申请、交付方式、交付人员、交付时间和交付状态维护
- 康复计划：计划编号、服务居民、康复类型、治疗师、计划时间和计划状态维护
- 康复训练：训练编号、康复计划、训练项目、指导人员、训练时间和训练状态维护
- 随访记录：随访编号、随访居民、随访类型、随访人员、随访时间和随访状态维护
- 回收提醒：提醒编号、借用器具、提醒类型、跟进人员、提醒时间和回收状态维护
- 器具维护：维护编号、维护器具、维护类型、维护人员、维护时间和维护状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护

- 前端：Vue3、Element Plus、Pinia、Axios、ECharts、Vite
