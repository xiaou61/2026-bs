# 水产养殖环境监测与投喂预警系统 PRD

## 项目定位
水产养殖环境监测与投喂预警系统面向毕业设计答辩场景，提供多角色登录、业务台账维护、状态流转、统计看板和操作留痕能力。

## 用户角色
管理员、养殖员、水质技术员、养殖经理

## 功能模块
- 养殖池塘：池塘编号、名称、养殖区域、水面面积和负责人维护
- 传感设备：传感器编号、池塘、设备类型、安装位置和电量维护
- 水质读数：读数编号、池塘、采集时间、溶氧、PH和状态维护
- 投喂计划：计划编号、池塘、鱼种、饲料类型和日投喂量维护
- 投喂记录：投喂编号、池塘、投喂时间、投喂量和操作人维护
- 鱼苗批次：批次编号、池塘、鱼种、投苗数量和投苗日期维护
- 生长采样：采样编号、批次、采样日期、均重和记录人维护
- 病害预警：预警编号、池塘、预警等级、症状和处理人维护
- 用药记录：用药编号、池塘、药品名称、用量和使用日期维护
- 养殖设备：设备编号、池塘、设备名称、设备类型和安装位置维护
- 水质规则：规则编号、规则名称、指标名称、阈值和等级维护
- 产量统计：统计编号、池塘、统计月份、产量和成活率维护
- 操作日志：关键模块操作人、动作、对象、详情和结果记录

- 前端：Vue3、Vite、Element Plus、Pinia、Axios、ECharts
