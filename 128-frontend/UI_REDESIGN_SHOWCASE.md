# 128-frontend UI 美化展示

## 项目信息
- **项目名称**: ESG数据填报与可视化报告系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: ESG蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"ESG蓝"设计理念，以蓝色为主色调，体现ESG披露的专业性、可靠性和透明度。通过图表、奖杯等装饰元素，营造出ESG披露、可持续发展的企业管理氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |

| 元素 | 动画 | 说明 |
