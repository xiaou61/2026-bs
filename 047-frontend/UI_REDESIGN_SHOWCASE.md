# 047-frontend UI 美化展示

## 项目信息
- **项目名称**: 剧本杀创作与预约平台
- **技术栈**: Vue 3 + TypeScript + Element Plus
- **设计主题**: 悬疑紫
- **美化完成时间**: 2026-05-29

## 设计理念

采用"悬疑紫"设计理念，以紫色和玫红色为主色调，体现剧本杀的悬疑、神秘和推理氛围。通过面具、放大镜等装饰元素，营造出沉浸式的剧本杀体验氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |

| 顶部条 | mysteryGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |
