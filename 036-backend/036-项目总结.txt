036-小梦想全球公益捐赠平台 项目总结
========================================

项目名称：基于SpringBoot的小梦想全球公益捐赠平台的设计与实现

一、技术选型（实现多元化）
━━━━━━━━━━━━━━━━━━━━━━━━
后端技术栈：
├─ 框架：Spring Boot 3.2.0
├─ ORM：Spring Data JPA（区别于其他项目的MyBatis-Plus）
├─ 数据库：PostgreSQL（区别于其他项目的MySQL）
├─ 认证：JWT
├─ 工具：Lombok、HuTool、Fastjson2
└─ 构建：Maven

前端技术栈：
├─ 框架：Vue 3 + Composition API
├─ 路由：Vue Router 4
├─ 状态管理：Pinia
├─ UI组件：Element Plus
├─ HTTP客户端：Axios
└─ 构建工具：Vite

四、数据库设计（PostgreSQL）
━━━━━━━━━━━━━━━━━━━━━━━━
表结构：
1. users                用户表

索引优化：
- 用户名、邮箱索引
- 项目状态、分类索引

- 项目进度关联索引

五、API接口（RESTful风格）
━━━━━━━━━━━━━━━━━━━━━━━━
用户接口：
POST   /api/user/register      用户注册
POST   /api/user/login         用户登录
GET    /api/user/info          获取用户信息

项目接口：
GET    /api/projects           获取项目列表
POST   /api/projects           创建项目
GET    /api/projects/{id}      获取项目详情
GET    /api/projects/my        获取我的项目
GET    /api/projects/latest    获取最新项目
PUT    /api/projects/{id}/status  更新项目状态
DELETE /api/projects/{id}      删除项目

捐赠接口：
POST   /api/donations          创建捐赠
GET    /api/donations/my       获取我的捐赠
GET    /api/donations/project/{id}  获取项目捐赠

进度接口：
POST   /api/progress           创建进度
GET    /api/progress/project/{id}   获取项目进度
DELETE /api/progress/{id}      删除进度
