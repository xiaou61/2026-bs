# 科研项目经费报销与成果管理系统 PRD

## 项目定位
科研项目经费报销与成果管理系统面向毕业设计答辩场景，提供多角色登录、业务台账维护、审批状态流转、统计看板和操作留痕能力。

## 用户角色
系统管理员、科研人员、财务人员、审批领导

## 功能模块
- 科研项目：项目编号、项目名称、负责人、所属学院、立项年份和状态维护
- 预算科目：科目编号、科目名称、经费用途、控制方式、负责人和状态维护
- 经费预算：预算编号、项目编号、预算科目、预算金额、已用金额和状态维护
- 报销申请：报销编号、项目编号、申请人、报销金额、申请时间和状态维护
- 发票记录：发票编号、报销编号、发票类型、发票金额、开票单位和状态维护
- 审批任务：审批编号、报销编号、审批节点、审批人、审批意见和状态维护
- 支付记录：支付编号、报销编号、支付金额、支付时间、经办人和状态维护
- 科研成果：成果编号、项目编号、成果名称、成果类型、完成人和状态维护
- 论文成果：论文编号、项目编号、论文题目、期刊名称、发表时间和状态维护
- 专利成果：专利编号、项目编号、专利名称、申请人、授权时间和状态维护
- 绩效统计：统计编号、项目编号、统计月份、报销次数、成果数量和状态维护
- 风险预警：预警编号、项目编号、风险级别、触发原因、处理人和状态维护
- 操作日志：关键模块操作人、动作、对象、详情和结果记录

## 技术要求
- 后端：Spring Boot 2.7.18、MyBatis 2.3.1 注解 SQL、PageHelper 1.4.7、MySQL、Redis、JWT
- 前端：Vue3、Vite、Element Plus、Pinia、Axios、ECharts
- 数据库：research_fund_134
- 默认账号密码：各角色账号均使用 `123456`
