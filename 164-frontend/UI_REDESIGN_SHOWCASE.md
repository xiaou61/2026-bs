# 164-frontend UI 美化展示

## 项目信息
- **项目名称**: 校园体育赛事报名编排与裁判评分系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 体育橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"体育橙"设计理念，以橙色和金色为主色调，体现体育赛事的活力、激情和竞技感。通过足球、奖杯等装饰元素，营造出生命在于运动的校园体育氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 体育橙 | #FF6F00 | 主色调 |
| 暖橙 | #FF8F00 | 辅助色 |
| 金色 | #FFA000 | 强调色 |
| 亮金 | #FFB300 | 装饰色 |
| 深橙 | #E65100 | 文字色 |

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | spin | 持续旋转 |
| 顶部条 | sportGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

164-frontend采用"体育橙"设计理念，通过橙色和金色的搭配，营造出活力、激情的校园体育赛事氛围。足球旋转动画突出了体育赛事的竞技特色。
