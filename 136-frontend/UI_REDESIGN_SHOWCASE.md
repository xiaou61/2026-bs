# 136-frontend UI 美化展示

## 项目信息
- **项目名称**: 导师课题双选与开题过程管理系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 毕业蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"毕业蓝"设计理念，以蓝色为主色调，体现毕业设计的严谨性、专业性和学术感。通过毕业帽、书本等装饰元素，营造出毕业设计、成果展示的学术管理氛围。
