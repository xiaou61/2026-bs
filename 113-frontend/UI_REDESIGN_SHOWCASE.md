# 113-frontend UI 美化展示

## 项目信息
- **项目名称**: 区块链农产品质量溯源与监管平台
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 农业绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"农业绿"设计理念，以深绿色为主色调，体现农业的生态、自然和可追溯性。通过稻穗、区块链等装饰元素，营造出从农田到餐桌、全程可追溯的农产品溯源氛围。
