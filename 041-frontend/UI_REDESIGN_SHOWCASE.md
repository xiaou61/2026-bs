# 041-frontend UI 美化展示

## 项目信息
- **项目名称**: 在线心理评测与咨询系统
- **技术栈**: Vue 3 + Element Plus + Pinia
- **设计主题**: 治愈绿
- **美化完成时间**: 2026-05-29

## 设计理念

采用"治愈绿"设计理念，以绿色为主色调，体现心理健康关爱的温暖、自然和治愈感。通过爱心、绿叶等装饰元素，营造出温馨、安全的心理咨询氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
