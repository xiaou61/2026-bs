# 高校毕业去向填报与就业帮扶跟踪系统 PRD

## 项目概述
本系统面向高校毕业去向填报、就业材料审核、岗位推荐、就业帮扶、协议归档和跟踪回访场景，提供学院专业、毕业生档案、用人单位、招聘岗位、去向填报、协议归档、材料审核、帮扶记录、岗位推荐、跟踪回访和就业统计的一体化后台管理能力。

## 用户角色
- 系统管理员：账号权限、基础配置和全部数据维护
- 就业中心：维护去向填报、材料审核、就业统计和就业服务规则
- 学院就业老师：维护学院专业、毕业生档案、协议归档和统计汇总
- 辅导员：维护毕业生档案、帮扶记录、岗位推荐和跟踪回访
- 毕业生：提交去向填报、查看岗位推荐、协议归档和帮扶进度
- 用人单位：维护单位档案、发布招聘岗位并协同就业推荐

## 功能模块
- 学院专业：学院编号、学院名称、专业方向、负责老师、启用时间和学院状态维护
- 毕业生档案：学生编号、学生姓名、毕业专业、辅导员、建档时间和学生状态维护
- 用人单位：单位编号、单位名称、行业类型、联系人、建档时间和单位状态维护
- 招聘岗位：岗位编号、岗位名称、岗位类型、发布单位、发布时间和岗位状态维护
- 去向填报：填报编号、毕业学生、去向类型、审核老师、填报时间和填报状态维护
- 协议归档：协议编号、毕业学生、协议类型、审核人员、归档时间和协议状态维护
- 材料审核：审核编号、审核学生、材料类型、审核人员、审核时间和审核状态维护
- 帮扶记录：帮扶编号、帮扶学生、帮扶类型、辅导人员、帮扶时间和帮扶状态维护
- 岗位推荐：推荐编号、推荐岗位、推荐类型、推荐人员、推荐时间和推荐状态维护
- 跟踪回访：回访编号、回访学生、回访类型、回访人员、回访时间和回访状态维护
- 就业统计：统计编号、统计学院、统计类型、统计人员、统计时间和统计状态维护
- 操作日志：日志编号、操作模块、操作类型、操作人、操作时间和执行结果维护
