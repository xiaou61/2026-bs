# 075-frontend UI 美化展示

## 项目信息
- **项目名称**: 汽车维修预约服务系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 汽修橙
- **美化完成时间**: 2026-05-30

## 设计理念

采用"汽修橙"设计理念，以橙色和深灰色为主色调，体现汽车维修的专业性、可靠性和活力感。通过扳手、汽车等装饰元素，营造出专业维修、安心出行的服务氛围。

## 颜色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 汽修橙 | #FF6F00 | 主色调 |
| 深灰 | #263238 | 辅助色 |
| 暖橙 | #FF8F00 | 强调色 |
| 金色 | #FFB300 | 装饰色 |
| 暗橙 | #E65100 | 文字色 |

## 设计特点

### 登录页面
- 深灰渐变背景
- 汽修装饰元素浮动
- 扳手🔧Logo弹跳动画
- 橙色流光渐变顶部条
- 登录/注册标签页切换
- 汽修服务标语

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | bounce | 弹跳缩放 |
| 顶部条 | autoGradient | 橙色流光 |
| 按钮 | hover | 悬浮上移 |

## 总结

075-frontend采用"汽修橙"设计理念，通过橙色和深灰的搭配，营造出专业、可靠的汽车维修服务氛围。扳手装饰元素突出了汽车维修的行业特色。
