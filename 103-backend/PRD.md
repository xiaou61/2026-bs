# 智能客服工单质检与知识推荐系统

## 项目概述

本项目面向企业客服中心和在线服务团队，提供客户档案、服务渠道、知识库、客服工单、会话消息、派单记录、质检规则、质检任务、质检结果、知识推荐、坐席绩效和操作审计能力。系统采用 Spring Boot + MyBatis-Plus + MySQL + Redis + Vue3 技术栈，围绕“客户建档-工单受理-会话记录-质检任务-结果复核-知识推荐-绩效统计”形成闭环。

## 功能需求

### 1. 账号权限
- 管理员维护主管、坐席和质检员账号
- JWT 登录鉴权和 Redis 登录态缓存
- 按角色限制工单、质检和知识库操作

### 2. 客户档案
- 维护客户姓名、电话、邮箱、等级和标签
- 支持按等级、标签和状态筛选

### 3. 服务渠道
- 管理网页、电话、微信、邮件等服务渠道
- 支持渠道启停和归属团队维护

### 4. 知识分类
- 维护知识库分类、父级分类、排序和启停状态

### 5. 知识文章
- 维护标题、关键词、正文、命中次数和发布状态
- 支持发布和下线

### 6. 客服工单
- 管理工单编号、客户、坐席、渠道、标题、内容、优先级和状态
- 支持受理、解决和关闭

### 7. 会话消息
- 记录工单中的客户和坐席消息
- 标记敏感话术和质检关注内容

### 8. 派单记录
- 记录工单转派来源、目标坐席、原因和状态

### 9. 质检规则
- 配置禁用话术、服务态度、流程合规和响应及时性规则
- 设置扣分项和启停状态

### 10. 质检任务
- 为工单创建质检任务并启动
- 生成质检结果、命中规则、分数、风险等级和改进建议

### 11. 质检结果
- 查看质检分、命中规则、风险等级和建议
- 支持人工复核通过或驳回

### 12. 知识推荐
- 根据工单关键词推荐知识文章
- 记录推荐分、采纳状态和坐席反馈

### 13. 坐席绩效
- 统计坐席处理量、解决量、平均质检分、满意度和等级

### 14. 统计审计
- 首页展示工单、知识、质检、推荐和绩效指标
- 操作日志记录关键业务动作

## 用户角色

- 管理员：账号、日志和基础配置
- 客服主管：渠道、派单和坐席绩效
- 客服坐席：工单处理、会话消息和知识采纳
- 质检员：规则、任务、结果复核和改进建议
