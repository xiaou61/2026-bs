# 实现计划

## 问题陈述

构建一个围绕设备寿命预测、备件库存和维保决策的前后端分离系统，覆盖设备资产、备件目录、出入库记录、运行指标、故障记录、维保计划、采购申请和风险预警。

## 当前状态

`119-backend` 与 `119-frontend` 已完成备件寿命预测系统的基础实现，当前进入精修阶段，重点核对认证安全、库存流转、预测口径和文档准确性。

## 实施方案

### 第一阶段：后端开发
1. Spring Boot 2.7.18 + MyBatis-Plus 3.5.5 + Redis + JWT
2. 创建 `spare_life_119` 数据库和 14 张业务表
3. 实现统一返回、异常处理、JWT 拦截和 Redis Token 校验
4. 实现 Entity、Mapper、Service 和 Controller
5. 实现寿命预测、维保计划、采购申请、库存流转和风险预警
6. 实现首页统计和操作日志

### 第二阶段：前端开发
1. Vue3 + Vite + Element Plus + Pinia + Axios + ECharts
2. 登录页、布局页、首页看板
3. 设备资产、备件目录、备件库存、入库记录、出库领用、使用记录、运行指标、故障记录、寿命预测、维保计划、采购申请、风险预警和日志页面
4. 业务按钮支持出入库、预测、预警确认、维保登记和采购申请

### 第三阶段：文档更新
1. 更新 `readme.md`
2. 更新 `readme_simple.md`
3. 更新候选清单

## 完成标准

1. 后端结构完整。
2. 前端结构完整。
3. SQL 包含演示数据。
4. 认证返回不暴露敏感字段。
5. 库存、寿命预测、维保计划和采购申请口径一致。
6. 文档描述与实际实现保持一致。
7. 不执行编译验证。
