# 130-frontend UI 美化展示

## 项目信息
- **项目名称**: 温室大棚物联网控制与病害预警系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 温室绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"温室绿"设计理念，以绿色为主色调，体现温室农业的生态、自然和智慧感。通过绿叶、温室等装饰元素，营造出智慧温室、丰收四季的农业管理氛围。

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |

| 顶部条 | greenhouseGradient | 流光渐变 |
