# 096-frontend UI 美化展示

## 项目信息
- **项目名称**: 线上医院挂号系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 医疗蓝
- **美化完成时间**: 2026-05-30

## 设计理念

采用"医疗蓝"设计理念，以蓝色和青色为主色调，体现医疗行业的专业性、可靠性和信任感。通过医院、医生等装饰元素，营造出关爱健康、呵护生命的医疗挂号氛围。

### 登录页面
- 蓝色渐变背景

- 流光渐变顶部条
- 登录/患者注册标签页切换
- 注册表单滚动区域

- 医疗健康标语

## 动画效果

| 元素 | 动画 | 说明 |
|------|------|------|
| 装饰元素 | floatDeco | 上下浮动旋转 |
| Logo | pulse | 脉冲缩放 |
| 顶部条 | hospitalGradient | 流光渐变 |
| 按钮 | hover | 悬浮上移 |

## 总结

096-frontend采用"医疗蓝"设计理念，通过蓝色和青色的搭配，营造出专业、可靠的医疗挂号氛围。医院装饰元素突出了线上医院挂号的医疗特色。
