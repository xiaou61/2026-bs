# 166-frontend UI 美化展示

## 项目信息
- **项目名称**: 农贸市场摊位巡检与食品追溯台账系统
- **技术栈**: Vue 3 + Element Plus
- **设计主题**: 农贸绿
- **美化完成时间**: 2026-05-30

## 设计理念

采用"农贸绿"设计理念，以绿色为主色调，体现农贸市场的生态、新鲜和健康。通过蔬菜、水果等装饰元素，营造出食品安全、源头追溯的农贸管理氛围。
