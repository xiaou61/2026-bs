# 网络钓鱼邮件演练与安全意识培训平台

## 项目概述
网络钓鱼邮件演练与安全意识培训平台面向企业安全运营场景，围绕员工档案、邮件模板、演练活动、目标名单、发送记录、点击追踪、培训课程、培训考试和风险评分构建完整闭环，帮助企业提升员工识别钓鱼邮件和保护账号凭据的能力。

## 技术栈
- 后端：Spring Boot 2.7.18、MyBatis-Plus 3.5.5、MySQL 8.0、Redis、JWT
- 前端：Vue 3.4.0、Element Plus 2.4.4、Vite 5、Pinia 2.1.7、Axios 1.6.2、ECharts 5.4.3
- 数据库：MySQL 8.0，数据库名 phishing_training_111
- 后端端口：8111
- 前端端口：3111

## 用户角色
- 管理员：维护账号、员工、部门和操作日志
- 安全专员：维护邮件模板、演练活动、目标名单和点击追踪
- 培训讲师：维护课程、考试、题目和考试记录
- 审计员：查看发送记录、风险评分和演练结果

## 数据库设计
- sys_user: 账号权限数据表
- employee_profile: 员工档案数据表
- department_group: 部门分组数据表
- mail_template: 邮件模板数据表
- phishing_campaign: 演练活动数据表
- campaign_target: 目标名单数据表
- mail_send_record: 发送记录数据表
- click_tracking: 点击追踪数据表
- training_course: 培训课程数据表
- training_exam: 培训考试数据表
- exam_question: 考试题目数据表
- exam_attempt: 考试记录数据表
- risk_score: 风险评分数据表
- operation_log: 操作日志数据表

## 核心流程
1. 管理员维护员工档案和部门分组
2. 安全专员创建邮件模板和演练活动
3. 安全专员导入目标名单并记录邮件发送状态
4. 系统沉淀打开、点击和凭据提交追踪记录
5. 培训讲师发布安全意识课程和培训考试
6. 系统根据点击行为和考试成绩生成风险评分
